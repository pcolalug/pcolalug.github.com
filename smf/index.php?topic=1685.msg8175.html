usually what i do every so often is su over to root, run:

swaret --update && swaret --upgrade -a

the -a in --upgrade tells it to upgrade everything without prompting, so i can run that command and leave it be for a while till it's finished, you can also upgrade by keyword if you want and so on, can also install/remove packages by keyword and their appropriate add/remove options in swaret

after thats done I usually run the command dropline-installer which brings up the dropline-gnome installer/updater and upgrade it while im upgrading everything else

can also do these as cronjobs if your incredibly lazy
