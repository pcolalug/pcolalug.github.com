I've got Slackware 9.1 running well on my laptop, except for one thing - can't get it to see my battery status.  I'm a little confused on ACPI and APM, and which one is needed/wanted.

I've added "acpi=on" to my kernel boot line in Grub, with no apparent difference.  When I've had Mandrake on this laptop (HP Pavilion N5270), I had a nice battery monitor that worked properly, so don't think it's a hardware issue.

Somehow I vaguely remember that APM may not be included in the default Slackware kernel (I used the default "bare.i" during install, 2.4.22).

Anybody got this function working on a laptop?  Any ideas?  TIA.

Dan

Title: Laptop Slack
Post by: imnes on December 24, 2003, 08:37:27 pm
From a console as root:
modprobe apm

Then edit /etc/rc.modules to uncomment the modprobe apm line.  If APM doesn't work on your system then try ACPI but I've never had much luck there.
