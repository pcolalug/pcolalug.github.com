Title: Significance of Assembly
Post by: imnes on February 06, 2004, 09:33:20 pm
Depends on what kind of programs you're writing.  If you're working on something where you need optimal performace, then knowing ASM is nice.  Even if you're coding in C or Pascal or Fortran, it gives you a better understanding of how you can write your code so that the compiler can optimize it better.  You'll learn about memory management, you'll learn about the stack and heap.  If you're writing business apps in delphi I guess that doesn't really help you.  Knowing ASM is kind of useless, but all the in-depth knowledge that you get along with it is good I guess.
