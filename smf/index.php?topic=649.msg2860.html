I suspect your server is not configured to autocreate the variables based on the form values. You should be able to configure your server to do this for you. Try checking out the online php docs for more details (pay attention to the part about register_globals).

Another alternative is to get the value yourself. Near the top of accesslist.php, add the following line before the "if":

$passin = $_REQUEST['passin'];

HTH
