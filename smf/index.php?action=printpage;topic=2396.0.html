Title: Re: Enlightenment DR 17
Post by: Brent on December 01, 2005, 04:26:04 pm
I for one, choose fluxbox.  Primarily, because it is easy to configure and customize to one's taste.

Example from my flubox menu config

      [exec] (console) {aterm -tr -sh 50 -fg green}

Example from fluxbox's keybinds config:

Mod1 F5 :ExecCommand aterm -tr -sh 50 -fg green

I can edit my menus and keybinds faster in fluxbox than I could even think about in kde, gnome, or (god forbid) iceWM.

AFAIK enlightenment uses binary config files to make things run faster... but I from what I've experienced playing with it, configuration in E17 isn't the most straight forward process.  (It will likely get better with time)
