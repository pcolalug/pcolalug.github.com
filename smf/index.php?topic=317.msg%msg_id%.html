If you want to be sure if it is your firewall settings, you can probably just run this to flush all your iptables rule:
Code:
# iptables -F

Run that as root on the server, then try connecting to your RtCW server from another system on your internal network. You might want to take down your Internet connection because your box will be completely open to the world. If that works, I'm sure we can help you with adding a rule to open up those ports. I'm sure someone runs RH8.0 and can find the file that holds the configs.
