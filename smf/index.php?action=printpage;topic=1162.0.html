Title: Debiam ZIP problem
Post by: steve on November 17, 2003, 07:49:23 am
If the file is a true "ZIP" file, in other words, the extension is .zip then bzip and gzip will not work on it. You need a program called unzip.

Here is the Debian Info on this program. (http://packages.debian.org/stable/utils/unzip.html)
