4  Distributions / Debian Based Distrtibutions / Re: authenticating ubuntu to AD on: February 06, 2010, 03:50:27 pm
All right here you go works every time.
https://help.ubuntu.com/9.10/serverguide/C/likewise-open.html

Notes
 
After successfully joining an Ubuntu machine to an Active Directory domain you can authenticate using any valid AD user.
To login you will need to enter the user name as 'domain\username'. For example to ssh to a server joined to the domain enter:

ssh 'example\steve'@hostname

[Note]    

If configuring a Desktop the user name will need to be prefixed with domain\ in the graphical logon as well.

To make likewise-open use a default domain, you can add the following statement to /etc/samba/lwiauthd.conf:

winbind use default domain = yes

Then restart the likewise-open daemons:

sudo /etc/init.d/likewise-open restart


The above is a must to make it easy for the average user.

Please see the troubleshooting section before using.
You must make sure the domain Controller is the first entry in the /etc/resolv.conf file.   ***Note your AD should always be your primary DNS or you will get very long long in delays ***
Also need to edit /etc/nsswitch.conf

Hope that helps this link above is as simple as it gets. Let me know if you have any troubles.
