Welcome to the group Will.

The meetings are third Saturday of the month at 6:00 pm. The actual date can be found at the main page: www.pcolalug.org.

I would recommend a distro that makes installation of new software pretty simple. There are Suse (http://www.novell.com/products/suselinux/downloads/) and Mandriva (http://www.mandriva.com/download) that seem to be pretty popular. If you have a somewhat recent machine (Windows ME age range is not recent enough) you might try FreeSpire (http://freespire.org/). It is pretty easy to install software from. It will require an Internet connection to do so, but simple to do.

Welcome to the group.  It is always great when someone else expresses interest in learning more about Linux and alternatives to the dark side.

Another great way to get your feet wet with Linux is to try out a Live-CD distro.  These distros just boot and run right from the CD.  A few of my favorites are DSL (Darn Small Linux), Puppy Linux, and Simply Mepis.  An excellent web-site for info on all types of distros is: http://distrowatch.com/.

Again, welcome.
