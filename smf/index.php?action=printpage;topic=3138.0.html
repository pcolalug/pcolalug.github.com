Title: K3B hit 1.0
Post by: dpeach on March 23, 2007, 08:34:20 pm
K3B is a great CD burning program (http://www.k3b.org/). It has come a long way since I started using it in 2002. That was back when I found it easier to burn CDs from the command line (http://www.pcolalug.org/smf/index.php?topic=172.0) than from any available GUI at the time. We have come a long way since then.

The 1.0 final of K3B was released March 16th, 2007.
