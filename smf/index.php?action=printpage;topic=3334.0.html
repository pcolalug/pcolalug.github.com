I can't really help you with the Frontpage question, but I might be able to suggest some alternatives as far as video editing.  I've used Cinelerra (http://cinelerra.org/) in the past, though I'd say that Cinelerra is more like Adobe Premier than it is like Powerlink , so it's not as intuitive as you might be accustomed to.
