Post by: dpeach on January 25, 2005, 10:43:39 pm
Copy this line into a "terminal" window (or console, you know that DOS looking thing).
Code:
sudo apt-get install gstreamer0.8-plugins

Now hit enter. After that is all done put this in and hit enter:
Code:
sudo apt-get install w32codecs


That should do it.
