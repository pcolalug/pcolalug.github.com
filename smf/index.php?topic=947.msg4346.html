I have not read NVidia's site, but I would suspect that you need to start your line with:
./

Therefore, your command to start the shell script would be (from the directory that the script is in):

./ sh NIVIDIA-Linux-x86-1.0-4496-pkg2.run

That forces it to run the script by that name from the current directory. Very nice trick to know as you may have to do this from time to time. I always run scripts with the ./ and that seems to take care of it.

Not sure if I get what it is you're trying to do here..

But one way to execute the nvidia installer script, is to do the following:

Code:
chmod u+x NIVIDIA-Linux-x86-1.0-4496-pkg2.run


The above step sets the executable bit on the file. Afterwards, do a:

Code:
./NIVIDIA-Linux-x86-1.0-4496-pkg2.run


with no spaces in between the dot and the slash and the filename.
