Hey, welcome to the forums

if you find that your wireless lan isn't fully supported in linux or the drivers are still being developed, something you can try is ndiswrapper
http://ndiswrapper.sourceforge.net/

I'm using that temporarily on a toshiba tecra S1 running centrino 802.11b with slackware linux 9.1 and 2.6.1-mm5 kernel....it actually runs relatively well and haven't had any problems with it as far as being buggy....might be worth a shot if it's your only choice, but if support is in the linux kernel then go with that option rather than ndiswrapper
