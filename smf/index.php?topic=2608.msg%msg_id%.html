I disagree that Linux will be the MS killer. The more I have been playing with and talking to people about OS X, the more I think that is the next big thing. I never felt Apple was poised to do this until this move to the Intel processors (that is assuming we will be able to use OS X on the same hardware you can run Windows on). People will begin by dual booting Windows and OS X. Then they will start spending more time in OS X. Then only booting Windows to play a few legacy games. Soon, OS X will have the market share.

The good side of this is that OS X will bring Linux right along with it. But, I am beginning to think that our biggest competition in 5 years will not be MS, but Apple. Honestly, I am very happy to see that.

I have NEVER been an Apple fan because I did not like the hardware. (Much like my distaste for notebooks). But, if we can get OS X on Intel/AMD processors and run much of our current hardware, I think Apple has a shot to kill MS. And that would be just fine with me.
