Title: Seeing Linux to Linux on a Network
Post by: imnes on October 14, 2004, 11:19:30 am
You'll need to setup file shares on the boxes.  You can go either samba or nfs.  If you don't have any windows machines to worry about, nfs is fine otherwise do samba.

Once you have your samba shares setup, it should be easy to browse from the other machines through konqueror or nautilus.

Title: Seeing Linux to Linux on a Network
Post by: rugmonster on October 14, 2004, 01:43:46 pm
nfs is the more "unix" style. It's is very easy as long as the kernel is configured to use it. Also, if you get Service for Unix, your Windows box can mount and export nfs shares.

You can edit your samba config (usually in /etc/samba/smb.conf depending on your distro). Samba is one of the best documented open source projects out there. Their howto's are absolutely amazing. Go check them out at http://www.samba.org
