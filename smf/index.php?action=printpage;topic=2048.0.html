Title: Debian Sarge
Post by: imnes on December 17, 2004, 03:24:26 pm
In Synaptic do this:

1.  Press "Reload" which will refresh the list of available packages.
2.  Press "Mark all Upgrades", and select "Smart Upgrade" if prompted.
3.  Press the first column in the list of packages to sort by selection.

That should list all updates first, they should be marked in that checkbox and highlighted.  There's probably a cleaner way to do it but I'm not a debian power user, but it works :)

If you want to actually apply the updates at that point, just hit Apply.

Title: Debian Installation
Post by: digger on January 27, 2005, 11:31:33 pm
Looks like I'm way late here, but the easiest way I have found to get a good Debian install is with the net install CD (about 107-113MB).  It can be downloaded from the Debian site.  It installs a base systwem, then depending on what one wants (desktop O/S, server, database, or whatever) downloads and installs the rest.  Broadband is a must

I installled this system 31 Dec. came with KDE and Gnome (2.8) & soon got an upgrade on KDE.  Uses the Gnome login manager (gdm) to choose between GNOME and KDE,  If gdm is not installed or gets removed it defaults to GNOME at startx command.  I never got a system I could use from the "Woody"  CDs: at best command-line only (& I'm too ignorant for that) or GRUB would hang.

Like others pointed out, the safest way to do Debian upgrades is "smart upgrade" with synaptic, or (at the command line) with apt-get update followed by apt-get distr-upgrade , as those routes seem to avoid dependency and/or incompatibility problems.
