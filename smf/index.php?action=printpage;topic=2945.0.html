Post by: rugmonster on July 27, 2006, 12:11:19 am
Google is your best resource. Take a machine and decide something you want to do with it. You aren't going to really learn a whole lot if you say you want to install Linux and learn how to use it without a purpose in mind. You've got to have a goal. My first project was to build a firewall/router. In the process of learning how to accomplish that, I discovered other possibilities and I have been branching out since then.

I wouldn't spend any more money on basic linux tutorials. There is enough information out on the web for free.
