Title: Request for Comments
Post by: rugmonster on June 07, 2003, 09:51:24 am
RFCs can be boring. I'm talking dry, technical, cure-your-chronic-insomnia boring. For those of you who are really geeky, they can be alright. If you ever want to learn more about a given protocol, you can go look up the RFC on it. I found rfc-editior.org (http://www.rfc-editor.org/rfc.html), which has a full library and index of all of the RFCs. If you ever get bored or you find a reason to learn more about a part of the Internet, swing over there and find the corresponding RFC.
