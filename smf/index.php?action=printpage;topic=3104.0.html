Title: Re: Remote Login
Post by: rugmonster on January 10, 2007, 12:27:05 pm
PuTTY is an ssh client that will give you command line access. Chances are, ssh is setup on your install. If not, it is trivial to install.

VNC will give you non-secure access to an X session, but requires more setup.

NX is similar to VNC, but faster and secure since it is tunneled through ssh, but it can be difficult to setup.

All three require clients, but by doing some googling, you can find plenty of documentation, howto's, etc on each.
