The executable would take up more space, since it has the library code included, but not large enough that anyone with a fairly recent system would mind. 30  Misc / IT Jobs / Re: standing out on: October 26, 2005, 01:41:14 pm
I graduate this semester from UWF with a BS in Computer Science so I hear you about the lack of programming jobs in the area. I really don't care to design military applications, but if that sort of thing interests you there are jobs around here with Lockheed, Tybrin, and a few other military contractors. I've had an internship and a few basic programming jobs, but I don't know yet how much they have helped me to stand out. If you only have a year or two left then you should be eligible for a co-op. They normally take three semesters. Even if it doesn't help you stand out, you'll learn things on a good co-op that you probably won't get in class(and possibly not on an open source project either).

Once, I was in charge of taking an application I knew nothing about, learning it, and adding new functionality to it. It was a smaller cgi script written in C++, but it was a good experience. I think if there is an open source project out there that really interests you, that you'd be surprised how much you could figure out about it if you just take some time to look it over. It helps to have some functionality in mind you want to add or at least a bug you are tired of seeing. Or start your own project. Either way it has to be something you want to see happen or you will stop working on it. You can even help with bug testing or documentation until you learn the application better. Those aren't exactly fun, but it can get your name linked to a big name project possibly.

Also, what you learn depends on what you think you may want to do. Web development, applications development, embedded, games. What do you want to do? Once you know that, check out some books on the topic from the library or subscribe to some magazines. Find some websites related to the subject. One of those will give you some insight into what you may want to learn. If you don't really care then you can always just learn all the buzzwords of the week.

Clubs and community involvement are usually nice to have. A minor in an unrelated field you enjoy can also show that you are more than just a computer guy. Foreign languages can be helpful too. Most of this advice I have heard from others, but it sounds sensible to me. Maybe something there will help.

Todd
