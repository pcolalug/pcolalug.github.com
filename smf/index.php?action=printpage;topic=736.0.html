Title: LTSP
Post by: bhoff on June 06, 2003, 05:27:27 pm
OK..OK I went ahead and browsed over the LTSP+OpenMosix information.
What you basically get is a master able to run whatever app you choose better than before how much depends on how many child processes are created to be migrated to the nodes, next is how many nodes.
while the Nodes get to be used as x terminals like normal with all there extra computing power being used by OpenMosix.
Here is the kick thou.
OpenMosix likes to have room to swap on each node....so to play nice you need like a 500meg hd on each node or prefered is 128meg ram much faster.


(NOTE!) here you bet much better response from 100 vs 10 on your network so be sure you are using the proper switch or better yet use dual cards in each node and get true bi-syn communication between each node.

Also note I am removing Mandrake from everything and putting RedHat 7.2  also gives performance boost....Really nice............

Title: Success!
Post by: on October 07, 2003, 10:05:14 pm
Well, I now have a 4 node cluster up and running, but I've noticed that
some processes such as apache and mysql will not migrate because they use shared memory.

It looks like OpenMosix is a good idea, but it still needs some work.

If separate Apache and Mysql processes would migrate across the cluster, this would mean huge performance gains for web and database servers, hopefully causing big corporations to give a lot more serious thought to Linux.

Now add the fact that a master node on the cluster can be set  up as an LTSP server and now you have all sorts of old machines which would normally be scrapped converted to X-terminals.

One machine to administer, one machine to install software on, one machine to back up. A client-server environment.

Configure the same master node to run the most common Windows apps using Wine, and you have a recipe that would bring Microsoft to it's knees.

An operating system tha can be made more secure than Windows will ever imagine...

An operating system that is more stable than Windows will ever be...

And you can run Windows apps on it...

The only drawback I see is the power bill for running all those computers...

Here's hoping that the OpenMosix, LTSP, and Wine groups get together and bring this vision to light...
