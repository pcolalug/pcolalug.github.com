Title: New Printer
Post by: danc on August 31, 2005, 06:21:49 pm
Hi all,

Just got a new printer yesterday, and I'm so happy with it that I wanted to pass on a quick review of it.  I picked up a Brother HL-5140 Laser printer.  It's a midrange type printer which holds 250 sheets of paper and prints at 21 ppm (which is pretty fast).

I got it working perfectly with CUPS, but had to do a little more work than I have in the past.  This printer wasn't listed in the CUPS database, so I had to get the driver and install it myself.  For those of you who have not been there, the site:  http://www.linuxprinting.org  is a wonderful place to get info/drivers for printers.  I downloaded the required ".ppd" file for this printer from there, and placed it in the /usr/share/cups/model directory.  After restarting CUPS, my new printer is now listed, and I could install it normally.  I actually have it attached to a Linksys print server on my LAN, and it works perfectly.  The print quality is outstanding.

It was also a breeze to set up for the couple of Windows machines that my wife and daughter use.  If you are in the market for a decent printer (that works well in both Linux and Windows), I would definitely recommend this one.

Dan

I just recently picked-up the Brother HL-2070N (specifically because it says "Linux Compatible" on their website).  

These brother printers are fairly nice, dunno how well they will hold up over time, but definitely easy to use.  I think this thing emulates about 8 different printers (some IBMs, a few HP LaserJets, PostScript, etc), and can appear to your system as a JetDirect print server, windows print server, IPP (CUPS) Print Server, LPD print server, etc.

So I'd say I also have to recommend the Brother laser's to any of you guys looking for a new printer.
