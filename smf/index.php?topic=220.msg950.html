Well, I installed from CD but I have also installed via FTP before, so that works if you are networked.

I haven't gotten around to getting my network card working yet, I need to compile a custom kernel.  But I'm using a theme off themes.org, so it currently looks like:



Runs fine, WindowMaker is pretty easy to use once you get used to it.  Since my laptop can only do 800x600 on the LCD screen, the fact that there's not taskbar on the bottom of the screen is helpful, leaves me with a little more room.

And it actually runs good.  Gnome and KDE were too slow to use really.  With Gnome / KDE running I had around 5mb ram free on average, with WindowMaker running I have like ~ 32mb free (80mb total system ram).
