This is cool. You need to try AmphetaDesk. It's an RSS agregator. (It collects headlines from sites that you customize. You install it and run it on your local computer and use it via your web browser. I read about it in my latest copy of Linux Magazine. Anyway, the link is right here. (http://www.disobey.com/amphetadesk/)

They even have a Windows version.

Anyway, for a quick install:

Make sure you have Perl installed.
Download the source tar file
at your prompt type "tar -zxvf amphetadesk-src-v0.93"
"./AmphetaDesk.pl"

It may differ slightly on your machine and setup.

When you run the .pl file you may get an error and it may not run. If that's the case, you need to install the perl extensions listed in the error.

Once it runs, it will tell you at the bottom where to point your web browser to use it.
