Title: Floppy disk image
Post by: on August 26, 2003, 09:27:11 am
I have a firewall on a floppy (for a dedicated firewall computer). I downloaded the floppy disk image and after formatting the floppy with:
fdformat /dev/fd0u1680
I then copied the image to the floppy using
dd if=/home/edward/Bering_1.2_img_bering-1680.bin of=/dev/fd0u1680
Then I booted it up on a computer to configure it with my settings and saved it. Now my question is how can I copy the customized image from the floppy back to my hard drive so I can save it to a CD. Just mounting it and copying it won't work and I want to have an image to make another disk should I need to. I would appreciate any help, thanks, Ed

Title: Floppy disk image
Post by: crowe on August 26, 2003, 10:17:59 am
This should be rather trivial with the cat command.

Try doing this, without mounting the disk first:

Code:
cat /dev/fd0u1680 > /home/edward/foo.bar


This should create a bit-for-bit image of whatever disk is in the drive, with the name foo.bar.

Or alternatively, you should be able to do a:
Code:
dd if=/dev/fd0u1680 of=/home/edward/foo.bar

and end up with the same results.

Either way, you should then be able to mount the freshly-made image via the loopback device:


Code:
mount -o ro,loop ~/foo.bar /mnt/misc


Assuming that the mountpoint /mnt/misc is valid, you should then be able to treat it like you would any other filesystem.  Check that all is good with the image, then unmount it and do whatever you like with it.  8)
