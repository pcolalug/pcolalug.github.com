Title: Re: Fedora Install trouble at disk 1

When you are getting a distribution, go through the distro's website to find approved mirrors.

For Fedora, see http://fedora.redhat.com/Download/

When burning an ISO, you shouldn't be asked about any boot disk or anything. It should burn the image onto a cd/dvd and that should be the end of it. I don't know what kind of junk you got. Delete the image and throw out the disk. Download your image from the link above and start over.
