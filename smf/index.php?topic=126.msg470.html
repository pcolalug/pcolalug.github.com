I've talked it  over with David and Steve and I propose to create a mod for phpBB that will allow us to interact with our "library" via these forums.  So far the specs are as follows:

phpBBLibrary_Mod would be a mod to phpBB, implemented as a new forum type,Library, and some associated additions to the admin console and user
priveleges.

1. use amazon api to tie isbn to image, title, description, link to amazon
2. users allowed to use library can check out materials that are available,
get on a waiting list if they are not, and see what they have and when its
due back. Can optionally receive reminders when something is due. Users with overdue materials cannot check out additional materials.
3. Admins can configure who uses the library, how long things are kept, how
and how often reminders are sent to users about returning/overdue materials.
4. Librarian (Library moderator) will confirm that each trasaction (check in
and out) actually took place before committing to the library.
5. Librarian can see at a glance what is due back and when and by who. Can optionally recieve periodic reminders when due dates are close.
6. Catalog can be broken into categories to ease browsing/searching.  Books may, however, appear in multiple catagories.

This thread is for you to post comments, sugguestions, and feature requests, as well as chime in if you want to lend a hand.  I tentatively plan to have this thread up for a week or so, after which time we'll have a feature freeze and start designing and coding the monster.
