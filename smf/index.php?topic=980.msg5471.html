Well, I just finished my install of Gentoo Linux yesterday. It was suprisingly easy since I now have a bit more than rudimentary knowledge of Linux.


I just have to say that Gentoo Linux is one of the most robust and all around pleasing distributions that I have tried yet. The portage feature makes the headache of installing packages go away, and I love all the little features


Anyone fed up with the crappiness of Red Hat or the other commercialized distros should defniitely give this one a try

One warning however, Gentoo is a bit more unforgiving during the install than your candy coated RH or Mandrake. Not quite as steep a curve as Slackware, but definitely not for the garden variety Linux user

Does compiling software really get in the way? I mean, yes, it did take me about 36 straight hours of compiling to get the desktop/java set up (USE flags are the shizznit). But once everything is up and running (say, after a weekend), I wait for nothing.

What I do is [ctl]-[alt]-[f2] over to a vtty and run
Code:
sudo `emerge sync` && `emerge -u --deep world`

then hop back over to KDE ([ctl]-[alt]-[f7], natch) and continue on with what I'm doing while portage takes care of the rest. Occasionally, a browser will scroll a little jumpy, but it really doesn't get in the way too much.

BTW, if you're curious, I run an Athlon XP 2000+ with 1024M of ram.

Well, it's been a productive weekend.  Just finished installing Gentoo, and I must say that I'm impressed (and kind of proud of myself!).  I did the Stage 3 tarball install, and so far I've got the base system, XFree, and Gnome (including Mozilla) running.  One little glitch I had was with the nVidia drivers - you have to have the "MTTR" flag enabled in the kernel for it to work, so I had to re-compile my kernel again to fix that.

Still have some to learn about the options for "emerge", but it sure is slick.  This thing boots up faster than anything I've seen, and the desktop seems very fast for everything.  Looking forward to getting some more apps installed.  All in all, it was not as hard as I thought it was going to be, largely due to the EXCELLENT online docs at the Gentoo website.  Just outstanding.  The whole install was fun, and a great learning experience.  If you've been thinking you should try Gentoo, go for it!
