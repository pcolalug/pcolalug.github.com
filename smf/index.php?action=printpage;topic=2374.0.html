Title: Network Services Setup
Post by: rugmonster on April 10, 2005, 01:02:54 pm
For the level of file sharing I do between my systems, yes I do need a domain controller of some sort. I don't want to put more strain on the WRT54G because they aren't very robust hardware and the software is lacking as well. I want to run DHCP/DNS because I like to be able to dynamically assign IPs and access those systems without having to know the IP. If I need to change my subnet, I change the DHCP scope and renew the IPs and it's all done. I don't have to worry about changing IP addresses and host files on all of the systems on the network. Also, a great deal of what I do is for educational purposes, so while I may be running a small network, my interest is in systems which can scale to very large environments.

I'll have to look into webmin. I haven't ever seriously looked at it as an administration tool, prefering command line and straight config files. Time for my more of my elitism to go away. Thanks guys.
