Title: xubuntu
Post by: Brent on July 03, 2007, 03:11:33 pm
I've been using xubuntu 7.04 for a good little while now.  It has all the good stuff I noted in my previous review.
(see http://www.pcolalug.org/smf/index.php?topic=2993.0)

I've noticed it's also fixed a number of things I found lacking in 6.10.  For instance, burning a cd now works  ::).  Also driver support for ivtv is now integrated into the kernel, but I imagine that's standard across all the buntu distros.

Finally, xfce now ships with it's very own compositor that handles 'real' transparency and other xgl/compwiz like candy, but it is much more lightweight than the main compositors.
