Probably going to embarass myself, but here goes.

The first # is not really by itself. It is hooked with the ! to make what I believe is called a "shabang." The shabang tells the script to start and coupled with your /bin/bash, it that it is a bash script.

The $ on line 3 is a string variable (as well as line 4) which is defined somewhere in some file which borders on religious faith since I only trust that it is actually there and have not seen it myself.

The final # may or may not be necessary. I do not know. In my thinking that comments out the line that says "end script" and is not read. But I don't know that for sure. It may be a required part of a shell script.
