Title: Re: Widget like OSX?
Post by: rugmonster on April 26, 2006, 08:36:33 am
You can probably do this command as root:

yum install gdesklets

Look at the gdesklets site for applets. One note on gdesklets: I've used them and the quality of the applets is sometimes leaves a lot to be desired. Like Konfabulator, they tend to be buggy, slow and memory hogs.

If you just want a system monitor app, look at gkrellm. Works with lm_sensors to get temperatures on your processor and motherboard and will also do monitoring of memory/swap/disk usage, network traffic, processor utilization, etc.
