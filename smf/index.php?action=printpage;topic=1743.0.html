Title: Nvidia 6106 driver
Post by: danc on July 01, 2004, 04:09:25 pm
For those who care, Nvidia came out with a new driver, version 6106 (yesterday).  I have been using an older one (version 4496) because it was stable and worked well, but I figured it was time to update...

Well, it installed OK, but at first running it failed with an error about permissions.  Turned out to be a PAM issue which is well documented in the README.  After some editing and chmod's/chown's it was working in 3D mode.  However, I noticed a decrease in OpenGL framerates as compared to the 4496 driver (from 3450 fps to 3330 fps).  This is measured using the "glxgears" test, which I know isn't completely accurate but is fine for direct comparison between two drivers.

So, I'm back with the 4496 driver, and not concerned with things.  As I said, this driver has always been solid for me, and have seen lots of people complaining about the newer versions (5336 and 5328).  Of course, I'm still using a 2.4.x kernel too, so you people with 2.6.x may have a different experience.  In fact, I'd be interested to hear what your results are if you try it.

Dan
