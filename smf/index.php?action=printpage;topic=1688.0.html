Title: Lindows with multiple disk drives
Post by: tuxmoose on June 07, 2004, 05:14:11 pm
I'm not a techie, but I am a somewhat experienced Lindows user (rburt's Lindows-guy-at-work).  

The Lindows installer provides basically two options: (1) take over an entire disk drive:  This will repartition/reformat the single disk drive selected with no option to take over multiple disk drives.  (2) take over an existing partition:  This option requires a pre-existing partition with no resize, etc. options.

Under the hood, Lindows is basically Debian Linux with the file system defaulted to Rieser-FS.  With Lindows installed on hda, you could use the standard command line (terminal mode) utilities to format/partitiion the other drives.  When that is done, there are a couple of techniques to auto-mount the other drives when rebooting.  Maybe some of you more advanced users can help fill in the details here...

The word in the Lindows (sponsered) user community is that Lindows 5.0 (built on the 2.6... kernel) will include a more advanced installer which will provide many more options for disk drive setup.  When is it due to be released?  No one knows for sure except it supposed to be realsoonnow... :shock:
