Title: Book review: PERL CD BOOKSHELF
Post by: bhoff on March 31, 2004, 11:32:27 am
O'reilly has done it again!

The PERL CD Bookshelf version 4.0 is what every Script writer, System Admin and Programmer needs to have in there
Library. It has a hardcopy of PERL in a nutshell 2nd edition and the following on the CD for your quick use.
PERL in a nutshell 2nd edition, Learning PERL 3rd edition, Mastering Regular Expressions 2nd edition, Programming PERL 3rd edition, PERL Cookbook 2nd edition, Learning PERL objects, references, and Modules, and a bonus section of related materials from O'reilly's Network, Boy what a find this is. I can honestly say this CD will replace all of my other PERL books and with the master search tool provided I can quickly find what I am looking for. I wish all tech books where like this. Also noted These materials also cover the newer PERL 5.8.
