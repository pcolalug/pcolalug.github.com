slack 8.1 dropline gnome 2.2
« on: February 15, 2003, 12:30:14 am »
I've been using Gnome 2.2 in Slackware 8.1 for about a week now.  And so far I really like it.  Besides some clean-up to the general user interface, they've made some enhancements to gnome that I really like.

They've cleaned up the fonts and themes.  Especially making them useful on hi resolution displays.  Previously I was pretty much maxed out at 1024x768.  But now I can use it 1280x1024 and 1600x1200 without problems.

Speed.  As they continue to optimize and tweak gnome, nautilus, mozilla.  The apps and desktop are getting faster and better.  Still stable as ever, no crashes so far.

Handy features, like CD Recording are built right into the desktop.  Open the file browser and browse to "burn:///".  Drag and drop files you want to put on CD, arrange them however you like.  Then click on the Applications menu and select "Burn CD".  (You have the option of writing the files to CD or creating an ISO image file).  So simple.

They've made the move (I think RedHat started it) to abstract program names.  Instead of "OpenOffice Writer" you have "Word Processor".  Instead of GAIM you have "Instant Messenger".  Insetad of "Mozilla" it's just "Web Browser".  Again, this is a feature I really like and personally the way I think desktops should present software.

They've integrated more apps which I had never used before but really like.  A media player called Totem which plays back audio, video, vcd's, and DVD's.  (Also XMMS when you need just a straight audio player).  

They've included everything needed for most people to be productive, and left out multiple programs.  (You get one web browser, not Mozilla, Netscape4, Netscape7, Galeon, and Phoenix).  

The Gnome2 integration with recent versions of Wine is nice.  (I think it works with KDE too but I can't say for certain since I don't use it).  When I install a windows program in wine, it gets integrated into my start menu and windows desktop shortcuts show up on my gnome desktop.

Nautilus desktop handles mounted volumes correctly.  When I mount a filesystem (CD, Zip Disk, Floppy, etc), an icon to the device appear on my desktop.  It gives you an easy GUI based way to browse the device, unmount, eject, format, all the other standard stuff.

There's now (I never noticed one before) a "Recent Documents" menu integrated into the menu system which tracks files opened through nautilus and allows easy access.
