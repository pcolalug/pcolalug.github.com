Title: SwiftFox
Post by: Brent on June 19, 2006, 10:59:31 pm
We all know how those unwashed gentoo hordes are aways raving about the speed up you get from compiling software for your specific arch.

Well, I ran across an optimized build of firefox called swiftfox.  It's built off the same source as the standard release, and all the extensions I've tried work as well on it as they do on firefox.  You can find it http://getswiftfox.com/

Let me know if you notice any real speedup.  I have, but that could just be suggestibility talking

Title: Re: SwiftFox
Post by: Brent on June 24, 2006, 12:59:40 pm
Ok, perhaps it was my goof.  When I tried running it in slackware it complained of a pango-cario dependency.  As long as you have that installed, you should be fine.  However, it is NOT part of the default slack install, so I'm guessing you'd need gnome to satisfy it.

As I previously reported though, firefox was just as fast on slackware as swiftfox was on ubuntu so it may mainly be useful for the more heavyweight binary distros.

Also, regarding the net tweaks, there's a extension out now called fasterfox which will change all those net speed settings for you.  I recommend setting it to one above normal, as the highest setting actually seemed to slow things down for me.
