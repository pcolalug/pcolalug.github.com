Title: Re: javascript editor
Post by: rugmonster on January 11, 2007, 09:33:36 am
I use Eclipse with the PHP IDE addon from Zend (http://www.zend.com/pdt). It support Javascript in addition to PHP. For a non-free (as in speech), but free (as in beer for personal use) editor that you may want to try out as well, there is JSEclipse (http://www.interaktonline.com/Products/Eclipse/JSEclipse/Overview/), also an Eclipse addon. I've used both, but the PHP IDE seems to meet my needs more since my focus is more on PHP and less on Javascript.
