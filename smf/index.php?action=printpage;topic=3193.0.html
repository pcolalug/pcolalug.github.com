Title: Ubuntu root password
Post by: sandspur on July 02, 2007, 09:30:33 am
Just did my first install of Ubuntu 6.06 on a laptop.  Relatively painess.

In the process I was prompted to make a user name and password but not set a root password.

Subsequently learned that by default there is no set root password.  With a little Googling I learned that setting the root password is pretty easy.

In a terminal type:

sudo passwd root

You will be prompted for a password.  Type in your user password.  You will ger a prompt:

 Enter new UNIX password:

Type in your desired mind-boggling entropy-heavy root password and hit enter.

You will be prompted to type the password again.  Do it.

Neato!  Done deal.  Now I can really do some damage.
