About Firebird and mailto links. I remember I used to have this problem, but don't now. (I am using Windows right now due to not being able to get on the net in Linux with my notebook without a LAN connection, so I can try it this next week in Linux).

I am currently using Thunderbird mail as my mail client. And it works fine. I don't remember if it worked with OE or not. Thunderbird may be the answer to having Firebird work correctly. But, as much as I like Thunderbird as a whole there are some things that I think might keep people from using it at the moment.

I love the way it handles multiple accounts, but this is seen as one of it's weaknesses. Each account has it's own set of folders (Inbox, Sent, Deleted, etc.). For me that is great. I can have one user profile set up and have different accounts for my wife, son and myself. Each one of us can just select their account from the list on the left and see all of their own mail. With OE, I had each person's mail highlighted with a different color because OE just dumps everyone's mail into one folder (ie: one Inbox, Sent, Deleted, etc. for everyone). But you still had to dig through someone else's mail to find yours.

Therefore, I like the multiple accounts being separate. But that is one of the complaints that others don't like. It should be an option to handle it one way or the other instead of making you use it just one way. For those who think this way, I would say that OE does not give you a choice either. Both clients will let you set it up the other way, but only by going through message rules and telling everything where you want it to go. So it can be tweaked to play the way you want. To me, I like the way Thunderbird does it.

That is way off topic, but it does answer the question about Firebird and mailto links (at least for Windows at the moment).
