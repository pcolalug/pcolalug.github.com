Post by: Skreyola on December 29, 2007, 12:04:19 pm
A lot of DVDs are encrypted using CSS (not to be confused with Cascading Stylesheets). To view them on your computer, you need to download the libdvdcss package. It's not included in Debian or Ubuntu repositories, if memory serves me, because of copyright issues (various countries have differing laws on such things as fair use), but you can download it directly using this command at the command prompt:
Code:
$  wget http://download.videolan.org/pub/videolan/libdvdcss/1.2.9/deb/libdvdcss2_1.2.9-1_i386.deb
Since you're running Ubuntu, you can download the deb package. Other distros would get the tar.gz and compile from source. (There's an RPM package for RH users)
Then either become root and run:
Code:
#  dpkg -i /path_to_deb_file/libdvdcss2_1.2.9-1_i386.deb
or if your system was set up without root (common with Ubuntu):
Code:
$ sudo dpkg -i /path_to_deb_file/libdvdcss2_1.2.9-1_i386.deb
Making sure to change the path to wherever you were when you downloaded the deb file. If you run the sudo command in the same directory, you can omit the path portion, of course.

I didn't want to waste your time with stuff I think you probably know, but if you need help on getting to a command line, just post a reply and one of us will walk through the steps.

For burning CDs/DVDs, the standard seems to be the X CD Roaster. If you've tried this and are having trouble, post the error messages you're getting.

Watching commercial DVDs on your computer is fair use, but downloading unlicensed copyrighted material is, of course, illegal.

As far as watching Flash(tm) files, I just use my browser and the flash plugin, so I don't know anything about playing them in a standalone media player.
