Title: Nifty utility
Post by: sandspur on May 07, 2010, 07:38:29 am
About to take possession of a new netbook, an HP 210 mini.  Comes with Windows XP.  I intend to install Fedora 12 on it.

Like most netbooks size prohibits a built-in CD/DVD reader/writer.  Unwilling to spring for the money for an external USB reader/writer I'm going to try to do the installation from an installable live ISO on a USB memory stick. 

Stumbled upon a cool utility for getting the ISO on a memory stick:  liveusb-creator.

Installed it via yum install liveusb-creator.  It is also available for Ubuntu with apt-get.

You download the ISO to the directory of your chioce first.  Then you run the utility and see a nice GUI.  You select the ISO source and then give it a target:  The USB stick.  Very painless.  Even does a CHECKSUM of the ISO for you.

Booted up a laptop with the live Fedora version.  Will try to install from that point with the new toy.  Will keep you posted with my experiences in the next couple of days.
