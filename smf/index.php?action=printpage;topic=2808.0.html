Title: Wireless Networking Guide 2006
Post by: rugmonster on March 05, 2006, 11:50:54 am
Tom's Networking (a branch of Tom's Hardware), has a great article entitled Wireless Networking Need To Know 2006 (http://www.tomsnetworking.com/2006/02/27/wireless_networking_ntk_2006/) that gives a overview of all of the current and upcoming wireless technologies. Mostly vendor neutral with some great pros/cons and recommendations for each technology. Well worth a read if you are planning on starting a wireless network or looking to upgrade soon.
