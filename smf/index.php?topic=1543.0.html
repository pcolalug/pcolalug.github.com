I've got it running, posting from it now.  SSCCHHWWEEEEEEETTTT!

It seems very fast, and very pretty.  The new "spatial" mode of Nautilus will take some getting used to...  Did not have to reinstall my nVidia drivers, and this X.org seems to be using the same XF86Config file as always.  Don't really know much about X.org, or it's differences, but so far seems to be working fine.  All the apps are very much updated and work fine.  Don't know whether it's the X.org, or Gnome 2.6, or the (new) default Dropline theme, but it just looks fantastic.  I did have to reduce the speed of my mouse a little, but no other changes have been needed.  I am still using a 2.4.22 kernel on this too, BTW.

I did have one problem during the install.  The new Dropline installer requires itself to be updated before anything else can be updated (nothing new there).  After I did that, it removed the little "auto-indicator" icon in my tray, and said to run "dropline-installer" to proceed.  So I tried, but could not open a terminal.  Could not seem to open anything, in fact.  So I ctrl-alt-F6'd to a console, logged in as root, and rebooted.  Uh-oh.  System would not come up, was hanging at the "Starting up X11 session manager" point.  Gave some error message about a missing file called libiconv.so.2  and just stopped right there.  Well..........., luckily I have Slack on my laptop too, and found that file living in /usr/lib/.  After booting the main system into Mandrake, mounting my Slack partition manually, and scp'ing the missing file over from the laptop, it rebooted fine again into Slack and I proceeded with the rest of the Dropline upgrade.  Moral of the story:  might not be a bad idea to copy that file somewhere else before you start in case it gets eaten like mine did.

Anyway, this beast is looking good, and I recommend Slack + Dropline to anybody.  With the way things are going in the Mandrake world (update mirrors all screwed, Cooker updates accidently mixed into Community updates, etc...), I'm just about fed up enough with Mandrake to dump it for good.  With this new development in the Slack world, gonna be a tough choice between this and Gentoo for my everyday machine...

Dan
