Title: Choosing a new distro for an old box
Post by: dpeach on March 03, 2004, 07:04:04 am
I believe the challenge of Linux is in the setup. Once you get it running, and you do not change the hardware, you should be fine.

The reason I would recommend Mandrake, especially to someone installing it themselves for the first time, is that it seems to handle most of the setup for you. One of the reasons to run a distro like Slack is that you don't want 50 drivers loaded for your one video card. Slack is much more streamlined than Mandrake. The "fatness" of Mandrake is due to it's wanting to support everything it can the first time around.

You will learn alot about Linux if you run Slack. You could learn about Linux if you run Mandrake. Mandrake for me just works. I would recommend it to anyone who is interested in just using their machine. But running Mandrake will not force you to learn all the intricate details of Linux unless you learn them on purpose. Running Slackware will give you much more opportunity to learn how Linux really works.

I have never run Slack, but I do run Gentoo on one machine. I learned more in the first 2 weeks about Linux with Gentoo than in the 6 other years of Linux use. That may be a bit of a stretch. But the point is, Slack will be alot more work to get going, but you will probably enjoy it tremendously if you are interested in learning and tweaking. Mandrake is a little boring, but you certainly can get alot of work done without knowing the details.

My 2 pennies.

Title: Choosing a new distro for an old box
Post by: rburt3 on March 03, 2004, 08:20:36 am
You can also address performance issues by the desktop packages you choose to run. Instead of running kitchen sink desktop environments (KDE/Gnome), try one of the lighter packages such as XFCE (http://www.xfce.org) or Fluxbox (http://www.fluxbox.org). You won't get a lot of the bells and wistles with these, but you will get a noticable increase in performance.

Also, lloydr is right in that the newer kernels would probably give you a boost in performance, but again, you'd want to configure it to be lean.

Ultimately, I'd reccomend Gentoo, Debian, or Slackware for older hardware. However, from the specs you mentioned, you shouldn't have any problems running any modern distro. I was running Red Hat 8.0 on a PIII 650 for almost a year with no problems.

If its an XFree config problem, just bring the box as-is to the install fest and we should be able to get you up and running, barring really bad video card.
