Title: Dropline Gnome Update: Sept 16 2003
Post by: imnes on September 16, 2003, 09:24:21 pm
Gnome 2.4 (http://www.gnome.org/start/2.4/) is out, and that means a new installment of the ever popular Dropline Gnome (http://www.dropline.net/gnome).  

One thing I've noticed immediately, the rounded corners of the 'menu panel' are gone.  That's because gnome no longer differentiates between all the different panels, there's just one generic type that can be made to handle anything, and put anywhere.  So that's ok, speculation is a theme / applet will be released soon to give people their rounded corners back, but I don't miss it.

The interface has been further refined and cleaned up.  New icons (look like they're going with Ximian's icon set instead of RedHat's).  Mozilla, Firebird, and the rest of the browsers have been replaced with the very fast and nice looking Epiphany browser.

Gnome now has improved "Accessiblity" support, including an on-screen keyboard, a magnifier, and a screen reader.  That's very cool.

Otherwise, all the standard applications we're used to having in Dropline are still there.  Some of them have cleaned up icons, and everything feels more unified.

One thing you'll notice (or I notcied) is that gnome, overall, seems much quicker in this release.  Nautilus has seen some visual improvements, the interface has changed a bit, and it loads and works quicker.

The Screen Preferences app is back, so you can change your resolution in X (really change it, not just CTRL+ and CTRL-) without having to exit and reconfigure and restart.

I'll post more as I play around with it.  If you're into slackware, give it a shot.
