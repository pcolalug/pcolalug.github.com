Title: File Attach Feature
Post by: steve on October 29, 2002, 12:58:45 pm
I've been thinking about this file attach feature for the forum. Do we really need it?

1. It still isn't going to allow us to embed pictures in our posts. (I haven't seen any forums where you can actually embed the pictures in a post. All of the pictures I've seen are actually linked from other sites)

2. Due to bandwidth considerations, the file sizes would have to be limited to less than 1MB.

3. File attaching would have to be limited to certain people. I don't want the law comming in and busting me because we have illegal files in our database.

4. What sort of files, meeting the above criteria, do we need to post on this forum?

Steve
