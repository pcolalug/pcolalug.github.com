Alright after trying many many different distros (I actually liked Vector quite a bit.):

I did like Puppy Linux and DSL also, but the graphics kept messing up on both.

I decided what I need was a heftier system on the back end, with a very light windows manager on top. I
ended out with Xubuntu, installing icewm and rox as the desktop/windows manager.

So remember these guys have never used linux before, so I downloaded a silver xp like theme for icewm, and then put a link for pidgin (which includes a myspace im'r, which was important to them.) I also put a link to firefox on the desktop and a link to dillo on the panel. Realistically dillo is just not enough many times. I also put a text file on the desktop with many useful commands.

So I realize that many of you are going to think, "Why didn't he put slackware on there?" or "Why not use tiny[insert distro name] on there, it would have been faster?"

The answer is that I wanted something that I did not have to spend the rest of my life configuring, and I wanted apt-get. I love that tool, and I would be lost without it. Really, I would just wander in circles.

The install was very easy, and once I got Ice-wm installed the hardest thing to do was get a desktop.
I tried to use idesk first, and I could not get it to start with the wm, so I got frustrated and looked for something else, eventually I got a link to a great guide on Icewm, and it suggested Rox. He also gave me a great startup script for it. I love plug and play. I did that, and boom. I was up and running.

I would add that I am using the nm-applet from ubuntu for network control. That configures itself automatically.

Other than firefox, everything is set up to open a small weight program first, with the option to go with a heavier program. The idea is to use low ram programs if at all possible.

Anyway. I'm gonna present it to them, and show them a few things on the computer on Sunday. If anyone has any suggestions on it before then, let me know.
