Post by: danc on June 22, 2004, 09:32:46 pm
Harold,

A followup - I was bored, so installed Vector Linux 4.0 on a test machine.  For a 224Mb ISO, it's very nice.  Some apps are a little dated, and it uses kernel 2.4.22, but works great.  The install is easy, and similar to Slackware.  If you want to use X, it comes with Xfce, Fluxbox, and IceWM window managers only.  I had forgotten how wickedly fast these WM's are (as compared to Gnome/KDE).  Truly amazing.  I think this one's gonna keep a spot on the test machine for a while.  For an older machine with limited horsepower, I think this distro would be hard to beat.  Give it a look!

Dan
