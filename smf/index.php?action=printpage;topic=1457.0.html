Title: et "out of range"
Post by: imnes on March 05, 2004, 10:02:16 am
Okay this took a lot of googling and experimenting.  Assuming you are running your desktop at 1024x768 16-bit 60hz and it looks okay, this will run your game in 1024x768 and not switch refresh rates.

Create a new file ~/.etwolf/etmain/autoexec.cfg

It should contain:

Code:

seta r_displayRefresh "0"
seta ui_r_displayRefresh "0"
seta r_mode "6" unsafe
seta ui_r_mode "6"
seta r_depthbits "16"
seta r_texturebits "16"
seta r_colorbits "16"
seta ui_r_depthbits "16"
seta ui_r_texturebits "16"
seta ui_r_colorbits "16"


Save the file, start your game.  Let us know if it works.   The biggest key is to match values that you are using for your desktop.  If you are in 24 or 32 bit, change the depthbits options accordingly.  If you are in a different resolution, change the mode options to one of those shown below.

Some more details for those interested:

Set those to whatever you like. Valid modes are below.  Just match your desktop.
Quote

 // Mode  0: 320x240
// Mode  1: 400x300
// Mode  2: 512x384
// Mode  3: 640x480
// Mode  4: 800x600
// Mode  5: 960x720
// Mode  6: 1024x768
// Mode  7: 1152x864
// Mode  8: 1280x1024
// Mode  9: 1600x1200
// Mode 10: 2048x1536
// Mode 11: 856x480 (wide)

 
Might also have
Quote

seta r_fullscreen "0"
seta r_fullscreen "1"


in which case you could get the game working in windowed-mode at least.
