Title: PCLinuxOS-2007 Released
Post by: djoed_pcola on May 24, 2007, 08:05:05 am
The official release of PCLinuxOS-2007 is now available.

Last night, I downloaded the iso, burned the CD, and installed on an older Dell PIII 866 machine with 512 Meg of memeory.  The install went smoothly and quickly.  I have only briefly experimented with this distribution, but I have to say that I am impressed so far.  "Out of the box", it was able to play a Windows Media video from cnn.com.  Ironically, my Windows XP Pro machine currently is unable to play a Windows Media video.

If you haven't tried it, and you have 20 minutes and a blank CD to spare, take it for a spin yourself.
