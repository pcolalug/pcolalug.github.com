2  Hardware / Network / Wifi Success with Linspire (aka Lindows) on: November 26, 2004, 11:31:42 am
Success!!!!!
I E-Bayed an Orinoco Wifi (802.11b) card.  It is a Lucent card re-branded by Cabletron.  Shipping & handling came to a total of $32.99.
It is connected to my home network via a Linksys WAP-11 (802.11b wireless access point).

Lessons learned:  
1.  I normally run my access points with 128 bit encryption.  This older card only supports 40-bit encryption, so I had to select 40/64 bit encryption on my access point.  
2.  Linspire correctly identified the card, but the Control Panel software does not expose all of the significant configuration elements needed.   Recommend that you use the control panel to set channel number and to set the mode to AD HOC rather than using the "Automatic" options.
3.  Using iwconfig in a console window, I had to manually set the RTS and FRAG parameters.
4.  After having done all of the above, nothing worked until I rebooted--then everything lit up!!!!.

8  Distributions / Debian Based Distrtibutions / Lindows with multiple disk drives on: June 07, 2004, 05:14:11 pm
I'm not a techie, but I am a somewhat experienced Lindows user (rburt's Lindows-guy-at-work).  

The Lindows installer provides basically two options: (1) take over an entire disk drive:  This will repartition/reformat the single disk drive selected with no option to take over multiple disk drives.  (2) take over an existing partition:  This option requires a pre-existing partition with no resize, etc. options.

Under the hood, Lindows is basically Debian Linux with the file system defaulted to Rieser-FS.  With Lindows installed on hda, you could use the standard command line (terminal mode) utilities to format/partitiion the other drives.  When that is done, there are a couple of techniques to auto-mount the other drives when rebooting.  Maybe some of you more advanced users can help fill in the details here...

The word in the Lindows (sponsered) user community is that Lindows 5.0 (built on the 2.6... kernel) will include a more advanced installer which will provide many more options for disk drive setup.  When is it due to be released?  No one knows for sure except it supposed to be realsoonnow...
