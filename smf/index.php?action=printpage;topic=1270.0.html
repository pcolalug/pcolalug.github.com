Better take a look at that Grub bootloader section above.  Looks like you've got an extra "zero" on the Kernel line.

"Kernel /vmlinuz-2.4.20-80"  should be  "Kernel /vmlinuz-2.4.20-8"

The file you need to edit is called either "grub.conf" or "menu.lst", and is located in /boot/grub.  Perhaps the improper shutdown added the extra character or something.  Just edit that file and reboot.  Let us know if it works...
