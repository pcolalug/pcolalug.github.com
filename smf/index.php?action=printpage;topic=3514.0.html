Title: Linux friendly laptops

Based on past experience, I have found that laptops tend to be the least friendly area to Linux.
I recently got a newer laptop and found that Ubuntu 9.04 works perfect with no need to download or install or use any hardware.
I got a Compaq CQ60-211DX, it has a single core Celeron 585 @ 2.16GHz, 2GB DDR2, Intel 4500M video, 160GB hard drive, 15.6" widescreen, and a full keyboard with the full numpad to the right side as well.

So I had an excellent experience with this laptop, anyone else have others to add to the list of linux friendly laptops?

Title: Re: Linux friendly laptops
Post by: danc on May 10, 2009, 10:11:04 pm
I've had great results with this Dell XPS M1530.  Core2Duo T8100 CPU's (2.10Ghz), 4GB RAM, Nvidia 8600M GT video, 200GB 7200rpm hard drive, 15.4" widescreen, wired and wireless network, Intel HD audio, built in webcam and microphone, DVD R/RW burner.  All working perfectly with Slackware 12.2.  Also, even the hardware buttons for volume control, CD play/eject buttons, and screen brightness work.

One thing I haven't gotten working is the "5-in-1" card reader slot (for camera media, etc).  I think it's just a matter of getting a proper driver for it ("sdhci"), but I haven't really needed it so haven't bothered.

I really like this laptop and recommend it.
