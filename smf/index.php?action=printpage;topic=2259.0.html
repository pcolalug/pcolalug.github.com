Title: Slackware 10.1 released
Post by: dpeach on February 08, 2005, 04:27:19 pm
I have it up and running (nvidia drivers installed as my second official act, Firefox was the first). Having some LILO issues and can't get back to my main install. Maybe this will become my install of choice for lack of choice.

So far I like what I see. On the one hand I don't like when distros highly customize the desktop environment so that you have trouble telling which window manager you are using. Slack has done a good job with this. This is just straight KDE from what I can tell.

On the other hand, I do like when a distro cleans up what is installed by default and not give you every program known to man on the menu. Just the ones that Slack installed (I realize that many of these are KDE specific tools) I have 105 programs on my launch menu. This, of course, will not include most all of the command line tools available. 6 different media players!! And mplayer is not one of them. Not that it has to be, but Goodness Gracious! Why not!? If you are including all the others, what is one more?

And no Firefox by default? What is up with that. The most popular OSS browser to come along and at a time that it is so popular. Seems like an odd thing to leave out.

I don't want to sound too negative, but it is easier to say the few things I don't like rather than go through the whole list of things that make me say "Wow!" Since it is Linux, I am getting over that "Wow!" stage. I am very impressed with where Linux has come to. A tremendous change from several years ago.
