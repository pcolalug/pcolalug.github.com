Title: Re: Linux .wmv player
Post by: danc on February 20, 2010, 12:07:06 pm
Mplayer has always worked well for me.  There are various codec packages around that might be required, you would need to search the Fedora repositories for something like that probably, although there may even be some codecs on the Mplayer homepage.  I know that there is also a browser plugin that enables watching "embedded" .wmv's on webpages.

Title: Re: Linux .wmv player

Title: Re: Linux .wmv player
