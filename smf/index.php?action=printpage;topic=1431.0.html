When I was having problems with a USB to RJ-45, I booted up Knoppix. With Knoppix running the card/adapter worked fine. I was able to see what driver it was using and any other information I needed. I then went back into Mandrake and configured Mandrake to use the same information.

Sorry that I cannot help you with more information than that. I had someone holding my Linux stunted brain through this process. He was able to tell me the commands to use to get the information I needed.

Give Knoppix a shot. If it works in Linux, it will probably work with Knoppix.

Title: Linux (2.4x) + Wifi+ USB = 0

The word in the Lindows/Linspire community is that Wifi via USB is a lost cause for any pre-2.6 kernal, and not a sure thing even in 2.6.  While 802.11b is readily doable, 802.11g is only for those with a great deal more time and luck than I'll ever have.

Just get an Ornioco card PC-Card (PCMIA).  They simply work.
