Title: Making my computer a wireless bridge.
Post by: OmnipotentEntity on July 04, 2007, 09:18:51 am
I'm getting a new, auxillary computer for to make a media slave out of it.

I was planning on broadcasting a low power radio station out of my house so I don't have to run speaker wire everywhere.

The plan was to put it in my room near my workstation, and I have a wireless card now set up on that station.  I now have, as a result, an unused NIC card on my computer.

So I was planning on running some crossover from my workstation to my media slave and I was wondering if anyone knew off the top of their head how to turn that interface into a network bridge so that anyone can change the song playing from anywhere within my home network.

Title: Re: Making my computer a wireless bridge.
Post by: rugmonster on July 04, 2007, 07:53:42 pm
You'll need the bridge utilities first to give you brctl. Once you have that, follow this:

Code:
# brctl addbr br0
# brctl addif br0 <wirednic>
# brctl addif br0 <wirelessnic>
# ifconfig <wirednic> 0.0.0.0
# ifconfig <wirelessnic> 0.0.0.0
# ifconfig br0 your.ip.address.here
===or===
# dhcpcd br0
===or===
# dhclient br0
===or===
# your-dhcp-client br0

That's all there is to it. Explanation of the above here (http://linux-net.osdl.org/index.php/Bridge). By the way, very cool and amazingly obvious idea.
