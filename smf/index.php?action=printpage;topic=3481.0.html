Title: LXDE
Post by: Yossarian on February 27, 2009, 09:23:14 pm
Lately I've been using the Lightweight X11 Desktop Environment in Ubuntu.  I must say, for a "lightweight" DE it's quite good, not as quite as light as fluxbox, but much more functional.  It resembles a stripped down KDE 3.

The main reason I've been using it is because games run much smoother under it, compared to when they run under GNOME with compiz-fusion.  I've tried a program called "compiz-switch" but it doesn't work worth beans.

It's available in the synaptic repositories, and is about a 30 mb download.
