Anyone have a modem and drivers that will work with Ubuntu ??

It's been several *years* since I tried to mess with a modem... but, as far as I know, nearly all internal modems these days are "win-modems" which rely on Windows software to function.  Some of them can be coaxed into working under Linux, but it's not a fun road to travel.  The best solution if you're stuck with using a modem is to use an external modem, which relies only on the serial port drivers (built in), and whatever app/dialer you're going to use (such as 'ppp').

If you can, just skip the modem stuff entirely, and get broadband access.  You'll be glad you did.

Title: Re: modem for linux
Post by: danc on March 07, 2008, 11:52:10 am
It's been a LONG time since I tried a modem, but any external modem should work fine since it depends only on the built in serial port drivers.  Of course you'll need an application or dialer (such as minicom or ppp) to get connected to anything.  As far as brands, I always had good results with USRobotics modems.  I would stay as far away as possible from anything using USB.  Good luck.
