36  Club Stuff / Introductions / Re: New Member on: May 18, 2009, 05:43:27 pm
In linux or unix (linux is a flavor of unix) you have an administrative account called "root" or "superuser" and user accounts similar to administrator and regular user in winxp. Linux users login with their login name to an account that does not give the user administrative privilages. That is part of the security model linux uses. As a regular user you can't change or delete system files. Root is the name of the superuser account.  When login in as root you can do anything to your system even destroy it if you don't know what you are doing. I don't run ubuntu but I think the root account is disabled by default and to do administrative work you have to log in with a command called sudo. To learn more about Ubuntu please visit there forum where you can get better information than I can give you. A distro is short for distribution. Linux operating systems are called distributions. They are all basically the same but at the same time can be very different in respect to looks, desktop managers and included software packaging systems. Here is a link to the Ubuntu forums. This is the starting page that has a wealth of information for you. There is a learning curve to linux like all things so don't feel overwhelmed by it. Just start and learn at a pace that fits you. Ubuntu is user friendly so if you use winxp you can run Ubuntu just as easily.
Here is the link http://ubuntuforums.org/
