After much battling and compiling I"ve finally hit on a good combination.  Latest wine from codeweavers, combine with Win ME dll's, and RedHat 8.0, I can run windows apps in linux.

Here's a few I've already tested:
Kazaa
Startup Window
Searching for Stuff
Download Manager

Also playback (at least of an mp3 file) works great.

Also tried out Windows Media Player.  Worked fine for audio.  I tried a mpeg video file but colors were screwed, I didn't have any more to try with so I dunno if it was just that file, or in general.
Media Player in Classic Skin


I don't plan to run Media Player or IE or Office from Linux since it has (IMHO) better versions of all of them.  But being able to run Kazaa, and a few other apps will pretty much free me from windows.  The next one on my list to get working is QuickBooks.  Anybody have any more suggestions?  What's keeping windows around for you?

BTW: If anyone is looking for a cheaper alternative for MS Office in Windows check out Ability Office.  Seems to be feature compatible with MS Office XP, but is smaller and faster, and cheaper.  Even the database program in it handles native MS Access files and SQL Servers without a problem.  

You can download and try it free before you decide if you want to register.  I really like it though.  You can check it out at http://www.ability.com/sales/products/office.php

Seems to be a more complete product than openoffice in windows, and may be the least painful way to wean someone off a MS Office addiction.
