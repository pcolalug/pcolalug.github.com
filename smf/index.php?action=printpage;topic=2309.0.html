Title: Linspire Coupons
Post by: imnes on March 05, 2005, 04:30:21 pm
Saw on deviant art a coupon for a free copy of Linspire.  Don't know if it's still valid but if anyone's interested in trying:

http://news.deviantart.com/article/15965/

Says to visit linspire.com, put a copy of Linspire in your shopping cart (should cost $49.95), then apply the coupon code deviant9 which will apply a $49.95 discount.

Keep in mind though that will give you a basic OS install of KDE, OpenOffice and other desktop applications.  Linspire also has their CNR service (Click and Run) which is subscription based $49 a year.  There is a 2 week trial offer going on for that.  CNR is the way you usually add software to a Linspire system.

There is a 3rd party tool that does the same thing as CNR (but it's free) and supposedly going to work with all debian packages, it's called klik (http://klik.atekon.de/) and works in Linspire 5.0 (not yet available), Kanotix, and Simply MEPIS.
