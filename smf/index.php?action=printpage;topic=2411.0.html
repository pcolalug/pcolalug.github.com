***Ant: The Definitive Guide, 2nd Edition
Publisher: O'Reilly
ISBN: 0596006098
A must-have for Java developers, this second edition has been reworked,

ISBN: 0974514071

http://www.oreilly.com/catalog/1932111956/index.html
