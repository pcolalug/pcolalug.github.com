Title: Re: Linux Flash Player 9 Available
Post by: dpeach on November 09, 2006, 10:06:52 pm
Alan, I had the same problem. I fixed it by removing any links to Flash Player 7. This was not easy as it was snooping around and finding flash player 7 in all kinds of places.

Go to about:config in the URL bar. Then do a search for: plugin.expose_full_path   Change that to true. Then go to about:plugins and see if there are any references to the Flash Player 7 (or any other version). Remove it from wherever it says it is installed and then start over with 9 and see if that helps.

Hope so. It works fine for me now.
