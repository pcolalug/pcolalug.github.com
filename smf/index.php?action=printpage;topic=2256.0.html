Post by: lloydr on February 06, 2005, 04:20:22 pm
The problem would seem to be with the filters without going farther into it.

The line going to the DSL modem does not need a dsl filter and could possibly be one of the main parts of the issue. put the dsl filters on all lines going to your phones/faxes/etc...basically anything that uses a phone line needs a filter omitting the dsl modem itself.

hopefully that helps.
