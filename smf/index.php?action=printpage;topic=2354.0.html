Hello, realized I hadn't introduced myself yet.

My name is Michael Greb and I've been using linux for over 10 years.  I started in late 93 when the kernel was still pre 1.0, 0.97 if I remember correctly, running Slackware Linux.

Today I run Ubuntu hoary on a dual head desktop and a ppc iBook.  In addition, I run Debian sid on my Xbox, which functions as a MythTV frontend, and a colocated server hosting my personal website, email, and a few websites for bussinesses and friends.

I'm fairly active in the open source community with contributed code in MythTV, the xbox-linux project's kernel, and hopefully soon Asterisk.  I am a contributing member to Software in the Public Interest, the legal parent of Debian, and have been staff for the Open and Free Technology Community for a bit over a year now.

I work for Linode.com (http://www.linode.com/), a User-Mode-Linux based virtual hosting provider.  There I respond to customer trouble tickets, admin 44 co-located servers, write admin automation and business process software for handling day-to-day business and occasionaly preform forensic analysis of rooted customer's machines.
