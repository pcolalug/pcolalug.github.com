Has anyone seen or used this program? Apparently it is part of KDE, but I had never seen it before. I came across an article for it in PC Magazine (Spanish). I don't know if it was in, or will be in, a recent issue in English.

You can get it at their Sourceforge site.

It is a tool that graphically tells you how much space a certain file or directory takes up. Run as a user and it will just show you that user's info. Run as root and you can see the whole drive. You can also have it just give you information about a particular directory. This is helpful in being able to see how much space is used up by a program or folder. If you are looking to clean up your drive, you can see where big chunks of information is stored.

Here is a shot of it in action. Click for a larger image.



The small red box in the bottom left corner is highlighting the directory I have selected above. Each of the boxes in the bottom half is a file. You can see that some are very large and some are smaller. If you click on a box below, it will take you to the file above where you can see information about the file or even delete it.

Just something that I found that I thought someone might be able to use. For me, it has helped me see that I have some really big files that could be eliminated when I need to free up some space.
