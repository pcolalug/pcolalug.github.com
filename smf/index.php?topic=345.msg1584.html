Quote from: martext

Bah...there is plenty you can do with a 120! My first linux box was a 486DX4/100 with 32 MB RAM. That thing was a great Slackware box with Apache and WU-FTPD. I also ended up setting it up as an IPFWADM and IPMASQ firewall/connection sharing box (dialed to the ISP for me). Then it became an IPCHAINS system and was then upgraded to a K6-266 box with 96 MB of memory. I threw in a trident video card(w00t) and was able to run X finally. I ended up using WindowMaker and pretty much used X to have a bunch of terms going on multiple desktops. I ran that box was used as my proftpd server and iptables firewall until just recently when I aquired a K6-2/500. When I get back from the desert (see my introduction), I plan on building the old K6 up again for the firewall agian and then using the k6-2 as a second desktop box...maybe even experiment with making a small 3 box beowulf cluster...haven't decided yet. Anyway, the point is, anything above a 386 is great for linux. It may not be blazing fast, but it will do the job. The power of Linux (or any Un*x) is in the command line.

(rant over)

Quote from: martext
I'm hoping to learn more soon though, as I'm planning on dual booting Gentoo and WinXP.


Have you run Debian before or another of the more command-line-centric distros before. Oh yeah, you ran SuSE. All I'm saying is, the install is one of the tougher ones out there and is not geared towards newbies at all (ex. you have to name all of the optimization flags for gcc and the docs weren't that good the last time I looked). Also, the installation doesn't always go quite as planned as it messes up since you are compiling everything from source and the scripts are fool proof. I'm not saying to forget about it and choose something else, it is just that it is sometimes more work than what it is worth, but then that is what seperates the geeks from the rest of the world.

For a learning experience, though, run through a linux from scratch install. http://www.linuxfromscratch.org. I am going through my second one right now and I forgot how much fun it is. You really learn a lot more from that than installing any distro because you can examine and optimize everything exactly the way you want it. You could probably even integrate Portage into it if you were slick. I have seen instructions on how to integrate Debian's apt...

Anyway, enough of my long winded post. Welcome and I look forward to hearing more from you in the future.
