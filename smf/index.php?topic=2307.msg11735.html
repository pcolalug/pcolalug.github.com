One quick way I look for hardware drivers is to boot up Knoppix. That is a live CD with very good hardware detection. I don't guarantee it will work, but it is a good start. After booting that up, you see if your soundcard works. If it does, determine which driver it is using and all the information you can gather on it. Now go back to your Linux that you are having issues with and try to get the driver Knoppix was using to work for you.

By Live CD, I mean a distro that does not have to be installed on the hard drive. It runs from the CD and RAM. So nothing gets installed/damaged from your current install.

The Spinich Alfredo link did not damage my Firefox experience.
