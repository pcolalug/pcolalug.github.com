Allison,
You'll need to tell the laptop's BIOS to boot from the CD when starting up.  Usually there is a key to press (during the boot up process) to make this happen for a "one-time" boot from CD.  Watch the screen carefully and look for something that says "Press Esc for Boot Menu" or something similar.  If you get that to work, you'll see a little popup that lets you choose whether to boot from hard drive, CD, or floppy.

Another way would be to actually enter the BIOS Setup program and change the drive that the system boots from to the CDROM drive.  You get into the BIOS Setup menus in a similar fashion, by pressing a key early in the boot up process, often "F2" or "Del".  Once you get into the menus look for things like "Boot Order" or "Boot Device" and set it to look at the CDROM drive before it tries the hard drive.  Save settings and reboot (with CD in the drive), and it should boot up on the first Mandriva CD.

Let us know if this is unclear or doesn't work...  Good luck!

Dan

Allison, another thing to be aware of is the difference between creating a bootable CD versus a regular data or music disk.  Look for an option in your CD Burner software to create an "ISO" CD.  Some software for burning CD's doesn't include this option (my software didn't).  There is some free software called "BURNCDCC" which is specifically designed for creating bootable CD's.
