Post by: danc on December 02, 2004, 01:14:33 pm
Quote from: SiliconFOXX
That's what I changed to 4, but at the end of the startup it would say, "starting X11," and then the screen would go blank, a minute or two later click back to the console, go blank, ad infinitum.  Real weird.

Nothing weird about it.  That just indicates that X is not configured correctly.  You must get X configured properly, so that it starts when you use the "startx" command.  Then (and only then) you can change the runlevel in /etc/inittab to start X on boot (runlevel 4).

You need to run "xorgcfg" (or "xorgconfig", can't recall which...) to get yourself a working X setup.
