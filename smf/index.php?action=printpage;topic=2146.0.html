Yep, I did both, and I've got it working like a charm.  Know what it was?  Something very, VERY easy to miss.

nVidia boards do NOT use UHCI, that's an Intel and VIA thing.  nVidia boards use OHCI and the nForce3-250 uses two OHCI interfaces and one EHCI interface for USB.  As I had no idea previously during my kernel config about how the USB interfaces are derived I just accepted what had already been there and figured if I ran into an issue I'd solve it.

Well how about that, guess I solved it.  :lol:


The answer came from "lspci -v" which will tell you, under stable settings like the 2.4.26 kernel in Slackware, what interfaces everything is using.  Absolute gem of a tool.
