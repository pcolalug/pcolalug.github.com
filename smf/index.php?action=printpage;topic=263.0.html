Title: Unreal Tournament 2003
Post by: rburt3 on December 19, 2002, 01:37:35 pm
The reason it works only with the newer NVidia drivers/cards and, recently, the new ATI drivers/cards is because UT2k3 uses some proprietary texture compression (SV3 or some such) routines only to be found in the GLX libs that come with these drivers. So, its not the hardware choice, but yet another example of the limitations of proprietary, close-source, software.

Keep in mind, too, this is a Linux-only situation. It should run with whatever card under Windows (even the Voodoo), so long as it meets spec.

Now if you're going to game, you really should upgrade at least one of those cards to a newer ATI or NVidia (GeForce3 would be sufficient) card. 'course, with these cards you *will* have to taint your system with non-free software in order to play the latest/greatest games. Then again, you could just game in Windows, but you'd still probably want a newer card.
