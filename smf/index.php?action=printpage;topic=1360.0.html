Title: lan woes
Post by: hitechwizard on February 05, 2004, 10:02:47 pm
If your regular router is using the same subnet routing scheme as your WIFI buddies (and they both are probably using 192.168.0.0/16 or the Class B private subnet), then your computer's default route is out the router, not the wifi, which is why you don't see them. Easiest thing to do would be to make the WIFI's DHCP pool use 10.0.0.0/8 addresses (the Class A private subnet). Then you would have a default route out the 192.168/16 port for internet, but any 10. addresses would route correctly through the WIFI, and CHEESY-POOF, you see the game going on the WIFI segment.
