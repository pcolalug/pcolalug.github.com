Title: Re: Off-site Storage/Backups
Post by: hitechwizard on November 01, 2006, 02:26:52 pm
Yes, I still do remote backup service (www.remotedatalink.net) that utilizes a specialized rsync system in order to backup your data to an offsite location via an SSH tunnel. As a benefit, it allows you to use the recovery website as a secure means for retrieving your data via our web data recovery interface. For active sites, we synchronize on the hour so that the worst case scenario is a loss of 59 minutes of data.

We have tried using our client on Windows, but unfortunately there are issues with tunneling rsync via ssh on a Cygwin platform that have not been resolved yet. For these clients, we have a "mini linux" server that will attach to your Windows shares via Samba, which allows us to backup the data securely.

If you are interested or need more information, let me know and I can either discuss via phone or in person if needed.
