Title: Quick Fix: Slackware 9 MySQL Database
Post by: imnes on August 18, 2003, 03:27:34 pm
If you want to use the included version of MySQL database with Slackware 9, you'll probably find that it doesn't work after initial installation.

Follow these steps after installing MySQL:

Login as root.

Execute these commands:
Code:

cd /var/lib
chown -R mysql mysql mysql
chgrp -R root mysql mysql
su mysql
mysqladmin -u root status
cd /var
mysql_install_db


Then become root again, and copy the boot script:
Code:

cp /usr/share/mysql/mysql.server /etc/rc.d/rc.mysql
chmod a+x /etc/rc.d/rc.mysql


Now it should work.  Run the script to start the database, and have fun.
