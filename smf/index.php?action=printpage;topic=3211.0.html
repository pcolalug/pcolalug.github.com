Title: SonomaSunshine Update

I've rewritten it since the comical broken version I demoed a few months ago.  It's WAY better now...  :)

It's a correctly functioning Django program now.  It gives you a website (meant to be used on a local net) to create and edit Flash movies (Flash 9).  Since the edit features are URL based, it could be used as a web service for another program also.

The latest SonomaSunshine generated pages are here :
