Title: Bootloader?
Post by: kilar on September 14, 2004, 05:20:44 pm
There are two major bootloaders in Linux--GRUB and LILO.  Most people put the bootloader in the Master Boot Record.  If you do not want Mandrake anymore, you can not simply delete Mandrake.  If this is what you want, you will need to use the Windows boot disk to boot the system and fix the MBR.  If you want to keep Mandrake, you can rewrite the MBR in Windows and use the NT Loader to boot Windows or Mandrake.  Or, you can install a different boot loader or different distro and still boot Linux or Windows.

You will always have the ability to load Windows, as long as you do not delete the Windows partition. The MBR resides seperately from Linux or Windows.  The MBR can be 'controlled' by either OS.
