Title: USB drives
Post by: woodyharold on April 14, 2008, 09:25:58 pm

I just finished wrestling with "the bug" so while spending time indoors I was able to get in some extra computer time to delve into areas previously ignored. Crowe recently mentioned using some diagnostic apps he had on a jump-drive and that piqued my curiosity about uses for the little pendrives that are getting so popular these days.Piqued enough to do some googling about pendrives, booting from USB drives and a little bit about remastering live cds. Until now all I had ever used the USB drives for was for temporary storage and to transfer files from one computer to another. I was amazed ( but then a lot of things about linux amazes me) at how simple it is, using the tools available, to make live cds and bootable jump-drive operating systems. WWW.pendrivelinux.com has a wealth of helpful information on various distributions including remastering ubuntu. I know there must be many more uses for the little drives than the few I have been using.Maybe someone can let us know of their experiences with them.

Woodyharold

Title: Re: USB drives
Post by: Yossarian on April 16, 2008, 04:13:14 pm
USB drives are great for running live distros.  They're even better if you can set up a persistent home on them so you can save data.

However, I've tried installations from usb rather than burning a CD, and I've noticed that whenever I do that, the GRUB boot loader ends up displaying multiple copies of the install after a few boots.  Not sure if this has happened to anyone else though.
