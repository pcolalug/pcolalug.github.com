You will need the 1.02 patch which you should be able to find there as well. You will also want to download the map pack that is mentioned in this thread (http://www.pcolalug.org/phpBB2/viewtopic.php?t=1233&start=15). I assume that is the default install location.

I just tried the links in hightechwizard's post and they did not work but there is information in that thread on other places to get maps. You can also just start the game and point it at the server.
