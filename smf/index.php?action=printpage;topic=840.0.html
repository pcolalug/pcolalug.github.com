Title: TeamSpeak
Post by: danc on July 04, 2003, 12:20:10 pm
Howdy,

   Last night Narsil and I tried out something new, called TeamSpeak.  It's a voice-over-IP client which lets you talk with other players in online games.  I used it last night in Windows, and have now also gotten it to work in Linux.  This allows you to coordinate movements/tactics with your teammates, and really adds a lot to the gaming experience.  No noticeable impact on game playability, very small bandwidth usage.

   I'll tell ya, if you could get a few decent players working together on a team with this voice chat going, I think you'd be unbeatable.  It's very easy to set up and use.  All you need is headphones/microphone.  I did have some issues with my soundcard in Linux, but have solved that.  If you'd like to check it out, it's available at:  http://www.teamspeak.org and has both Win / Linux versions.  Great stuff!

Dan
