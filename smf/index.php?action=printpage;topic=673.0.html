I fixed the problem page but uncovered some more issues.  It looks like the PHP server that this site is hosted on is configured differently (or has been reconfigured) from my home server.  

What's happening is variables are automatically being exported / imported to the program.  If I have a session variable called $admin_rights, on my server I have to updated it with:

Code:

$_SESSION['admin_rights'] = 1;


But, on this server, just a call to what would normally be a seperate variable:

Code:

$admin_rights = 1;


is actually updating the session variable!  This is annoying and I'm not sure how to easily fix it.  I think I will end up having to rename all the session variables to make sure this doesn't happen.  Anyone else have any thoughts on it?
