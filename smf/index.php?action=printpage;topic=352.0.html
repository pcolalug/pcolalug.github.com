Okay, the -/+ buffers/cache is the line that is important. See where is says used and free, that is more accurate as to what is really being used. If you are seeing that 445 meg being used in that line, then you might be having issues. Another question for you. Reboot the system and before you load up X, run free. Also, check your memory usage over time. You may have a leaky program...

Just a few ideas anyway.
