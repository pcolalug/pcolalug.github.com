I finally got 10.0 Official downloaded and installed. Had to download at a friend's house who had a cd burner and also a bad stick of ram, so you can imagine how much fun that was. Anyways, some of my thoughts...

I was using 9.1. I did a fresh install instead of an upgrade. The install was basically the same as before. Quick, easy, and painless. Everything in KDE seems to run a bit faster and it looks nicer(Fonts are crisper, nicer themes). I haven't used GNOME so I can't comment on it. Otherwise I don't see a whole lot of change on the desktop.

I had a few problems. For some reason Hardrake would not configure a PS/2 mouse for use with the wheel for me. I tried massaging XF86Config with no luck. Finally, I just switched it to the USB port and it worked. XMMS kept crashing, bringing down the entire system once, until I installed ALSA XMMS and configured XMMS to use it. Mplayer kept giving me some video output not found garbage. Instead of bothering to fix it I installed xine and it worked fine.

I like the older Drakconf better, it seemed more intuitive, but I doubt it'll take long before I am just as comfortable with this one. I have not yet used this system for any programming. I do have PyQt and Eric3 installed and it seems to run fine. I haven't installed Apache or any databases yet either. As far as an every day desktop, I have Firefox, Gaim, Yahoo Messenger, and Evolution all running well and they look great. It is not revolutionary, but I am well pleased with it so far.
