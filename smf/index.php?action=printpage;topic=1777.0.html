Title: firefox bookmark trick
Post by: ctrlz on July 16, 2004, 11:56:54 am
some of you might already know this, some might not. i sure as heck didn't until today  :)

question: do you frequently visit a site just to submit a search? would you like to make this a more painless and integrated process?

answer: use firefox's "keyword" field when adding in a bookmark.

i frequent linuxpackages.net for all my slackware needs. too often, i go there just to search for a package. in a nutshell, i can now type in the address bar: searchpkg firefox . and voila! there's my search results.

how can you do it?

1. click "bookmarks" > "manage bookmarks"
2. click "new bookmark"
3. type in a generic name for your bookmark, in my case "linuxpackages.net"
4. use the url for the search results, and replace the variable you want to search on with: %s .

example: http://www.linuxpackages.net/search_view.php?by=name&name=%s

5. type in a keyword. this is what you will type in the address bar in firefox.

example: searchpkg

6. click ok
7. in the address bar, type: searchpkg firefox

use your imagination! if you think of any other good sites to use this trick on, let us know!
