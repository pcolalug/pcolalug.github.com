Title: Script to check your LAN
Post by: danc on March 17, 2005, 08:34:12 am
Here is a script I did for checking which of the machines on my LAN were online at the moment.  It's quite basic and yet does the job it's intended to do.  BASH is cool.   :)

In case you wondered, the "\e[32m" type stuff are escape codes to colorize the returned text (green for online, red for offline).  You should be able to easily make some modifications to this for use on your own network, if you'd like.

Code:

#!/bin/bash

# Script to check online status of machines on the LAN.
# Original version 3/16/05

Dan=192.168.254.104
Laptop=192.168.254.50
Teena=192.168.254.105
Michelle=192.168.254.103
Printserver=192.168.254.222

for name in Dan Laptop Teena Michelle Printserver
do
     eval ip=\$$name
     if ping -c 1 -w 1 $ip >/dev/null
     then
         echo -e "\e[32m$name ($ip) is online \e[0m"
     else
         echo -e "\e[31m$name ($ip) is offline \e[0m"
     fi
done
