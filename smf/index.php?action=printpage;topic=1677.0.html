Are you burning them as ISO's are meant to be done?  I don't know how much you know about it, so if you already know this, ignore...  ISO's must be burned as "images", not as a file.  Look at one of the coasters you made, and see what you see (with Windows Explorer).  Is there a single file on there named "<Mandrake....>.iso"?  If so, you burned it incorrectly.

Not familiar with CDRwin, but look in menus for things like "burn ISO image", or "write image", or something along those lines.  You mentioned above that you were using CDROM format (not CDROM-XA).  When burning an ISO, those kinds of choices should not even be offered.  You just burn the ISO, and there are no choices about ISO9660 type, Joliet extensions, mode 1, mode 2, etc...

Have you ever succesfully burned ISO images before?  Let us know...
