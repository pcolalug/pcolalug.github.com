Let's make some improvements...  First you've got to be "root", so open a terminal and type "su" and then the root password.  Now change directory, like this:  "cd /etc".  Make a safety copy of the file:  "cp fstab fstab.safe".

Now, to edit the fstab file, type:  "pico fstab".  On the "/dev/cdrom" and "/dev/fd0" lines, change the word "owner" to "user".  This change will allow you to mount these without needing to be root.

For your USB drive, add the following line at the bottom of the file:
/dev/sda1   /mnt/usb  vfat  umask=0   0 0

Now save and exit the file by using "Ctrl-O" / Enter / "Ctrl-X".  Last thing to do is make the directory referenced in the line above, so change directory to the /mnt dir ("cd /mnt") and type:  "mkdir usb".  This is done while you still are root...

Quote from: LinkZero

and i've tried to following to mount my usb:
mount /dev/usb /mnt/usb

After the changes above, you should be able to mount the USB device by just using the command:  "mount /mnt/usb".

I do not know which distro you are using, but most distros have the command "lsusb". So, as root, with all your USB devices connected, try to see if the USB ports are working with
Code:
lsusb
(that is l as in list in the beginning). If you see a reference to your USB device, then you should be able to do
Code:
cat /proc/partitions
and see if your device is under sda1 or usb1 or whatever. For example, if you did "cat /proc/partitions" and saw this:
Quote
cat /proc/partitions
major minor  #blocks  name

   3     0    6297480 hda
   3     1    6000246 hda1
   3     2          1 hda2
   3     5     297171 hda5
   3    64   39082680 hdb
   3    65    9765976 hdb1
   3    66     500472 hdb2
   3    67   28816200 hdb3
 254     0    6000246 dm-0
 254     1     297171 dm-1
 254     2    9765976 dm-2
 254     3     500472 dm-3
 254     4   28816200 dm-4
   8     0      64000 sda
   8     1      63728 sda1
then you know that the USB device in question is sda1. Your distro may use usb1. If it happens to be sda1, then do
Code:
mount /dev/sda1 /mnt/usb
If that works, then you can add it to your fstab as such:
Code:
/dev/sda1 /mnt/usb vfat user,noauto 0 0
