I've never heard of telinit, but init 3 would do it for you.  But you don't even need to do that.  Go into yast and search for "nvidia", install the package (which is the commercial driver).

Make sure the driver works without errors, get to a console (you can do this in X while it is running) and type "modprobe nvidia".  It should just return to the prompt with no errors.  That means it's ok.

Now to enable the driver you could go into SaX2, but I prefer the old fashioned way.

Edit your /etc/X11/XF86Config and change

Driver "nv"

to

Driver "nvidia"

Remove the extra lines stating Chipset and other junk.  Mine looks like :

Code:

Section "Device"
  BoardName    "0x0312"
  BusID        "1:0:0"
  Driver       "nvidia"
  Identifier   "Device[0]"
  VendorName   "NVidia"
EndSection


I just left BoardName / BusID / Identifier alone.

And restart X (either logout or ctrl-alt-backspace).  Should get the nvidia splashscreen.
