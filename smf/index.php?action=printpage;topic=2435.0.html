Title: HoneyPots
Post by: lloydr on May 15, 2005, 05:21:46 pm
I've helped a friend setup a honeypot as a decoy. The LUG library has a book called "Know your enemy" that is completely about honeypots and has mentions about the honeynet project. I believe that book is checked out current by crowe. But you probaly would be ahead by checking out the honeynet project. They created the honeyd which allows you to setup decoy honeypots act like many different operating systems and many different server daemons.

The one I helped a friend setup is running on an OpenBSD platform with everything closed off, 2 interfaces, and running honeyd. He has 1 interface for management and such that is closed off from the outside world and the other interface is for honeyd to be running on. the honeypot is the first box directly behind his firewall for a decoy like you were talking about.
