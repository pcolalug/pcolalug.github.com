wireless managment
« on: May 30, 2008, 03:02:48 pm »
I don't know about you but one of the biggest annoyances to me in Linux has been a really good wireless connection manager that will handle wpa and allow me to see all the possible connections and even handle wired connections.
Well I have finally found it is called Wicd and I was first introduced to it in the latest BT3 live cd and fell in love with it.
the link can be found here. http://wicd.sourceforge.net/.
Let me know if you have any problems I had a minor issue with it in Slackware but was able to resolve it quickly with Google.

Wow I am on a roll now I have also found Wireless Assistant that is the default manager on Back Track 3 and it works really well also.
I have installed it flawlessly on slackware as there is a slackpkg for it along with most other distros. the tool from command line is called wlassistant. IT can be found here. http://wlassistant.sourceforge.net/
Hope that helps some of you.
