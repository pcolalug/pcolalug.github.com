33  Distributions / Debian Based Distrtibutions / Freespire First Release on: July 21, 2006, 07:05:17 am
The first beta release of Freespire is available for download.



OSDir has a  screenshot tour.

I've installed it on a machine, and it works really well.  You can use the linspire CNR service if you want, or the standard apt-get tools with it (or both if you want).  I've installed some commercial apps through my linspire CNR service on it and it works fine.  (Cyberlink PowerDVD for Linux is a good legal DVD player for $9.95 if you have a CNR subscription).  The only 'bug' I've noticed is that when you try to play full-screen you can still see the KDE task bar, so you have to configure the taskbar to let other programs draw over it. 


http://www.linspire.com/lindows_products_details.php?product_id=25183&pg=screenshots&ss=2#bottomhalf

Comes with the dev tools I use daily, Sun Java SDK and Eclipse.

All the video files I tried playing have worked, they have legally licensed codecs on the system for free. 

Worth taking a look at for anyone who likes KDE.  I wonder if the Freespire project will spawn a gnome variety?

34  Hardware / Other Hardware / Niagara II Running on: July 18, 2006, 09:07:18 am
Sun Microsystems has booted its Solaris operating system on a server with a prototype of its forthcoming Niagara II processor, one key milestone for the company's attempt to restore the relevance of its Sparc processor family.

The Niagara II system booted on May 26, spokesman Alex Plant said, about three months ahead of the late August or early September schedule. The processor is still scheduled to ship in systems in 2007, he said.

Niagara II keeps its predecessor's relatively low 70-watt power consumption and extends its ambitious design elements. Niagara II has built-in 10-gigabit-per-second Ethernet networking, multiple floating-point units to speed mathematical calculations, and new encryption and decryption abilities.  It has multiple processing cores and execution threads: eight processing cores, each core will run eight threads. Each system could have dual processors.  So that's, 256 threads per machine?
