Title: A good Virus Scanner
Post by: steve on July 07, 2003, 06:22:45 pm
I just found that my favorite FREE virus scanner (http://www.avast.com/) for Windows is now available in beta for Linux.

Avast! (http://www.avast.com/) has got to be the best free virus scanner on Windows. It's free for home use, does automatic program updates and automatic database updates. It is also the only Windows virus scanner I've found that can do a low level, before windows XP/NT/2000 starts up, virus scan. I run it on all of my home Windows computers.

Anyway, I'm looking forward to testing the Linux port of this excellent Virus Scanner.

There are  a couple of viruses out there that target Linux and unix systems.
