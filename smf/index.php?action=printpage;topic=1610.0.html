Title: 2.6.5 strangeness
Post by: rburt3 on May 15, 2004, 08:16:57 am
There are some really nice things in 2.6, but many of the things you're used to are gone or changed a bit.

First, ALSA is part of the kernel now. There's a nice ALSA guide here (http://www.gentoo.org/doc/en/alsa-guide.xml). Its for Gentoo, but if any time it says to emerge something, just install it like you would on your distro instead.

devfs (/dev/*) is deprecated and turned off by default in 2.6, so you have to explicitly select support for it when you compile your kernel. I heard a blurb somewhere that devfs won't be in 2.7 at all, so we'll all have to start learning about udev (the new device file system) sooner or later.

As for your module loading problems, did you install the new modutils package? 2.6 requires that you do.

All that being said, danc is right. Unless 2.6 gives you something you really need, like bluetooth or better USB support, then I wouldn't stress too much about it until your distro "officially" supports it.
