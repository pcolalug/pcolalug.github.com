Title: My personal agony with Linux and wireless
Post by: kilar on January 30, 2005, 09:53:47 am
In general, cable Internet should work on any Linux platform with no problems. DSL, on the other hand, can get tricky due to PPoE. You should have no problems getting any cable Internet working, once you have the wireless up and running.

Cable Internet is straightforward--just plug the ethernet cable into your ethernet card and go.

If you can not get wireless working, you can try ndiswrapper. The catch is that you will need to use the command line.
