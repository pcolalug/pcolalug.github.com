For this the front-end DVD::Rip was created and looked like a solid piece of software...and also the only solution I could find.  The issue with it is that it has a LOT of requirements in terms of software and while I've beaten down nearly every one of them I'm stuck on one final bit of the software prep that keeps it from running.  

DVD::Rip is perl-based and apparently needs some modules for perl, the "Gtk" module specifically.  I've tried to follow their directions for making the gtk module (complete with the --no-guessing bit) and installing it as well as separately making the Pixbuf part but both have failed miserably after following their directions to a "t."

Without these making the "makefile" for DVD::Rip fails horribly so apparently these are seriously required.  There's also no Slackware packages so I can't have that ease of installation.




I can post more specifics at home, but any thought/opinions on this?  Should I just look for another program?  I'm not really aware of any others that are used for DVD backup in Linux and while I "can" do it in Windows I'd really like to spend that time in Linux if at all possible.
