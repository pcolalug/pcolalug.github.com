9  Software / Games / Postal 2 howto install the AW7 patch. on: April 24, 2006, 07:12:50 pm
First install postal 2 (share the pain) and postal 2 AW (apocalypse weekend).
I installed those in my /usr/local/games dir.
Then you will need the AW7 patch, be sure to download AW7Final.zip and not the self extracting windows version.
You can get it from here: http://www.postalnetwork.org/files/file.shtml?ID=322

Unzip the file in the ApocalypseWeekend dir which is in the postal2 dir. ( /usr/local/games/postal2/ApocalypseWeekend/ )

Download
http://webpages.charter.net/kgroombr/Postal2AW7.ini (Should this link stop working let me know and I will find a new one)
and replace the existing file:
/usr/local/games/postal2/ApocalypseWeekend/System/Postal2AW7.ini.

If you have played AW already then delete
.lgp/postal2/System/Postal2AW7.ini.
in your home dir.

Copy:

postal2 -ini=Postal2AW7.ini -userini=Apocalypseweekenduser.ini -savedgameini=ApocalypseWeekendSavedGameInfo.ini -BASEDIR=../ApocalypseWeekend/System

into a file named postal2aw7. 
Be sure to chmod 755 postal2aw7
I stuck that in /usr/local/games/postal2 dir since postal2 and postal2aw are both there.

Simple as that and Postal 2 with the AW7 patch should start.
