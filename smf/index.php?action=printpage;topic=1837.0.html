Title: Fun with Linux PGP
Post by: sandspur on August 20, 2004, 09:48:53 am
This little discussion might be placed in the Recreational Paranoia area.

I'll attempt to justify it by saying that security awareness is more a state of mind rather than reliance on specific hardware or software.  Any tidbit is worth knowing.

What we bang out on a keyboard is subject to keystroke logging software and hardware, video monitoring, TEMPEST attacks, and stuff of which I am unaware.

When I generate text files I am going to encrypt I prefer to do it on a stand-alone machine.  In this case a relic DOS HP palmtop computer.  I save the file to a Flash Memory card in the palmtop.

(Thanks to LUG members for teaching me to get a memory card reader working on my Linux PC.  It started off as a technique to get photos off a memory card from a digital camera.)

I take the memory card out of the palmtop and put it in the memory card reader on my Linux box.  I'm running FC2 at the moment and the memory card is located at:

/mnt/flash along with floppy and cdrom.

At a terminal I'll type mount /mnt/flash.

I can cd to /mnt/flash and type ls to see what is on the memory card.

Among other things I may see the file I generated on the palmtop, lets call it note.txt.  

Here comes what I think is the cool stuff.

With Linux PGP I used to copy the file on the card to some directory on the Linux PC and then encrypt and wipe the original file.

Let us say the text file is in my /home/sandspur directory and I''m going to encrypt it for my PGP pal Mike and then wipe it.

pgp -ew note.txt Mike

Big deal.  Here is what I have discovered.  If I have the flash card mounted I can give PGP the absolute path to the text file like so:

pgp -ew /mnt/flash/note.txt Mike.

The file is encrypted and wiped on the memory card.  Now I can copy the encrypted file to the PC and send it to Mike as an e-mail attachment.

I'll not forget to type umount /mnt/flash when I'm done.

SUMMARY

The text file was generated by what I consider a secure stand-alone computer and when the memory card was on the Linux box the file was encrypted and then wiped on the memory card.  THE TEXT FILE WAS  NEVER ON THE PC'S HARD DRIVE!

OK everyone.  You can wake up now.

I thought you might want to know about some tools along this line that I ran across in another life.  

Knoppix STD http://www.knoppix-std.org/

I can't say enough about this cd.  It comes with TONs of security related programs.  (crypto, stego, pen testing, and much much more)

Tinfoil Hat linux http://tinfoilhat.shmoo.com/

Remember that standalone palmtop you were talking about?  This is a linux distro on a floppy, designed to manage PGP keys.  Before you say it's vunerable to hardware loggers, EM radiation detectors, Power consumption monitors, alien probes or a direct hit from a thermonuclear device read the README http://tinfoilhat.shmoo.com/readme.txt
