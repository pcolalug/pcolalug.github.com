I installed ZenWalk 5.0 on an ancient Gateway PII 233 with 128 Meg of ram.  This machine originally came with Win 95 back when I bought it new in the mid 1990's.  The install went smooth, fast, and without problems.

Of all the lightweight distros I've tried, ZenWalk is my favorite so far.  If you have an old machine laying around gathering dust, you might want to give it new life with ZenWalk.  You just might be amazed.  I was.
