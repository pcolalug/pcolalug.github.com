Post by: Madwilli on February 04, 2004, 11:53:17 am
Wow what distro are you using anyhow? (Sounds more like a BSD system.)

That aside,  it sounds like you just need to add a system account for 'gdm'.

Try using the adduser, or useradd, scripts.  If that doesn't work, just edit your /etc/passwd: cut and paste and update the uid and change the gid


GDM is probably trying to start as root and then drop privileges to the gdm user so that any bugs don't end up being root exploits..
