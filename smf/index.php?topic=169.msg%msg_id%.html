Well, here is some info that I have found out. We were discussing before in a post about the OpenOffice.org HTML editor, how I would like to be able to set a site root. I think I have found a program that does this.

My argument before was that most WYSIWYG editors do not allow you to set site root, and therefore you have to code in the whole path to an image to get it to display. With a text only editor this is not that important (except when you do a preview). I think I have found a program that helps with that. It is Screem. Site CReating and Editing EnvironMent.

It allows you to set site root and it has a preview pane that will let you see the page as to how it might look in a browser. Of course each browser is different. It will allow you to specify an external browser as the preview browser. Doing this though, I am not sure that your links to other pages and images will work right. You have to use the internal browser for that.

Screem is written for GNOME. It works well with KDE too. It loaded with Mandrake 9.0rc3.

Any other HTML options you would like to throw in, please feel free. (I mostly just use vi though)
