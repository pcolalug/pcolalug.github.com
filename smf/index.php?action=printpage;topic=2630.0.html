This is odd.  MIT is no longer offering Pretty Good Privacy, PGP, for downloads.

http://web.mit.edu/network/pgp.html

They say the freeware is outdated and does not work reliably on newer computers.

I'm using Linux PGP v.6.5.8 on a PC with an AMD Sempron processor running SuSE 9.2.  Works like a champ.

My girlfriend is usung the save version of Windows PGP on a PC with an AMD Athlon processor running Windows XP.  No problems.

Call me paranoid.  I wonder if they caved into government pressure not to distribute PGP lest it be used by tourists or other bad guys.
