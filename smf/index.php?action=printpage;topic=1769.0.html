Post by: rugmonster on July 13, 2004, 07:08:46 am
I got K&R's The C Programming Language (http://www.booksamillion.com/ncom/books?id=2897241128647&isbn=0131103628). It has a lot of exercises with each chapter that make good use of the topics covered. The learning curve is a little steep at times, but once you make it through the book, you will have a very good understanding of C.

If you want to cheat, or compare your code, there are sites out there with solutions to the exercises.

From what I understand, Stroustrup's The C++ Programming Language (http://www.booksamillion.com/ncom/books?id=2897241128647&isbn=0201700735) is supposed to be THE book on C++. I've never read it, but have heard good things from other people. I think it has exercises with each topic, like K&R's.
