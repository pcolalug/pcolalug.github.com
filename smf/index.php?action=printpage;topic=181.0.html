Title: Installing RtCW Full Version
Post by: danc on April 16, 2003, 07:31:39 pm
imnes,

   Your original post is pretty old now, but thought I'd ask if you ever got things working?  If you haven't and are still interested, the basic procedure is:  install the linux version (currently 1.4, about 70Mb) and the update patch (1.41, about 4-5Mb), which are available at www.rtcwfiles.com among other places.  You then must install about 6 other files (map packs called *.pk3) which are only on the retail CD.  However they're not easily extracted from there, and the easiest source is from an existing Windows installation.  Anyway if you still want to try it and have any trouble, gimme a holler.
