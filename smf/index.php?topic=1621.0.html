It works!  I run Mac OS X 10.3 on my Intel P4!  Thanks to a PowerPC emulator project called PearPC.

The only lacking features seem to be Sound, and Networking (but only under Windows, under Linux networking is supported).

So if you've been dying to try out OS X, get some install CDs, this free emulator, and have at it.

My processor is an AMD Opteron 150.  I think it's 2.4ghz but not sure.

This thing emulates a G3, so officially I'm running a 1.27ghz G3.

I haven't tried any sound stuff so I don't know if that's working.  Although it didn't complain when I played with the volume control.  But I didn't check to see if sound was actually working.

Network works fine I am able to get online fine.  Downloading Xcode dev tools right now.

Safari crashed on me once but seems solid now, IE also works fine.  I was able to watch movies on quicktime embedded in the browser, was getting maybe 15fps on those - not full speed but watchable.

Java applets in web pages work fine and seem to be full speed.

I'll be installing more software and testing some more tomorrow.
