Title: Changes at Mandrake
Post by: danc on March 21, 2005, 07:19:38 pm
Some changes are in the works at MandrakeSoft...  They have announced that there will be no version 10.2, and will be a "transitional release" this spring before the next official release in the fall.  After that, versions will be numbered by year ("2006") and will be only once a year.  New versions will include some Conectiva features after their recent merger.  Not quite sure yet if this is a Good Thing or not...
