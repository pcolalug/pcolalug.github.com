Okay this is a linux group, so in that tradition here's the source.  Feel free to install it on your server, modify it, make it better.  I don't know XML/SOAP but it would be nice to automate the book entry process with that (keep in mind though that some of the books aren't cataloged by amazon so don't rely 100% on that if you try).

Keep in mind that I only spent a few hours a week in my spare time working on this so it's not the cleanest / greatest code.  Also this is only the second website project I've worked on (and the first one to use PHP + SQL).  But it should be easy enough to follow.  There's a docs directory included which will help you get started.

This was written in PHP4 on apache with MySQL as the database.

Enjoy
