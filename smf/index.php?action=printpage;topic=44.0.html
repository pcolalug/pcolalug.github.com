Title: New Kernel
Post by: imnes on October 05, 2002, 02:15:36 pm
My machines sit behind a firewall, so I'm not really gonna gain anything by having the latest security patches.  If your machine was a web server or something then I could see installing for that reason.

If you aren't having any problems then I say do not update the kernel.  Especially if you'd have to custom build the kernel (I don't know if the updates are pre-built or not).  But building a modularized kernel on a redhat derived distro is a bit of a pain, usually takes me a few tries before I can get everyting working happily.

So I would have to say, if it 'aint broke don't fix it. :-D
