7  Software / Other Software / Key Touch and Key Touch Editor on: January 12, 2010, 05:09:54 pm
Today I read about key touch and key touch editor. I had bought my wife a new Toshiba E105-S1802 laptop. It has touch keys on the right side of the keyboard for the multimedia which didn't work with linux. PCLinuxOS is what is on it. I installed the two programs and ran Key Touch. My keyboard was not listed so I had to run the editor and make a file of the keyboard to import into Key Touch.  To my surprise the editor recognized everyone of the multimedia keys and I saved the file, imported it into Key Touch and applied it and all the keys are working. If you have a keyboard or laptop with keys that don't work in linux give these programs a try. I tell you KDE4 in PCLinuxOS running on this new laptop, dual core AMD, 4 gb ram and 500gb hard drive makes one fine machine now that all the keys work.
