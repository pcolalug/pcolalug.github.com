I'm not sure if anyone here is into MUDs but I just got back into them when I heard about  Achaea on ScreenSavers.

For those that don't know what a MUD is, it stands for "Multi-User Dungeon".
It's basically a text based RPG.  You can find more info at The Mud Faq.
They actually date back to 1978 as you can see on the Online World Timeline

Anyway, I just got into Achaea and it's a pretty cool world.

For those that are currently playing Muds:
Which ones do you frequent?
How long have you been playing?
What's the best Mud client for Linux? (Anything close to Zmud?)

Zmud for linux would be great indeed but its not gonna happen.  I have a registerd copy of zmud that I used when I used windows but now I don't use windows at all and zmud will not work in wine or winex due to the elicense issue.
There is a mapping function in Gnome-mud but its realy not much and gnome-mud is buggy at this time.  If you new to mudding you might tolerate the java interface but since it doen't have support for macros, that I am aware of anyway, I tend to avoid it.   When you get into macros you will discover that tinyfuge has no equal but if you find one for linux that has all the features of zmud please let me know.
I emailed zuggsoft about zmud for linux or support in wine and they basicly said it was not going to happen.  That was about a year ago so you might want to check again.
