Well, my overall impression is outstanding.  I don't have a whole lot to compare it to, as I've used MDK for most of my Linux experience so far, but I can say that I like it MUCH better than Redhat 8/9 versions (Bluecurve modification).  I also like it much better than KDE, which seems too "busy" to me and has too much clutter.

One thing I do really like is the various setup utilities packaged by Mandrake, most of which are contained in the "Mandrake Control Center".  Very easy to use and they work.  I do some of my configurations by hand with an editor, but for some things these tools are easier.  I also like the Nautilus file manager very much.  The other menus include the usual collection of apps, which I suspect is pretty much stock Gnome 2.2.  Overall it seems fast, organized well, and it just works.

I will try to post some screenshots of Mandrake Control Center if I can find some time this weekend.  I'd recommend you try it if you are thinking about switching distros.  The "urpmi" software function is really slick and makes upgrading things easy.
