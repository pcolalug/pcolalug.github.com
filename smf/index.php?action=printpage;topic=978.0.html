Title: XFree86
Post by: on August 31, 2003, 07:15:23 am
David I have never tryed this but read it in the Mandrake Definitive Manual. You can log in as root in console and run XFdrake. It will bring up the configuration tool. If you pick graphic card and scroll to the bottom of the list there are selections for XFree3 and XFree4. These are drop down list. I tried it and picked a different card for my system under XFree4 since I am running it, it would let me make the changes or test it. I backed out and selected a card under XFree3 and it wanted to install software, I assume XFree86 3.3.6 (which it on the 9.1 powerpack disk set) so I backed out. So you might try reconfiguring it that way.   Also I read on Mandrake club to install a different XFree86 rpm, get it, then run as root in console:
urpmi --allow-nodeps --allow-force XFree86-x.x.x-xmdk.i586.rpm
Maybe some of this will help.
