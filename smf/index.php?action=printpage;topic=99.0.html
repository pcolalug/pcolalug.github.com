Title: OO.org HTML editor
Post by: dpeach on August 16, 2002, 09:10:02 pm
I did a basic test with the HTML editor in OO.org. It came out OK.

I did not throw anything really complicated at it. It was able to indent my page margins as I told it to. Though it did it differently than I would. It also made my strange font formatting do fine, with the exception I explain below. Hyperlinks were very simple as well as graphic insertion. Both handled as I would have expected. The graphic insertion put in a hard path to the graphic on my hard drive, which will not work when uploaded. But that is an argument for an earlier post.

The one thing that it did not handle the way I would want is how it did the font manipulation. The formatting I did was with font type, size and color. Here is the output from their editor:
<FONT COLOR="#000080"><FONT FACE="Comic Sans MS, cursive"><FONT SIZE=5 STYLE="font-size: 20pt">Here is some more text</FONT></FONT></FONT>

Here is the way I would have put that:
<FONT COLOR="#000080" FACE="Comic Sans MS, cursive" STYLE="font-size: 20pt">Here is some more text</FONT>

Cleaner, less effort. But I guess when using a WYSIWYG editor you are eliminating the effort of typing in the code. And, if it bothered you, then cleaning it up afterwards would not be much of a problem.

So there is my short review on their product. Have your wife give it a try and see what she thinks.
