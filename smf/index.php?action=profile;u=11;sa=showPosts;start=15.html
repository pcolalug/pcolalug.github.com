16  Distributions / Other Operating Systems / Splashtop on: November 04, 2010, 04:58:56 am
Splashtop is a proprietary Linux mutation that comes with  several varieties of netbooks.  The idea is an "instant on" experience.

It boots up in RAM before your main OS boots up. 

Cool videos of it are here:

http://www.splashtop.com/videos.php

Pretty good discussion of it at Wiki:

http://en.wikipedia.org/wiki/Splashtop

Things I have observed:  Wired and Wifi connections work fine.  Forget wireless broadband.  Forget printing.  It will 'see" external storage devices like USB sticks and SD cards so you have limited ability to download or upload files.

So far I have been unable to conjure up a terminal to snoop around the file system.

The browser is a version of Firefox.  Access to web-based services like e-mail or Google Docs.  Turn your perfectly good computer into a thin client.

Splashtop does what it advertises well.

21  Distributions / Debian Based Distrtibutions / Jolicloud on: July 02, 2010, 08:32:11 am
Recently got an HP mini 210 netbook that came with Windows 7 installed.  Eager to purge it of M$ I installed Fedora 13 and then Ubuntu 10.04.

Everything was peachy with each distribution EXCEPT that I could not get the built in wi-fi to work.  Lots of moaning and groaning trying to get it working.  Wired networking was fine.

Chanced to find Jolicloud, a version of Ubuntu tweaked for netbooks.

http://www.jolicloud.org/

It claimed compatibility with the HP mini 210.

Downloaded Jolicloud "Randy" Jolicloud-Pre-final.iso and burned it to a disk.  Using a USB external CD reader I installed it.  (wonder if Pre-final = Beta or if it is an intermediate stage of development between Beta and final product?)

Easy installation.  Wi-fi worked out-of-the-box.  Connected to household wireless router no sweat.

Distro is lean on included applications.  Got Abiword, GIMP, and wipe with apt-get and a .deb version of Acrobat Reader.

Netbook desktop is a littly cutsey looking but can easily be changed to a regular desktop appearance.

27  Software / Other Software / Nifty utility on: May 07, 2010, 07:38:29 am
About to take possession of a new netbook, an HP 210 mini.  Comes with Windows XP.  I intend to install Fedora 12 on it.

Like most netbooks size prohibits a built-in CD/DVD reader/writer.  Unwilling to spring for the money for an external USB reader/writer I'm going to try to do the installation from an installable live ISO on a USB memory stick. 

Stumbled upon a cool utility for getting the ISO on a memory stick:  liveusb-creator.

Installed it via yum install liveusb-creator.  It is also available for Ubuntu with apt-get.

You download the ISO to the directory of your chioce first.  Then you run the utility and see a nice GUI.  You select the ISO source and then give it a target:  The USB stick.  Very painless.  Even does a CHECKSUM of the ISO for you.

Booted up a laptop with the live Fedora version.  Will try to install from that point with the new toy.  Will keep you posted with my experiences in the next couple of days.
