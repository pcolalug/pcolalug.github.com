Title: Freebsd Networking troubles
Post by: rugmonster on November 17, 2003, 10:49:52 am
Being that you are using DSL, you probably need a PPPoE dialer for the session to start. You won't get far without that since it does a PPP session (remember dial-up) over Ethernet, hence PPPoE. You probably don't have one included with your installer.

That doesn't mean you can't find a home brew installer that has a PPPoE dialer included to do your network install. If I remember correctly, FreeBSD has a mini install CD and it might have one included.
