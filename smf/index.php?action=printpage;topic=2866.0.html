Title: OpenDocument Format is ISO 26300
Post by: dpeach on May 03, 2006, 04:31:30 pm
It is official. The Internationl Standards Organization has granted an ISO number to the OpenDocument (http://www.openoffice.org/press/iso_pr.html) file formats. This has been in discussion for some time, but it is now official. For software to be ISO compliant (which nothing says they have to be) they will have to use the OOo OpenDocument file formats. Great news I think.
