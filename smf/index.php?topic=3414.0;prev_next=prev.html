Oh, and I'm sorry for those of you that want to know how I did the same basic thing with Iframes, I did it by:
Code:
<html>
<head>
<center>
<title>Turn on and off Camera</title>
 
<script language="javascript">
function loadTwo(iframe1URL, iframe2URL)
{
parent.cam1off.location.href=iframe1URL
parent.cam1on.location.href=iframe2URL
setTimeout("alert('Done Baby.')",4000);
}
</script>
<br>
<br>
</head>
<body style="color: rgb(255, 255, 255); background-color: rgb(51, 51, 51);" alink="#0000ff" link="#0000ff" vlink="#0000ff">
 
<font style="font-size: 28px;" color="#9f1a1a" face="Arial"><b>Turn Off and On </b>
</font>
 
<br>
<br>
<hr width=230/>
<a  href="javascript:loadTwo('http://oneadvent2.viewnetcam.com:50000/mobileImageTransfer?S=160x120&Q=Mobile&N=0&E=0x0&C=25967732&T=0&M=3&O=2','http://oneadvent.viewnetcam.com:50001/mobileImageTransfer?S=160x120&Q=Mobile&N=0&E=0x0&C=87925761&T=0&M=3&O=2')"><img src="on.png" border="0"></a>
 
<a src="off.png" style="color: #ffffff" href="javascript:loadTwo('http://oneadvent.viewnetcam.com:50001/mobileImageTransfer?S=160x120&Q=Mobile&N=0&E=0x0&C=87920808&T=0&M=4&O=2','http://oneadvent2.viewnetcam.com:50000/mobileImageTransfer?S=160x120&Q=Mobile&N=0&E=0x0&C=87920808&T=0&M=4&O=2')"><img src="off.png" border="0"></a>

</body>
