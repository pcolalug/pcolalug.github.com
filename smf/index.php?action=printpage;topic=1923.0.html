Title: Video Recording in Linux
Post by: imnes on November 05, 2004, 06:55:47 pm
After trying to use lots of windows programs in Linux I found out that pretty much each program works well with a different version of wine.  Some programs need older versions of wine to even run.  I think it's because wine developers are constantly changing /  fixing their version of the windows system DLLs, and I guess when they code in a fix for one app it causes unknown problems in others.  I'm not really sure.

One thing you can do is use native Windows DLLs which works great.  But it's not legal to distribute those, so you build it and get it working on your system and that's it.

Some of the commercial packages like Crossover Office and WineX are using commercial code from Microsoft and other developers to make their versions work with a lot of software.

Some people have reported DVD Shrink to work with Crossover Office's version of Wine, I've tried Crossover 2.1.0 and 3.0.1 and I didn't have success in either version.

Right now I'm using DVDShrink on last month's release of Wine and 4 windows DLLs.  I need to find out if these DLLs are legal to redistribute, I think these ones are though.  

By default wine is setup to only allow one copy on your system at a time, one centralized config file etc.  So when I'm buililding wine to work with one program I build it to only run that program.  Instead of installing wine into /usr/bin, /usr/lib I installed into /opt/dvdshrink.  And instead of having the config files go into ~/.wine/config I have one config file located in /opt/dvdshrink.    This makes it easy to package the whole thing and send it to other people to use without disturbing any wine packages they already have installed on their system.  Crossover Office and Transgaming WineX I think basically do the same thing with their installs.

But basically getting software working in wine involves trying lots of different versions until you find one that works, and reading debug messages when wine crashes to find out what DLLs are causing problems.  

Dunno if that helps explain what's going on or not..
