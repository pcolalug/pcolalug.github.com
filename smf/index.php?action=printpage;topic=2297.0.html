Title: Flash card reader
Post by: lloydr on February 28, 2005, 02:53:18 pm
with Suse, are you using Gnome? I don't know what version of gnome that Suse would have, but versions 2.8 and above have something called HAL which allows you to easily access USB harddrives, thumbdrives, card readers, etc.  It mounts into the /media directory as you mentioned, also dynamically edits your fstab which is one part I don't like but makes it easier for most people.

If you have something called HAL loaded and your running gnome 2.8 or above, you should be able to find the card reader in the computer section of nautilus (kinda like My Computer for Windows).

If your running an older version of gnome than 2.8, running KDE, or running some other desktop environment then I might have to google around to find you an exact answer. But most likely checking out the UDEV system will lead you on the right path if nobody else can get back to you quickly on a good tutorial.

Hope this helps, sorry can't give you the best info but hopefully this gets you started.
