Title: Well that got me going
Post by: bhoff on October 01, 2005, 06:59:04 pm
first I had it right
second had to set permissions on id_rsa to 600
chmod  600 /media/LEXAR_MEDIA/id_rsa
kept going back so I aliased ssh with chmod 600
third copied /etc/ssh/ssh_config to ~/.ssh/ which is your home directory.
then found the line identifyfile /media/LEXAR_MEDIA/id_rsa and uncommented it.

now it works everytime
no ssh in without key......
