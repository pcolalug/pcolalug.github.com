Title: uninstalling et and aa?
Post by: imnes on March 01, 2004, 11:28:55 pm
If all else fails, delete the directory.

rm -rf /usr/local/games/enemy-territory

Then the symlinks to the binaries:

rm -f /usr/local/bin/etded /usr/local/bin/et

That should be it.

Nick
