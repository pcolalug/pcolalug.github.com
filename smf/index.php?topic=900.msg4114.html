Book Review
Using Samba, 2nd Edition
By Dan Clough

   Do you need to network several Unix/Linux computers together to share files and printing services?  Or, even more likely, integrate Linux and Windows machines on the same network?  If so, this book may be the best (and only) resource you need.  It is a very complete reference on all aspects of the Samba client/server software, including acquiring, installing, configuring, and troubleshooting.  The software is free, works extremely well, and offers a powerful and cost-effective alternative to running a Windows-based file and/or print server.  This edition is published in February 2003 and includes coverage of Samba versions 2.2.x and 3.0.

   The book is organized well, and the Table of Contents looks like this:
- Chapter 1:  Provides an introduction to Samba, it's capabilities, and describes some networking concept basics.
- Chapter 2:  Describes how to install, configure, and test Samba on a Unix/Linux system.
- Chapter 3:  Explains how to configure Windows computers to function effectively in a Samba network environment, including Windows 9x/NT/2000/XP.
- Chapter 4:  Discusses configuring Samba to work in a Windows NT Domain.
- Chapter 5:  Describes methods for using Samba shared resources from Unix/Linux clients.
- Chapter 6:  Devoted to explaining the multitude of options available in the main Samba configuration file, and how to configure it to your needs.
- Chapter 7:  Explains how network client name resolution and network browsing functions are configured and used.
- Chapter 8:  Goes into more advanced discussion of file-sharing options, including permissions, access lists, and locks.
- Chapter 9:  Discusses setting up authorized Samba users, and how to set up security and password functions for your network.
- Chapter 10:  Explains how to set up printers on your Samba network, and how to allow Unix and Windows client computers to access them.
- Chapter 11:  Provides some miscellaneous additional information about Samba, including scripting and internationalization.
- Chapter 12:  Explains how to troubleshoot Samba problems, and describes tools and logical strategies to find out what may be wrong.
- Appendices A - G:  Provide additional information such as configuration quick reference, example configuration files and options, and commands summary.

   The first couple of chapters help a user get things started.  For the most part, it is assumed that the reader does not have an extensive knowledge of networking principles.  Obtaining and installing Samba is covered, and the basic configuration to get Samba file and print sharing started.  The use of the SWAT (Samba Web Administration Tool) utility is explained in brief, to facilitate beginners getting a working config file quickly.  In my opinion, the explanations for SWAT should have been much more extensive, as this is how many Windows users wanting to migrate to Linux will be attempting to set up Samba.  The examples were very short and the topic seemed to terminate abruptly.  In comparison, the next couple of chapters are devoted to configuring the various versions of Windows (including XP) to operate with Samba networking, and the instructions and screenshots were very detailed.  This seemed a little out of place in a book about a Unix-based networking solution, but will be useful to Windows "non-techies".

   The following chapters discuss how the Unix clients are configured, and how to get file-sharing going between them and Windows computers.  Excellent detail is provided on manually editing the Samba configuration file for more advanced file sharing capabilities, logging options, computer name resolution, and network browsing.  Especially useful is the chapter explaining Samba users and security.  Many new Samba users find that they cannot browse their new network because of password and authentication issues, and this section provides clear step-by-step instructions to avoid that common problem.  Another very useful chapter explains how to allow the printer attached to your Unix machine to be used by all the computers on the network.

   One of the more valuable chapters is certainly the one on troubleshooting your Samba setup when things are not working as expected.  The available tools and utilities are explained, and the importance of checking logs for hints.  Then a logical fault-tree method of troubleshooting helps you isolate where to find the source of a problem.  Finally some additional resources are given for further help if needed.

   For those readers already familiar with Samba, perhaps the Appendices are all that would be needed for a refresher.  They make up about one fourth of the total book, and are extremely helpful.  The Quick Reference section provides detailed explanations of every configuration option, including default and allowable values.  The Command Summary discusses every option relating to the smbd and nmbd daemons, for more advanced users.

   Overall, "Using Samba, 2nd Edition" is an excellent reference book for those looking to install, configure, and use Samba in a Unix/Linux/Windows mixed network.  The book is clearly written and provides easy to follow explanations for commonly encountered problems.  The reader should be able to easily complete a basic Samba setup, and move on to more advanced configurations when ready.  Whether you are a professional network administrator or a casual home user, this book should be the only guide you need to succeed with Samba networking.
