Post by: imnes on February 14, 2004, 10:18:46 pm
Use whatever editor you use at school and already know.  Probably jgrasp or eclipse or netbeans?  They are all good for java.  KDE folks seem to like KDevelop for C/C++ programming.  I used to use vi for everything, but I'm using emacs now.  Those two aren't very user friendly, but if you're going to spend a lot of time editing code, especially over telnet, it's worth learning one of them.

I use slackware, and the latest jave JDK came installed and ready to use, I'm not sure if it's setup for you in mandrake, but if not it's easy to do, just let us know.

Post by: rugmonster on February 14, 2004, 11:07:53 pm
Kate is included in KDE and does code highlighting for a large variety of languages, but Eclipse is one of the better ones for flexibility. KDevelop is more of a full featured IDE. Eclipse is as well, but it supports more(?) plugins and is also written in Java, and has tie-ins for GTK and Win32. I use it and vim mostly, but like imnes said, use whatever you use at school if you can.
