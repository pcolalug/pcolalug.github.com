3  Distributions / Security / Re: Off Linux topic! on: February 03, 2009, 02:27:50 pm
I have dealt with a few friends and customers with this problem (at the time it was XP Antivirus 2008 but the exact same thing).

1) boot into safe mode
2) install SuperAntiSpyware (be sure to right click and Run As... administrative name) and run full system scan
3) after it is done, reboot into safe mode and re-run full system scan which may clear up a few remnants, then also install and run Avira anti-virus (AVG is crap and a resource hog anymore)
4) Reboot to your regular desktop and install and run CCleaner, run the normal cleaning and also run the registry cleaner several times, cleaning it each time. Sometimes I found after an infection, you have to analyze, clean and repeat 4-5 times before it is all gone.

After this is all done, install Comodo firewall and remove the SuperantiSpyware (it is a resource hog for everyday use) and install/use Spyware Terminator...

Best of all these are all free programs, and rated among the best or THE best.
