Title: Quake 4
Post by: rugmonster on October 23, 2005, 06:20:27 pm
In case the other gamers missed it, Quake 4 has been released and Linux support came only two days after retail release! Thanks Id!

I've been playing for a few minutes and I have to say, it's good ol' Quake with better graphics and a supporting cast. There is a loose story, but for the most part, it's run and gun fun. While HL2 brought us a lot of interesting physics puzzles and such, I haven't found anything like that in Q4 yet. In a lot of ways, it looks like D3, but then again it is based on the same engine. It isn't as scary as D3 is, but that isn't so much my bag. It's just fun. I haven't tried multiplayer yet, but it should be a good time. So if you could run D3, then you'll be good to go with Q4.
