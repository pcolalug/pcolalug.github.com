Storage is an exciting project to replace the traditional filesystem with a new document store. Storage is part of a larger design for a new desktop environment, more details on that to come. The current implementation offers natural language access, network transparency, and a number of other features.


With natural language searches, you could enter the search prace "All songs that aren't written by John Lennon but contain the word love."

Interesting stuff.  Not ready for use yet but something to watch anyway.  It promises to be more than just a tool for searching your documents (which Beagle (http://www.gnome.org/projects/beagle/) already does pretty well).
