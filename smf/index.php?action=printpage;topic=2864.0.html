Title: Vim 7 coming real soon now
Post by: rugmonster on May 02, 2006, 08:11:47 am
Looks like we have a new version of Vim coming out soon. Features include:

    *  Vim script enhancements: Lists, Dictionaries, profiling, etc.
    * On-the-fly Spell checking.
    * Translated manual pages.
    * Internal grep, faster and portable.
    * Printing multi-byte text.

According to this article at Linux.com (http://applications.linux.com/applications/06/05/01/2235242.shtml?tid=13), they are including code completion of HTML, C, JavaScript, Python, Ruby, and XML.

Ready...discuss!
