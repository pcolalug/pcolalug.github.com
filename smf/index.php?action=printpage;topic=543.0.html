Title: Windows account migration tool
Post by: imnes on March 12, 2003, 04:43:33 pm
Be careful about just replacing windows file servers with samba linux machines.  I tried that with a real estate company, but the old (very old) dos-based BTrieve software they used wouldn't work well with samba.  Unexplained lockups and dissapearing shares.  I imaging it works very well with newer software, but this stuff was ancient.  As soon as I took the machine back to being a windows server everything worked properly again.  Oh well - I tried...
