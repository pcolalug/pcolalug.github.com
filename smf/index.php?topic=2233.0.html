I believe right now he's using the default slackware kernel of 2.4.26, therefore hdc=ide-scsi would be nessesary.  

The important thing to look for in the dmesg is where the kernel decided to attatch  the usb device.  I use cdrecord --scanbus because it tends to be one of the backend cdburners for k3b, etc.  If cdrecord can see it, growisofs (or whatever backend the dvd burner software uses should work.
