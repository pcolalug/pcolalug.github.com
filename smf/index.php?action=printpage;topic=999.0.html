Title: Linux on vmware/virtual PC ?
Post by: imnes on September 12, 2003, 04:46:38 pm
VMWare should run the latest releases of Linux.  You might need to do a text-mode install on some of them.  Once you have linux loaded you have to install the VMWare Tools for Linux Guest.  

http://www.vmware.com/support/ws4/doc/new_guest_tools_ws.html#1008207

That will give you access to decent graphic modes for X.  There should be instructions with vmware as far as how to configure X, what video card / modes are available, etc.

I've never used Virtual PC so I can't comment on that.

Good luck,
