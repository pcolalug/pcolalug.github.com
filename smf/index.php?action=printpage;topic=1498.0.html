Not sure if anyone watches Screen Savers but they demoed a really cool program tonight.
It's called Synergy (http://synergy2.sourceforge.net/).

This program allow you to use a keyboard and mouse attached to one computer to work with two computers. It does this through TCP/IP. Right now, I have my Windows computer and it's monitor on and I'm using it's keyboard and mouse to type this message ON MY LINUX COMPUTER. I just drag the cursor to the left side of the linux monitor and it jumps over to my Windows computer's monitor. I can cut and paste text between the two monitors, also. This is one awesome program. It works with Linux computers and Windows computers.

Title: (2) Computers, (1) Mouse and Keyboard
Post by: dpeach on March 27, 2004, 09:12:02 am
I have not used this product because my situation is different and a KVM serves me well. But I can see a couple advantages of this over KVM in some situations.

The main thing is that you have two monitors. You can view the contents of one computer while working on the other. Cut from one computer and paste to the other. Neither of which you can do with a KVM.

I use KVM only when I am testing a new box to be put out on the network. My administration is either done through a web interface or physically at the computers.

Also, with KVM, you typically think of physical closeness of the boxes in question. With this, as I understand it, it could be any two computers on your network. No matter how close or far away they are.

Pretty cool concept.
