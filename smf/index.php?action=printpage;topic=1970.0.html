Title: NetBeans 4.1 Early-Access
Post by: imnes on October 27, 2004, 06:46:06 am
I haven't tried NetBeans in a long time, so I downloaded the latest greatest version to try out.

I haven't tried to get into any sort of j2ee / web development, but for Swing GUI development this has to be the best IDE I've used.

I'm used to using JDeveloper (rebranded and enhanced JBuilder), and to test your Swing applications you have to do a compile and run.  It gets annoying, especially when you're justing making lots of small changes trying to tweak your interface.

NetBeans has added a 'Test Form' button to the GUI builder.  You click that button and your swing form displays instantly.

I'm going to try to use this version of NetBeans for a java gui project I'm working on, I'll update on how it goes.  Also I'm going to try to setup for some web development with this thing, see how it stacks up on that end.
