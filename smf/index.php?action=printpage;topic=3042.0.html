Title: Ubuntu 6.10 "Edgy Eft" Released
Post by: rugmonster on October 26, 2006, 05:52:39 am
Yup, four months after the delayed 6.06 LTS "Dapper Drake" was released, we have a new release of Ubuntu!

Features include:
     
  • F-Spot, a photo management tool that enables tagging, photo editing and automatic uploading to on-line web management sites such as Flickr
  • GNOME 2.16, which in addition to new features such as enhanced power management, makes the GNOME desktop more secure, faster and more stable
  • Upstart, a replacement start-up manager offering a cleaner design, eye-catching effects and a substantially faster boot time
  • The latest Firefox web browser, ver 2.0
  • Proactive security features, preventing many common security vulnerabilities even before they are discovered

I've been using it for about a week and I like it. Upstart is nice. Faster and cleaner. It uses the standard sysV init scripts, but manages to make them work better. Firefox v2.0 is very nice. I especially like the spell check in text field. Also, the enhanced tabs are nice. So far, the only problem I've run into is with Cyrus IMAPd on my server. I had to recompile for BDB 4.4 support. It was compiled against BDB4.3 and linked against BDB4.4. Other than that, I had a very smooth apt-get dist-upgrade.

To make things easier, I suggest looking into Automatix (http://www.getautomatix.com/) to get the rest of the stuff that is really missing from the default install of Ubuntu to make your install complete.
