To see if hotplug/supermount is working correctly, try plugging in a device that you know works, then do
Code:
mount
to see that it is indeed mounted, then unplug the device and do
Code:
mount
and see if it unmounted correctly. If you had a USB memory stick mounted, then yanked it out without it unmounting, then any new device will not be able to use the same mount point, but it will try.

If you do all that and find that the device never unmounted automagically, then you know that you do not have supermount, and that you have to manually unmount your devices before sticking a new one into the same location.
