When you say you canot use the sound card, what do you mean exactly?  Does it appear as though it is working but you get no sound?  In that case, go to a console and run "alsamixer"

If you have an error about missing devices you might have to run the MAKEDEV script, not sure exactly how it's done on your system, or if it needs to be done.

To make sure it is running, go to a console and type "lsmod", if you see modules listed that you added to /etc/modules.conf, then you should be ok.
