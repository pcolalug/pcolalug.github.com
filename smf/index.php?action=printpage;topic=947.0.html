Code:
chmod u+x NIVIDIA-Linux-x86-1.0-4496-pkg2.run

Title: Nvidia Installer
Post by: on August 20, 2003, 06:05:31 am
I recently patched my Mandrake 9.1 kernel for Win4Lin. I downloaded the latest nvidia installer to my /tmp directory according to the installation instructions and after rebooting to run level 3, that is without starting X windows I logged in as root and cd /tmp. Then I ran this command:
"sh NVIDIA-Linux-x86-1.0-4363.run"
which is the name of the version of the installer I downloaded. I then followed the on screen instructions and it worked fine for me.

Here is an excerpt from the readme.txt installation file.
"After you have downloaded NVIDIA-Linux-x86-1.0-4363.run,
begin installation by exiting X, cd'ing into the directory containing
the downloaded file, and run:

sh NVIDIA-Linux-x86-1.0-4363.run    

The .run file is a self-extracting archive.  When the .run file is
executed, it extracts the contents of the archive, and runs the contained
`nvidia-installer` utility, which will walk you through installation of
the NVIDIA driver."

Hope this helps

Title: Monitor question
Post by: steve on August 20, 2003, 08:18:25 am
If the "sh NVIDIA....." doesn't work then try "sh ./NVIDIA...."


The "./" tells sh to look in the current directory for the file to act on.

One thing to remember with Linux. It is very path strict. Unlike DOS it won't run/reference a file in your current directory unless the directory is in the path environment variable or you reference the file using "./" which refers to the current dirctory.
