Title: FreeBSD 4.7 Released
Post by: imnes on November 10, 2002, 10:17:33 am
I just got 4.7 installed.  

I'm surprised actually, all my new hardware works fine, even the sound card (which is not working under latest versions of Linux).

NVidia GeForce 3 Ti 500 video card is working great with the official NVidia drivers.  (Typo in the manual, if anyone is trying to install these drivers you should update the driver line in your XF86Config files to read 'nvidia' not 'nv'.)

I'm running WindowMaker right now, I still need to finish configuring things, but I like it so far.  Using a file manager I stumbled across in the ports collection called "Endeavour" which is nice.

Here's my current desktop in it.  http://www.homeweb.org/~imnes/freebsd47windowmakerss1.png

Here's Endeavor in default setup.  I haven't configured anything (not sure what you can configure yet, I've just used it as-is).  

Viewing a file with it's built-in viewer, in Hex mode.  

There's nice features such as: The device icons on the menu bar, click them to mount / unmount those devices.

Also features a recycle bin, when you 'recycle' files they go to ~/.endeavor/username-recycle-bin/ or something like that, you can restore them from there if you like in the future, or delete them for good.

I think you can go through and configure the programs you use to view all different types of files, like videos, mp3's, graphics, pdf files, etc.
