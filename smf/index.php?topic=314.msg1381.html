Okay, now that I posted in my previous thread the wonders of gnome, I am beginning to get frustrated with it. Namely, this crap with GTK+ 1.2 and 2.0. You have one set of freaking applications which use the GTK 1.2 definitions which don't support antialiased fonts, including Evolution 1.2, which is a quarter of the applications I use.  You throw in Phoenix, which I'm guessing doesn't use GTK+ at all, but not entirely sure on that one and it is easy to have three different fonts within three different windows.
