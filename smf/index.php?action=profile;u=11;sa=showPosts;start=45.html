52  Distributions / RPM Based Distributions / FC 11 network upgrade on: June 16, 2009, 08:23:17 am
Stumbled upon these most excellent instructions about how to perform a network upgrade from Fedora 10 to Fedora 11:

http://www.howtoforge.com/how-to-upgrade-from-fedora-10-to-fedora-11-desktop-and-server

I used a cable modem Internet connection (Mediocrecom) and the whole process took just under two hours.  Did the upgrade on an IBM/Levono Thinkpad T42.

Recently installed FC 11 on another notebook so I got to compare the new installation with the upgrade.  On a new installation the default file system is ext4 while the default file system for FC10 was ext3.

Eyeballed /etc/fstab on each machine.  The upgraded system still has ext3 while the new install has ext4.

Was totally wowed because I had never attempted a network upgrade.

54  Distributions / RPM Based Distributions / Installation report on: June 11, 2009, 10:28:26 am
Downloaded the Fedora 11 live ISO and burnt it to a disk.  Installed to a laptop. 

Installation was straightforward. Applications installed from the live DVD were understandably limited (e.g. no games and there was the lean-and-mean Abiword word processor instead of the OpenOffice suite.).

Prior to the installation I read that Fedora 11 boasted fast boot times.  They were not exaggerating --- it booted to the graphical login in thirty seconds.

Connected to a home wireless LAN easily.  Plugged in a 1 GB memory stick and it was recognized no sweat.

No horror stories so far.
