Title: Dual boot option not working
Post by: Perquisitor on April 13, 2006, 02:32:35 pm
Two days ago I bought a new monitor, a Viewsonic 1912wb.  Previously I had a Viewsonic CRT monitor. 
To install the new monitor I followed the instructions.....shutting down the computer, disconnecting the old, connecting the new, turning on the new, then turning on the computer and booting the system.   Here is the problem........ I have a dual boot system, Mandriva and Windows XP;  I have always been able to select which OS I want to work in by "highlighting" either Mandriva or Windows with my up and down arrows on the keyboard.   After installing the new monitor however, my keyboard remains disabled during the boot-up process and because I have Mandriva as the default OS, I can only boot into it.  After my desktop installs, when the boot-up process is complete, my keyboard is fully functional again.

I don't know how my new monitor could be effecting the keyboard during boot-up...... and I never had this problem before installing this monitor.    I tried connecting the old monitor back to my system to see if I could isolate the problem, but the keyboard does not work with that monitor during the boot-up process any longer either.

If someone has the answer, or an idea........ that would be appreciated.

system:     Asus A7N266-E motherboard        512 MB RAM        Mandriva 2006        PNY GeForce 6200 AGP 8x 128 MB video card

thanks

Title: Re: Dual boot option not working
Post by: rugmonster on April 13, 2006, 04:05:52 pm
If you are using a USB keyboard, I would suggest finding the legacy device support in your BIOS and turn it on. This will make the BIOS treat your USB devices as legacy device (PS/2 style keyboard/mouse).

Title: Thank you !!
Post by: Perquisitor on April 13, 2006, 06:46:23 pm
Rugmonster,      Thanks so much for your help.   Your suggestion solved the problem.    I have no idea how the problem initiated......as I said, the previous monitor worked fine, and I've been using this same keyboard for a couple of years now.

No matter......... it's now working.   Thanks again.
