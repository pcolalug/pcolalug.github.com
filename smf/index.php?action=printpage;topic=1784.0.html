Title: Allowing user access to win partition
Post by: Brent on July 20, 2004, 11:01:13 am
Hi all,

As you all know, I run slackware.  I'm trying to setup access to /dev/hda1 using /etc/fstab.

As root I modified fstab to look like this regarding my ntfs and fat32 win partitions.

/dev/hda1        /mnt/windows     ntfs        ro,umask=000     1   0
/dev/hda3       /mnt/middleman   vfat        rw,umask=777     1   0

dev/hda3 is a fat partition that acts like a middleman btw linux and windows.  My ntfs partition is my windows partition.  I use it for my mp3 storage because it's ro.

Yet, I get permission denied trying to access either of these mount points under user mortal.

Help?

Brent
