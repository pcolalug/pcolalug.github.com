Been wrestling with getting BellSouth DSL service going.  BellSouth offers NO Linux support.  
Configuring the ethernet card and the DSL modem was straightforeward enough using RH9 GUIs.

Read in some RH documentation not to set a host name when using DHCP and I took their advice.  Don't know why that is important.

The problem rested with dealing with BellSouth's login proceedures.  Luckily my girlfriend's Windows XP computer was temporarily at my place.  I connected the DSL modem to it and installed BellSouth's software.  Rebooted and then doubleclicked on the MSIE icon and darned of a graphical login GUI did not appear.

I wrote down the URL for the login page and on my Linux box I made that URL the start page for Mozilla.  Without using the Windows box I'd have never guessed to do that.

If any of you try BellSouth DSL service rember to make the following URL your start page in your browser:  http://hometab.bellsouth.net.

Anyway, after living with connect speeds like 26.4 with a 56.6 modem out here on Pensacola Beach with a dialup account  the DSL is wonderful.
