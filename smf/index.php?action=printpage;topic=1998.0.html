Title: ClarkConnect
Post by: steve on November 04, 2004, 09:18:10 pm
Has anyone played with ClarkConnect (http://www.clarkconnect.org)?

I saw a review of it on another site and it looked pretty cool. I'm installing it now on a spare system I have. There is a FREE home version and a Commercial version. It can be installed as a Firewall/VPN Gateway or as a standalone server. It includes a mail server, web server, file server, FTP server, Photo Album, as well as a ton of other stuff. It's self installed from a CD or via FTP and uses a Web interface for managent.

Anyway, I'll let you all know what I think of it once I get it installed and tested. So far the installation is going very smoothly. Of course, I'm in the very beginning of the process.

Oh, also the latest version (2.2) has been out for about a year. They are suppose to have 3.0 out by the end of the month.
