I had posted several times earlier about my inability to install my broadband modem from Verizon on my Ubuntu Dell Netbook.  The modem works fine on my Windows Vista laptop.

From another forum I found out this modem won't work on Ubuntu 8.04, which I have.  Is there anyone who could upgrade my netbook from 8.04 to 9.10 or 10.04?  I have read instructions on how to do this, but it is too complicated for me.  I am a complete novice.  Thank you so much!

Info from the other forum (obtained from a review on Amazon):

"Don't worry about being a newbie

I found this link:
http://www.amazon.com/Novatel-Ovation-Broadband2Go-Virgin-Mobile/product-reviews/B002VPUNKE?pageNumber=2
which says "Ubuntu 10.04 is plug&play", and has easy instructions.

See here on how to get Ubuntu 10.04 on your laptop:
http://ubuntuforums.org/showthread.php?t=1497176 esp. post #5 .

Regularly updating Ubuntu 8.04 just gives you an updated
Ubuntu 8.04 version, and simply upgrading the Dell installed Ubuntu 8.04
software to Ubuntu 10.04 isn't possible according to the link above.

Maybe you can get support from a local Ubuntu or Linux support group
near you.

Good luck!

-Frans"

From Amazon:

Installation/activation wasn't too difficult. I had to use a Windows based computer (yuck) to do the initial activation. Once the device is activated you can use it with Macintosh, Ubuntu 9.10, and Ubuntu 10.04. The Mac is simple. Plug in device, open network preference, click connect under the Virgin Mobile tab. Ubuntu is a bit trickier but still easy to use. Ubuntu 9.10 required the VMDialer. Ubuntu 10.04 is plug&play. However, with either ubuntu there's a trick. When you insert the USB dongle give it a few seconds to load, it'll eventually show up as a cd-drive. Eject the drive first (click eject button) then it'll be recognized as a Broadbabd2go dongle. Use the network connections manager to connect.
