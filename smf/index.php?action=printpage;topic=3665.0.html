Database Administration and Web Application Services

 

Principle Duties and Essential Job Functions:

Participates in a Team environment, performs high level systems analysis, design, integration, programming and implementation of high priority systems/applications in a web-centric environment.  All tasks are performed under a full lifecycle methodology and participate in requirements definition, project management, applications design, development, documentation, testing, and deployment activities.  Performs analysis of requirements and programs using web technology products e.g., J2EE, JAVA, HTML, XML, or other tools directly related to development of products for the integration into a Portal. Prepares documents and designs/prepares technical reports, presentations, and briefs as required.

 

Basic Qualifications:

Prefer Bachelor’s with six (6) years related experience or ten (10) years in lieu with H.S. Diploma.  Two (2) years hands on experience with web technology products e.g., J2EE, JAVA, HTML, XML, etc.

 

Provide Database Administration (DBA) and Web Application services in a NIPRNET and SIPRNET environment.  Provide Oracle Database, Oracle Application Server, Microsoft SQL Server, Microsoft IIs, Apache, JAVA, and Oracle Reports and FORMS software services including installation, configuration, administration, maintenance, monitoring, deployment, and performance tuning. Develop processes and procedures for the administration of Databases and Web application servers.  Maintain Database and Web Application Enterprise Backup system.  Provide support for Database and Web application backups, administration, and maintenance. Create backup plans, provide restore services, verify backup completions, clean log files, and maintain documentation.  Develop and maintain procedures (Scripts) to automate startups/shutdowns and back up/recovery.

 

Security + certification is required. 

 

Key Competencies: Oracle Database and Forms and/or Oracle Reports.  Web Logic, Jboss and/or Oracle Application Server.  Apache and Tomcat and/or IIS.  Windows operating systems, Server 2000, Microsoft Office software, SMS type programs, OS patching, Antivirus applications.  Experience with J2EE, .NET, SQL, PL/SQL, and/or HTML/XML, scripting.

 

Clearance: Possess or be able to obtain and maintain a SECRET Clearance.
