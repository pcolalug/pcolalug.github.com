Title: Re: Good Place to Learn Z80 asm
Post by: Brent on November 01, 2006, 06:13:03 pm
I don't want to side track your post, but depending on your motivation for wanting to learn assembly, you might be better off using that time learning C, and using one of the many excellent compilers for that processor. 

If it's the learning experience you're after, go for it.  It'll give you a better fundamental understanding of what goes on behind the scenes, and make it easier for you to optimize code.  However, if your learning it just to write software for it, I'd recommend using C.
