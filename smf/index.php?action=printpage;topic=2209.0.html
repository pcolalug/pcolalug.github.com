Title: Bad Print Server
Post by: rugmonster on January 24, 2005, 09:43:51 pm
I got a Canon Pixma IP4000 (sorry, but no free Linux support yet). I got a Trendnet TE100-P1U print server for it. This one connects via USB 1.1 to the printer and is supposed to work great.

The first time I powered on my printer, it worked fine. That was until I changed the IP of the print server. Following that, it just wouldn't see a printer. I tried both of my printers and multiple USB cables, but no dice. I got an RMA number from NewEgg and I'll be getting a refund for it. I went ahead and ordered another Zonet since I know that will work. And yes, I got the one for the parallel centronix port.

Anyway, stay away from the Trendnet TE100-P1U if you are in the market. Thing had a really nice web interface too. Makes me sort of mad because they hit me with a 15% restock fee and I have to pay to ship it back to NewEgg. GRRR! :evil:
