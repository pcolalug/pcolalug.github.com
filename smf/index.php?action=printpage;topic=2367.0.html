... if I just click on it, it prompts to install, then says it's already installed...

Well, is it installed?

Have you tried installing from Mandrake Control Center? You can get there by going to  Launcher | System | Configuration | Configure your computer. It will ask you to supply the root password. Then go to Software Managment then *Remove*. (Not install yet). When that is ready type in "bittorrent" (without quotes) and hit enter. If anything pops up, it will tell you what is installed. That is an easy way to check what is already on the system. Then you can cancel out of that and go to the installer if you still need to install it.

If you are able to get on IRC then the ##mandrake channel at freenode has been really helpful to me. I have been hanging out there recently and am impressed with the helpfulness you can get there.

**I don't want to divert questions away from the forum, but I have found IRC to be an excellent source of immediate help.

While we encourage people to search google prior to posting here, this is a forum for support for users. Don't discourage people from asking questions.

Also, the lock issue sounds like permissions. To install a package, you need to be someone with permissions to do so, such as root. Try su'ing to root (or sudo is even better if you have it setup) and running that command again. If it says it's installed, then it probably is, just not in a menu anywhere. At a terminal type in bittorrent, or a part of it and hit tab. If it autocompletes, you can type "which bittorrent" (or whatever it autocompleted as) to find out exactly where it is installed.

There are other bittorrent clients out there, such as Azureus (my personal favorite), to try if you continue having problems with the vanilla BT client.
