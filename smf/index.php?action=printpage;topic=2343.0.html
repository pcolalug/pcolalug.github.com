Hey guys,

I recently downloaded a file manager called Rox (and it's dependencies).  For a while now I've been looking for a good light-weight file manager that was not tied to gnome or kde.  It took some work, but so far I'm very pleased with Rox and it's method of doing things.

It takes more work than konqueror, or nautilus.  But then, it's much faster than either of those.  Check it out if you need something light, but powerful.

Also, they've made a desktop around it as well.  Haven't tried that yet, not sure if i'd part with my beloved fluxbox
