39  Software / Desktops / Kommando ring menu on: February 19, 2008, 04:06:01 pm
Has anyone here used the Kommando ring menu application?
I just pulled it from the repository, and I like it. It runs well under IceWM, and unlike the XFCE panel, it doesn't block session logout when run from the startup script.
One thing that bothers me, though, is that it shows an icon in the center of the wheel depending on what your pointer is over, differentiating executables and folders, but it doesn't have any sort of text display. I think it would be much better if it displayed the name of the item you're hovering over so you can see what you're doing without having to be careful not to reuse an icon. I suppose it's possible I'm missing an option somewhere, but it doesn't look like it.

Want to try out the wheel launcher? Head over to http://www.kde-apps.org/content/show.php/Kommando?content=29514
Let me know if you have any thoughts on this.
