Title: Flac encoding
Post by: Brent on June 16, 2004, 01:39:16 pm
Hi all,

I recently bought some cds off of amazon.com for a pretty good price.   I know it's hard to believe but up until this year I never owned a walkman, or a cd player in my vehicle.

For some reason the linux cd players would not work.  However, I found a preferable solution.

I ripped the tracks using
Code:

cdparanoia -B "1-"


This ripped all the songs in the cd to a separate wave file.

Then I did this
Code:

flac --delete-input-file *.wav


So far I'm really impressed with the flac format.  It's completely lossless (100% cd quality) .
I think you can specify different levels of compression
