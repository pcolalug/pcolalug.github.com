Title: Re: abiword
Post by: lloydr on October 18, 2005, 11:18:13 pm
open a terminal and run the commands for the programs, in abiword, if you have no luck seeing what sort of error, try changing the verbose level
Code:
abiword -v

-v, --verbose=LEVEL                      Set verbosity level (0, 1, 2)

and the same goes for the other apps. when it crashes out, you'll probaly find missing libs or something. whatever is the cause, you'll find out from the console output.
