170  Club Stuff / Club discussions / Perhaps a Live CD Option on: March 01, 2005, 09:49:08 am
Some people may be curious about Linux, but aren't quite ready to install it on their machines.  A Live CD Linux Distribution will load Linux from a bootable CD.

From what I've read on distrowatch.com, SimplyMEPIS and Knoppix are two popular Live CD distributions.  Additionally, SimplyMEPIS allows installing to the hard drive once the user is ready.  I'm not sure if Knoppix has this feature.  Also, running Linux from a Live CD may help identify possible hardware detection problems.  One more thing, (I'm starting to sound like Columbo), Linux can be running on the user's computer in the time it takes to boot up.

I did a very quick Linux demo for a co-worker using a Live CD; he was quite impressed with the interface and all the included software.  Possibly another convert.

Anyway, just a thought.
