Title: Re: Slackware 11.0
Post by: geekforGod on October 05, 2006, 08:53:43 am
I installed Slack 11 last night. It is my desktop/development machine. Install went very smoothly, basically the same install as 10.2. I am using the 2.6.17 kernel from extra. Installed the Nvidia drivers with no problem, so I can play with 3d stuff. It actually found my sound card, which 10.2 could never do without the Nvidia drivers which mysteriously disappeared recently from their site. Only sound issue I had was with Flash(need my Strong Bad fix you know), but that was solved by changing the KDE setting for audio idle to a much lower value. I even got my webcam working with Kopete. I haven't really played with much else yet, but so far I am very happy with it.
