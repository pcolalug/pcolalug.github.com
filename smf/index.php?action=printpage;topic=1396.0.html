Title: wireless host name issue
Post by: rugmonster on February 13, 2004, 11:45:13 pm
add the name you called your system in /etc/hosts

Example, all my computers are named after Ender's Game characters. My main system is always Ender and the secondary is always Bean, and laptop is Dink. Well, I have an entry in my hosts files that looks like this:

Code:

ender       192.168.1.2
bean        192.168.1.3
dink        192.168.1.4
valentine   192.168.1.5
peter       192.168.1.6


A hosts file is like a simple DNS. In fact, DNS takes the place of hosts files, but on a small network, it is all that is necessary. Like DNS, it's a mapping of easy to remember computer names and their IP addresses.
