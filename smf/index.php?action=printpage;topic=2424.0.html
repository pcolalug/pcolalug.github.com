Title: Sound across your network
Post by: rugmonster on May 10, 2005, 06:52:41 pm
With all of the excitement of my new SuSE 9.3 desktop, I faced a little problem of not having speakers connected directly to the system since they were connected to my main Windows XP system. Since I'm running Gnome as my window manager, I took advantage of EsounD's ability to work across the network.

I found a cygwin build of EsounD (http://www.liquid-reality.de/main/projects/esound) for my Windows machine and fired it up (directions at linked site). Then, on my Linux box, I set the ESPEAKER environment variable to the ip of my Windows box, like so:
Code:
export ESPEAKER=jane.rugmonster.local:16001

Then I fired up Beep, selected the EsounD output plugin and magically, sound came out of my speakers connected to the Windows box, though slightly delayed due to the network latency of transferring an uncompressed audio stream. With a gigabit backbone, this would probably be elimintated, but this will suffice for now.
