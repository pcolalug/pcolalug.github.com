Title: Using a 17 inch monitor
Post by: danc on April 24, 2005, 10:01:57 pm
You are trying to send signals to your monitor (from the Xwindow display) that the monitor cannot handle (the signals are "out of range").

There may be graphical tools in Ubuntu to adjust the screen resolution / refresh rate.  Look through the menus for that, maybe under Gnome Configuration Center or something similar.

The other way to correct this would be to edit the configuration file manually.  To do this, first you will need to research the Horizontal and Vertical frequency ranges of your monitor.  Either look in the manual if you have it, or online, and find the numbers, usually something like 30-90 Horizontal, and 50-160 Vertical.  Then edit the file /etc/X11/xorg.conf and put those frequency range numbers into the "Monitor" section (you'll see some existing numbers there that apparently wrong).  Start / restart X windows after editing the file, and things should be fixed.

Not sure of your experience level, so if the above makes no sense, say so and we'll try another approach...   :lol:
