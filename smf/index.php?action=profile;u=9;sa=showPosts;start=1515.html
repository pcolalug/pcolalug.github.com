We (the group) recently had a strange install. The 2 or 3 different distros and versions we tried to install would get to a certain point and just hang. What was strange is that it happened on all of our installation attempts. Actually they all seemed to install fine, it was on the initial reboot that it would hang.

We finally pulled all the expansion cards (except monitor) and found that it was the sound card that was hanging the machine. You can probably find a cheap supported sound card amongst the group if yours gives you problems.

As far as the modem is concerned, just get broadband and forget dial-up. I know not everyone can do that, but I like mine so much that I want to encourage others to get involved. Then all you would have to configure is the NIC, which has been surprisingly easy in Linux for me.

Good luck.
