Title: Re: Looking for scripts
Post by: rugmonster on March 14, 2006, 12:03:00 pm
Brett, I believe you are wanting to look at your run levels. One of the major inconsistencies between Linux distros is how they handle runlevels, but typically, runlevel 3 is multi-user (full networking) console only and runlevel 5 is multi-user with X. I found this link (http://www.redhat.com/docs/manuals/linux/RHL-8.0-Manual/ref-guide/s1-grub-runlevels.html) from Redhat on how to specify the runlevel you want to start to when using LILO or GRUB.

If you want to start different X configs based on the kernel, then look into the uname command.
