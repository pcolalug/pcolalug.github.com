Title: RTCW Demo Problems
Post by: on May 26, 2003, 10:38:15 pm
Rugmonster,

If the server you are trying to join has the soundpak.pk3 file in the <RTCW/shrubmod> directory instead of the <RTCW/main> directory, AND you don't already have the soundpak.pk3 file, you will get this error message.

You can avoid this by downloading the soundpak.pk3 file and putting it in your <RTCW/main> directory.  Once you do this, your software does not try to get the file from the server, and all should be well.  There may be other reasons for getting this error message, but this is one of the biggies.
