Title: Working... not working
Post by: imnes on April 22, 2003, 07:33:32 am
I assume that KDE still uses the ArtsD sound server, or did they switch over to ESD?  Go into your configuration for XMMS and make sure it's set to use the ArtsD sound output.  If it's not going through a sound server and accessing your card directly then that would have blocked access for the rest of KDE from making sounds.  But if that's all it is the problem should have cleared up after a reboot if not on its own.
