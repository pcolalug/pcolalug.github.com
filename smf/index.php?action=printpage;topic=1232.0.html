Title: Using a USB thumbdrive in Linux
Post by: bhoff on December 26, 2003, 09:22:32 pm
How-to get a Lexar Usb-Drive working.
By: Brett Hoff

First I have not did this on my own!
I have brought all the information together and tried it for effect, But that is all.
As they say "If I have seen further than other men It is because I was standing on the shoulders of Great Men!"

I recieved for Christmas a brand new Lexar 128 meg Usb-Drive and found out right away that it was not supported to in Linux.
But that did not sway me. Being Linux is so flexible I simply did a google for Lexar usb in Linux and immediately was flooded with several pages of information.

Thankfully There are already modules built for Usb support as early as Kernel 2-4-7 and probaly earlier. I am using the 2-4-18 kernel with Redhat 7.2 and the following had me going quickly.

First I did a nice tail /var/log/message to see if my default kernel saw the Usb device and saw that there where several usb messages. At this point I knew that I had usb suuport and did not need to load any modules. If there had not been any listings in /var/log/message then I would have to load the following modules.
usb-storage and usb-uhci
they then list my Usb-Drive under /dev/sda1 showing as a scsi device.

All that is needed is to mount the Usb-Drive.
First create your mount point
mkdir /mnt/usbkey   #that is what I called mine anyhow...What ever you like.

Second mount the Usb-Drive
mount -t vfat /dev/sda1 /mnt/usbkey
now you should have your Usb-Drive mounted under /mnt/usbkey

Note: you may note that you have to use the -t vfat option to mount it.
I tried all the other options and that is the only one that worked.
Someone may try to convert to the ext2 or ext3 format...

To make things easier you may go ahead and set up the fstab file for the Usb-Drive. I set mine up so anyone could drop files to the drive.

vi /etc/fstab
Goto the first blank line at the end of your present listing and
i to insert then
/dev/sda1   /mnt/usbkey   vfat noauto,users,umask=000,gid=600 0 0
then the esc key and :wq to save.
There you go you are all setup to use your Usb-Drive .
It does not automatically mount you will have to type
mount /mnt/usbkey to do this.

Hope this Helps
Brett Hoff
