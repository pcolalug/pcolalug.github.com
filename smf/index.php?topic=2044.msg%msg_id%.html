Wow, I haven't had any wifi issues in linux yet, maybe I'm just lucky.  

I'd suggest trying Linspire, I've been using it on the laptop and  I've had no problems with my wifi.

If you have flash player:

http://media.linspire.com/howto/kiosk.swf
The top left says "About Linspire", click that and switch it to "Linspire Tutorials".  Then when it shows the new menu, pick "Wireless Networking".

Don't get freaked out by the mixture of Linspire and Lindows names in the tutorial - they still haven't finished changing everything over

If not:

In Linspire once you get it booted into X, check the L Menu (start menu) -> Settings -> Control Panel

In the control panel, click on the Network -> Network Connection tabs, then you can setup your wifi settings, security, interfaces etc.  Make sure to check 'enable' for your wireless card.

Wifi Success with Linspire (aka Lindows)
« Reply #13 on: November 26, 2004, 11:31:42 am »
Success!!!!!
I E-Bayed an Orinoco Wifi (802.11b) card.  It is a Lucent card re-branded by Cabletron.  Shipping & handling came to a total of $32.99.
It is connected to my home network via a Linksys WAP-11 (802.11b wireless access point).

Lessons learned:  
1.  I normally run my access points with 128 bit encryption.  This older card only supports 40-bit encryption, so I had to select 40/64 bit encryption on my access point.  
2.  Linspire correctly identified the card, but the Control Panel software does not expose all of the significant configuration elements needed.   Recommend that you use the control panel to set channel number and to set the mode to AD HOC rather than using the "Automatic" options.
3.  Using iwconfig in a console window, I had to manually set the RTS and FRAG parameters.
4.  After having done all of the above, nothing worked until I rebooted--then everything lit up!!!!.
