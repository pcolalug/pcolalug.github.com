Post by: dpeach on February 01, 2004, 02:15:25 pm
This works for Linux. I am sure it will work in BSD, but not certain. From your command prompt type:
Code:
mount -o remount,rw /

This forces the system to mount your root partition as read/write. Do your changes. Then you need to set it to read only before you shut down.
Code:
mount -o remount,ro /

Hope this works. Let us know.
