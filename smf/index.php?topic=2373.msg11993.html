Ubuntu 5.04 (Hoary) is released
« on: April 08, 2005, 06:44:50 pm »
Hoary is final and released today.  Installed and running here on my laptop.  Installation was smooth, things are all working well.  Of special note to me was that my wireless NIC (Atheros chipset, madwifi driver) was installed and working, "out of the box".  First time I've seen that from any distro.  The installer even asked me for the ESSID to use, and the WEP key.  Amazing.

The Synaptic package manager is surprisingly nice.  Used it to install Thunderbird 1.02, very painless.  This distro comes with Gnome 2.10 and kernel 2.6.10, and other stuff is very updated as well.  One thing I'd like to improve is the way fonts look, which is not quite as good as I've seen in Mandrake or Slackware.  Overall though, this is a promising release.

Dan

From a console, type:

Code:
sudo apt-get install kubuntu-desktop kubuntu-default-settings


Or if you're in gnome, select the System menu -> Administration -> Synaptic Package Manager.  Then click the "Search" button, and enter "Kubuntu".  Select those 2 packages and hit "Apply".

Should be all you need.  I haven't tried this yet so let us know if it works.
