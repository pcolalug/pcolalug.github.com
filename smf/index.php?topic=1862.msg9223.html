Well if you're not against emulation, I've had no problems running Dreamweaver in Crossover Office.  (I'm assuming you have a copy of dreamweaver).   Crossover Office is a Windows Emulator for Linux.  It does not require a copy of windows to work, it runs on it's own within linux.

http://www.codeweavers.com/site/products/cxoffice/?cw=89ae153df0759310fff25b21a9603626

The only version of crossover office I've tried is 2.1.0 I think (not sure what version they're up to).  Also I was using Dreamweaver MX (I couldn't get MX 2004 to run).

If you don't need all the fancy HTML drag-and-drop editing in Dreamweaver, then I also recommend trying Zend IDE.  (Although it's not free, it's built in Java and runs everywhere.  It's a pretty nice tool).

http://www.zend.com/store/products/zend-studio.php

It's kinda pricey though so unless you're gonna be doing a lot of PHP coding you probably don't need it.

My current system (Slackware + Dropline Gnome) has installed some html editor called Screem, but I haven't played with it yet so I dunno how it compares.
