Title: A couple more scripts
Post by: danc on November 05, 2004, 04:52:08 pm
Here are a couple of more scripts that I've made/modified, that I find useful.  Feel free to give 'em a try, and make improvements.  :lol:

This first one converts filenames to all-lower-case characters, which I prefer (for most things):
Code:

#!/bin/bash
 
# Converts filenames to all lowercase letters, by Dan Clough, 13Jan04
# Modified from an idea seen in forums on MandrakeUsers.org
# This version needs to be in the same directory as files to change...
 
   echo "Convert all filenames within this folder to lowercase? [yes|no]";
   read yn;
   if test "$yn" = yes; then
      for file in *;
      do
          mv -v -i $file $(echo $file | tr '[A-Z]' '[a-z]');
      done;
   fi


The following one converts MP3 files to WAV files, so they can be burned onto an audio CD which will work in car/home CD players:
Code:

#!/bin/bash
#
# This script converts all the MP3 files in a directory to WAV,
# for burning onto an audio CD.  Dan Clough, 7/12/03
#
for mp3file in *.mp3
        do
                wavfile=$mp3file
                wavfile=$(echo $mp3file | sed s/.mp3/.wav/)
                echo "Converting $mp3file to $wavfile"
                echo ""
                mpg123 -w "$wavfile" "$mp3file"
        done
