Looking at that screen shot, it looks like you were using IE. it is the only thing listed in the taskbar and the screen shot says IE. That was mistake number one.

However, I use Windows XP/2000 at my full time job and my part time job as well as one of my three computers at home. I don't get "ANY POPUPS at all" and I'm using IE. I do use FireFox primarily at work but also use IE when dealing with customer issues. And I use IE exclusively at my full time job. I don't have any popup blocker running but I did install Spybot on my home system and ran the immunize process in Spybot to protect IE.

People don't understand spyware programs like spybot and adaware don't work like anti-virus programs. They aren't normally memory resident and don't actually protect live spyware from infecting your system. (SpySweeper does do this, though) SpyBot requires that you click on some buttons, or in the latest version during the install to setup IE to block a lot of the spyware.

What 99% of it comes down to is where you spend your surfing time. I don't live on gaming sites, but when I do I make it a point to use Firefox. Once Windows gets a trojan or spyware on it, you'll be fighting a losing battle until it's all cleaned out.

Anyway, while Windows, by default, is more vulnerable to "Windows Attacks" it all comes down to how you use it and the time you take to secure it/protect it. I look at it like a Linux user always logging on as "root". That's not a smart move. You're asking for trouble. The same thing goes for Windows except, because of the number of dangerous programs targeting windows, it's 10 time more dangerous. If, like Linux, someone took the time to RTFM, read the security documents, and learn to use their OS correctly they wouldn't have 90% of the problems people complain about.

Just to wrap up, a couple of facts:
1.  I've been running and using Windows systems pretty much exclusively since Windows 95 was released.

2. I have never had a michine infected by a virus. (My scanner has caught a few, but never has the system been damaged/infected)

3. My Windows system has never crashed unless it was due to a hack that I was playing with or someone elses poorly written program. 9 times out of 10 if I'd taken the time to visit the programmers web site before installing, I'd have found a patch to fix the bug.

4. My Windows system has never been hacked. (As far as I know. And of course, my linux system has never been hacked as far as I know)

Finally if you think about it, It's not Windows that's bad. It's all of the bad software that targets it. (Try this, Install Windows on a machine. Do not connect it to the Internet or install any software. I bet you that unless you have a hardware failure or try to hack something, it will last as long as the computer. The same thing holds true for Linux.)

Post by: steve on August 14, 2004, 11:17:30 pm

That's a nasty one. Get HijackThis  (http://www.spychecker.com/program/hijackthis.html)and it will remove the doggy. You definitely have some spyware on there. Here's how to get rid of most of it. Also get cwshredder (http://aumha.org/freeware/freeware.htm) and run it. It will stop ceertain programs from grabbing your home page.

1. Right Click on My computer and click on Properties. Click on the System Restore Taband then turn off the system Restore. This will prevent Spyware from hiding in the system restore points while you're removing it. (Note: for those that don't like Windows, MS puts in system restore points for those software manufacturers that totally screw up the system when you install their software. This give you a good point to roll back to. Pretty smart, I think. I wish Linux had something like this so when I throw all kinds of trash in the system when doing a make/make install, I have a way to back out and get back to where it was when I started.)

2. Reboot and using the F8 key while booting, go into safe mode.

3. Find all temp directories and delete all files in them. Normally C:\Temp, C:\windows\Temp. Also delete everything in c:\windows\prefetch and c:\windows\Temporary Internet Files.

4. While in Safe mode run up Spybot and do a complete scan and remove. Make sure to have it check for updates.. (Only connect the computer to the network while checking for updates. Leave it disconnected until clean for everything else.

5. While in Safe mode run AdAware and do the same thing as with Spybot. (By default Adaware doesn't usually delete the spyware it only quarantines it. Make sure to delete it.)

6. Continue running 4 and 5 until they come up empty.

7. Run msconfig and go to the startup tab. Uncheck anything that you don't want to run when you startup. If unsure, google it.

8. Reboot the computer and hook it up to the network.

9. Go to Trendmicro (http://housecall.trendmicro.com/housecall/start_corp.asp) and run their on-line virus scanner. It's pretty good. (Run it until it doesn't find any infections)

If problems persist you can take it to Best Buy for a complete cleanup.
