Title: Re: Samba Help
Post by: rugmonster on July 26, 2006, 07:41:10 pm
If you really want to understand Samba and get it working right, read the official samba howto (http://us3.samba.org/samba/docs/man/Samba-HOWTO-Collection/). Every time I have to set up Samba, I will end up referring to this document and I have ever since I first began using it. Take your time, read slowly and it will eventually make sense. Also, what distribution are you using? It could be (very unlikely) that the samba config is being stored elsewhere. What exactly led you to believe you had it setup as a domain controller?

From the way it sounds, you don't really what you have setup right now, but that's okay. You need to scrap what you have done now and start over from scratch. Follow the howto and ask us questions. There is usually someone around in the IRC channel (#pcolalug on Freenode) that can help you out more immediately.
