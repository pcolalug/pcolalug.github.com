Post by: Brent on March 01, 2005, 05:22:04 pm
On the desktop you have many different options.  You can build your own, you can have someone build a system for you (as long as your guidelines aren't to stringent this can be cheap).  Or, you could just run linux on a generic pc which is probobly the easiest thing to do.  Lastly, you could by a special linux pc from a maker.  

But let's be honest here, this isn't solaris.  This OS was designed from the ground up to run on generic hardware, so unless your buying a laptop, this is the most expensive and probobly most underperforming option for the money.

So here's my breakdown, if i were building a system for gaming/linux, I'd build it myself, or at least order it custom built so that i could have it to my specifications.  If i just wanted to run linux?  Well, I've ran linux on some pretty old and 'out of date' hardware.  If i wasn't interested in pc gaming at all I'd just go with a base system with a cdrw/dvd drive.
