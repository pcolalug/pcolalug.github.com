Title: Java 6 and KDE
Post by: imnes on November 06, 2006, 04:34:21 pm
I was testing the latest build of Java 6 today, trying out the native GTK widgets and decided to try it under KDE.  A lot of the KDE distributions ship the GTK QT Style which uses QT as a theming engine for GTK/Gnome (so all your apps look consistent).  In the past this didn't work, java complained "QT Style unsupported" or some similar error, but now it seems to work.  So if any of you do run KDE and use Java apps, they should look a little better :)
