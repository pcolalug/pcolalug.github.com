I was digging thru the old post and found that Dan had volunteered a little script that I thought would make a neat little challenge.
How about we take this little script and only using BASH we see where we can take it.
Should prove very educational and entertaining.

Here is the script as posted by Dan
########################################################################

#!/bin/bash

# Script to check online status of machines on the LAN.
# Original version 3/16/05

Dan=192.168.254.104
Laptop=192.168.254.50
Teena=192.168.254.105
Michelle=192.168.254.103
Printserver=192.168.254.222

for name in Dan Laptop Teena Michelle Printserver
do
    eval ip=\$$name
    if ping -c 1 -w 1 $ip >/dev/null
    then
        echo -e "\e[32m$name ($ip) is online \e[0m"
    else
        echo -e "\e[31m$name ($ip) is offline \e[0m"
    fi
done
########################################################################

Let's post our results under here and see what happens.
