Title: Re: Boot Manager Uninstall
Post by: rugmonster on August 18, 2006, 02:01:39 pm
It depends on how you installed the boot manager. The recommended way is to install it to the partition and set that partition active (also known as setting the boot flag). When you do that, you don't touch the MBR. If by change, you did write your boot manager to the MBR, then you will have to replace it with whatever boot manager you wish to have. Overwriting the MBR isn't as catastrophic as it sounds though. Your partition table doesn't get touched, so all of your data is still there. You may just have to make the new boot manager aware of the various OSes you have installed. The most important question is the one Imnes ask: what exactly are you wanting to accomplish?
