Title: darker than normal graphics
Post by: danc on May 06, 2005, 12:25:52 pm
You didn't say what video card you are using, but if it happens to be an Nvidia-based one, and assuming you have the Nvidia accelerated driver loaded, try typing in a console:  nvidia-settings

This allows setting various things such as brightness, gamma, etc..  You may have to be root to run this command.
