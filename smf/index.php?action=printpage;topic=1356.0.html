Title: Nvu HTML editor
Post by: danc on May 18, 2004, 02:07:14 pm
Quote from: dpeach

Code:
./nvu-bin: error while loading shared libraries: libstdc++-libc6.2-2.so.3: cannot open shared object file: No such file or directory

The file above exists on my Mandrake 9.2 system, in the /usr/lib directory.  It is provided by the rpm package: libstdc++2.10-2.96-0.83mdk

I believe these are libraries needed by the C compiler/linker.  Not sure what distro you're using these days, but try installing that package and try again.
