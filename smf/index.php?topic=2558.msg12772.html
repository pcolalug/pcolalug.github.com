Yup, holding the reset button for 10 seconds will reset to factory defaults, so you should then be able to log in to 192.168.1.1.  Leave the username blank, and the factory password is "admin".

If you can't connect to it with a browser, remember that the machine you are using to do that with must also be on the same subnet (192.168.1.x).

1. Have you changed the essid on the router from the default Linksys?

2. Are you sure you enabled WEP and are you sure you're putting in the correct WEP key?

3. Driver/firmware(if required) loaded on his laptop (lsmod and iwconfig to see)

4. Ensure iwconfig sees the wireless extensions for the WNIC. (iwconfig)

5. Check to see if his laptop sees the AP by running (iwlist eth0 scan).

6. Your iwconfig above is incorrect. It should be:
Code:
# iwconfig eth0 essid blahid key xxxxxxxxx

Also, once you type that in, be sure to commit it with
Code:
# iwconfig eth0 commit

I wouldn't set a channel because it should find it and managed should be the default mode.

7. Why are you doing ifconfig eth0 down and up? Those aren't necessary.
