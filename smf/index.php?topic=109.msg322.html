« Reply #6 on: August 19, 2002, 02:44:13 pm »
Make sure you set your Mandrake computer to obtain it's ip address from a dhcp server. Go into the Mandrake control panel and setyup networking then go to the network card settings and set it to dhcp. You should not have to setup dhcpd for this as all you need is dhcpcd which is the client. Mandrake should set this up for you.

If you have this done then you should be able to run a shell and su to root. Then type ifconfig and it should show you your ip address.

Anyone running 8.2 and using a Linksys Router?
« Reply #11 on: August 21, 2002, 04:50:40 pm »
It sounds like your Mandrake box is set to IP address 192.168.1.1.

This has nothing to do with any of the servers installed. You can stop them all and your other computers still wouldn't be able to get out because the Mandrake computer is using the same address as your gateway. The one thing you need to do is remove the dhcp server program. You only need the dhcp client installed as Mandrake is only a client.

To do this, run up the Mandrake computer. Log in and, through the menu, go to Configuration/Packageing/Software Manager. Once you're in software manager click on the tab that says "installed" Then, in the "find" box type "dhcp" (without the quotes). The only program listed should be "dhcpcd" Make sure it is spelled exactly as I have typed it. Anyway, if there are any other programs listed, and you are sure you're in the Installed tab then put a check mark by the other programs and click on Install/Remove. This will remove any other dhcp programs you have. Once that's done restart your Mandrake computer. Your other computers on the network will also have to be restarted so they can get the correct IP information from the router.

Once the Mandrake computer is back up, run a shell and type "su" type in your root password to go into root mode. When you get the prompt, type "ifconfig" Make sure it doesn't list 192.168.1.1 as your IP address. If it does then you'll have to go back into Mandrake Control panel and setup your network adapter to use dhcp.

Steve
