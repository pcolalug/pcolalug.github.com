Title: Review: Programming PHP
Post by: imnes on May 02, 2003, 01:45:40 pm
Programming PHP is a great book for anyone interested in getting started creating interactive websites.  It takes you from the basics of the language, through advanced string maniuplation, database interaction, and even more obscure topics like using PHP to generate graphics and PDF documents on-the-fly.  The book ends with a great reference on the most common PHP functions.  The authors -- one of the the creator of PHP -- do a great job of delivering valuable information and insight in a concise and easy-to-follow manner.  The book includes enough examples to illustrate the concepts and keep the learning going.
