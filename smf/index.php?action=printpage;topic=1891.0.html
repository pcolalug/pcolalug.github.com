Very carefully!!

Mandrake has a tool built into the Mandrak Control Center that allows you to monkey with partitions, but there are certain actions that are destructive. For example. I have one partition that has never been formatted. I have it just sitting there so that if I need it in one application or another I can partition it for whatever. I can do whatever I want with that except make it bigger. Making it bigger means that one of my other partitions will have to shrink, and therefore destroy the data on the shrinking partition. That is what I assume anyway. It gives you a million warnings that you cannot shrink a partition.

I know there are other partitioning tools out there, but I don't know how destructive they are or not. This sounds like a good subject for someone to write an article about in our articles section of the forums. :hint hint:

Title: 100% conversion
Post by: rugmonster on October 01, 2004, 04:19:22 pm
parted (http://www.gnu.org/software/parted/parted.html) is a good tool for this. It supports most linux filesystems, including reiserfs, provided you install libreiserfs. You can create, delete, check, copy and most importantly resize partitions provided the start of the partition stays the same. I've used it before and find it very stable.
