Well, according to their site, the only way to legally get Lindows 3.0 is to pay them money. That tells me that they are charging for a product. As such, it would be considered stealing in this country to try to get a copy of it without paying the authoring company.

I think that sometimes there tends to be some confusion as to differentiating between opensource software downloadable for free and distributions using open source software for sale.

You are not paying for the open source software. You can download it from the many sites making the software. (Most of it on sourceforge)
What you are paying for is the authors ability to combine multiple opensource software packages and write an install routine to install it in an easy to use manner. You are also paying for any proprietary software included.

If you want the opensource packages used in Lindows then just go to sourceforge and download it. Otherwise, you should go to Lindows.com and pay them for the time and effort they put into developing their fine products.

Opensource and GPL software is free. The time people take to write routines to make all that software play together nicely in a distribution often is not free. If no one payed for Mandrake, SuSE, RedHat, Lindows etc. then Linux would IMHO be a dead OS. It's the people willing to support the distros with cold, hard cash that keep Linux in the forefront.
