Title: Book Review Cascading Style Sheets
Post by: dpeach on January 24, 2004, 11:25:44 pm
Cascading Style Sheets:
The Definitive Guide
by Eric A. Meyer
First Edition: May 2000

Though I had a basic knowledge of CSS (Cascading Style Sheets), I did not know the full power of them. CSS has the ability to do many things, but basically the strength of it over plain HTML is that you are able to define a "style" to be applied to your whole website. To change the look and feel, you simply change a few lines in your style sheet and your whole site is updated. CSS can be used even if you have a site that consists of just one page.

You can define how every H1, P or B tag will look throughout that page or site. Instead of using the FONT tag every time you change from a paragraph to a heading and back to paragraph. You simply define what the paragraph will look like with one simple line and it is applied to your whole site.

In other words, instead of typing:
Code:
<P><FONT color="purple" face="helvetica">Your text here</FONT></P>

You type:
Code:
P {color: gray; font-family: helvetica;}

The first example you have to type every time you want to use the <P> tag. The second example, using CSS, allows you type just the <P> tag and the style is applied. When you want to change the look of your page or site, you change it one time and it is done.

See CSS as "Site Maintenance Made Easy!"

Eric Meyer's book will put you on your way to using CSS. A basic knowledge of HTML is helpful to understand the first part of the book.
