Title: Mars rovers - Maestro software
Post by: danc on February 16, 2004, 02:31:17 pm
If you are interested in the Mars rover (Spirit, Opportunity) missions, you should check this out.  A public version of the software NASA uses to operate the rovers is available (Linux/Mac/Win), and it is very cool.  You can get updated packages of actual rover data to display and play with, from the same site.  I've been using the Linux version, and it works great.

It's here:  http://mars.telascience.org/home
