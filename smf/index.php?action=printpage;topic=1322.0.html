Title: centrino chipset / ATI radeon mobility support
Post by: lloydr on January 31, 2004, 12:10:55 am
well, if anyone runs into this problem in future...this is what I did:

compiled a 2.6.1-mm5(-mm5 patch gave me better centrino Pentium-M support) kernel with the toshiba acpi battery status fix patch.
just did a rpm2tgz on the ati drivers and went through an annoying process of making modules for the drivers and getting all that situated....and the intel centrino wireless I got running great using ndiswrapper

if someone is looking for support for a few of the not so well supported wireless chipsets in linux, give ndiswrapper a peek, I definetely recommend it because so far the results from using this centrino 802.11b wireless is pretty nice....isn't flawless, but it's working just fine  :D

and alsa sound works great in 2.6.1 kernel with the i8x0 modules
