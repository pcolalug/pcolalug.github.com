It is not generally recommended to write to an NTFS formatted drive from Linux.  It can be done, with the (somewhat) experimental "ntfs-g" driver, but I would never even consider using it on a drive which contained valuable data, as the chance for corrupting the entire drive is fairly high.  I don't know if PCLinuxOS includes support for ntfs-g in it's kernel, but that's what you'll need to use if you really want to do it.

Make sure you've got good backups...
