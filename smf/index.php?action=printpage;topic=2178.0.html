Title: OpenWRT
Post by: rugmonster on January 08, 2005, 10:40:27 am
Last night, danc pointed out OpenWRT (http://www.openwrt.org/) to me. I've been using the Sveasoft firmware on my Linksys WRT54G router, but found this distribution (it is a linux distribution) to be quite intriguing. Rather than getting a complete firmware with everything and the kitchen sink thrown in except for that one feature you really needed, OpenWRT gives you a base system with minimal functionality with an apt style package system for which you can load your own modules onto the router. I may try it out sometime soon.
