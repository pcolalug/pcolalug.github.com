Title: NVIDIA drivers in Debian?
Post by: rugmonster on December 19, 2004, 11:20:45 pm
You should be able to install the drivers directly from nvidia. It doesn't require a kernel rebuild because it just does an insmod for the module. The sources should be stored in /usr/src. It may or may not install the sources default.

Here is a HOWTO regarding this:

Andrew's Debian-nVidia HOWTO (http://home.comcast.net/~andrex/Debian-nVidia/)
