Title: Sound Blaster Audigy 2 - No Sound - FIX
Post by: imnes on July 07, 2004, 07:03:14 pm
Googled for a while and found this fix to get my sound working...

Open a console window, and run "alsamixer".  Using the arrow keys, scroll way over to the right and find the slider marked "Audigy Analog/Digital Output Jack".  With this slider highlighted, press the "M" key to mute the digital output.  This should give you your sound back, assuming the rest of the sliders are set to an appropriate level.

Then run alsactl store ; alsactl store 0 to get values to be reloaded after reboot.

This fix isn't needed under Linpsire (have I mentioned before how much I like that distro?)  But seems to be required most everywhere else.
