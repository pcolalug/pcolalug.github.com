36  Distributions / Debian Based Distrtibutions / Re: Hello to all Linuxians ... I fairly new to Pensacola and very new to Linux on: March 06, 2010, 05:37:35 pm
Welcome to Pensacola and the LUG Jim!

Danc's overview of the command line is a great place to start.  The terminal commands are by far the most important thing you need to learn when coming to linux, as once you get more advanced, you'll be doing a lot with bash.

Since you're new to Ubuntu, I'd recommend the Linux Starter Pack, which features a lot of overviews of common open source applications.
Link:   http://www.tuxradar.com/linuxstarterpack


Another great resource is the Ubuntu Pocket Guide, though it may be out of date on some things (like GRUB).
Link:   http://www.ubuntupocketguide.com



If you need any assistance feel free to post your questions or join us in our IRC channel:

Server:     zelazny.freenode.net
Channel:  #pensacola_linux

Or, use the webIRC client here:  http://pcolalug.org/irc/irc.cgi
