Ramdisk
Steps to create a ramdisk in Linux.
1. Edit /etc/lilo.conf adding one of the following lines depending on age of kernel.
ramdisk_size=10000 for 10 megs or
ramdisk=10000 for 10 megs.

2. Reboot system

3. From command line or set as script.
Mkdir /tmp/ramdisk0
mke2fs /dev/ram0
mount /dev/ram0 /tmp/ramdisk0

4. Use df -k to see the ramdisk.
Can be used like harddrive only faster. Note: anything stored here is lost when power is removed.
