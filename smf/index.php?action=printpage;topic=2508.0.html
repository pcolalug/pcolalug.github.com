Title: .bin extension
Post by: Brent on July 01, 2005, 09:04:11 pm
Well, here's how to fix it.

go root in the console, start up mc, have one window of mc point to /usr/bin and the other point to /yourrealplayerpath.

Now, press ctrl x, ctrl s.  It will ask you to make a symlink. Hit ok.  You should be able to start it directly from there, after you edit your fluxbox menu.

um.... there are quicker ways to make a symlink without using mc.

Code:

ln [OPTION]... TARGET [LINK_NAME]


for symbolic link in your particular case you'd do
Code:

ln -s /pathtorealplayer /usr/bin/realplayer
or
ln -s /pathtorealplayer /usr/bin/realplay

thought it was realplayer command, but above is for realplay or realplayer, but you get the idea

or probaly a little more appropriate,
Code:

ln -s /pathtorealplayer /usr/local/bin/realplayer
or
ln -s /pathtorealplayer /usr/local/bin/realplay
