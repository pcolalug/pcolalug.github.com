Chances are, as new as Ubuntu is, you won't find a book on it yet.

I looked up your laptop and it seems that most any recent release of a distribution should work.  I'm not entirely sure about the integrated wireless, but there is NDISWrapper if there is no native driver available.

Now here comes my opinion, so take it as that. I've tried Ubuntu and while they have some great ideas, there is a lot of polishing that remains to be seen on most of their "easy to use configuration apps". They're configuration wizards often crash and you wind up go into configuration files instead. There is however, a great support community with lots of Ubuntu fans frothing at the mouth to help you. Fedora on the other hand I have not had as much experience with. I'm sure some of this is true, but I personally have always felt like it was a more sluggish system. Redhat and later Fedora Core releases just never did it for me, so I never gave it much time.

As for newbie friendly systems, I like Mandriva and SUSE. Especially SUSE. I currently run Debian (which Ubuntu is based off of), but I'm the type that tends to avoid pretty wizards in favor of digging in the config files for the most part. My overall suggestion, try a few distributions out. Find the one that fits you. I've gone from using Caldera OpenLinux to Slackware to Debian to Gentoo to Mandriva when it was Mandrake to LFS to SUSE to Ubuntu and finally back to Debian. That's been since '99 and I'll still try different distros out as they pique my interest.

Oh yeah, books on linux are waaaaay overrated. Everything you need for general use is available online. Google is the best book alternative you have. Save your money and definitely don't spend more than you have to at the UWF bookstore. They price gouge there!

I was trying to give you a subtle nudge to go search Google. Ubuntu has documentation out there, and good documentation at that. There is another site out there called ubuntuguide.org that will tell you exactly how to do particular tasks, sadly with no explanation of why you're doing it.

For Firefox, I did a search on Google using the terms "ubuntu firefox 1.5". Bam, first page was from the Ubuntu wiki on how to install it and an explanation as to why it won't be going into the backports. If you want to find out about the backports, I bet a search of "ubuntu backports" would find you something.

If you want to install the Nvidia drivers, GO DOWNLOAD THEM AND READ THE INSTALL GUIDE!

Wireless networking...Did you even click on the links provided?!?!

I want to help you, but you have to help yourself first. I won't speak for the rest of the group, but I will speak for this forum. This forum is not a Google proxy. Don't abuse it as such. You will not learn a thing if you ask for the answer without looking yourself. There is a lot to be learned in simply looking for the answer. I got curious about things I had no idea existed in quests to fix problems. I would read about something that sounded foreign, do a quick search and go "Oh wow, that's cool! I'll come back to that later and check it out".

Head over to http://www.google.com and do some searching. I promise, you will quickly find an answer to most of your questions, probably find some new questions, but stop back by Google and search some more. If you have a specific problem installing a driver, such as it doesn't work right or something gets messed up and you don't know how to make it go back (keep a log of what you've done...on paper), come back to us.
