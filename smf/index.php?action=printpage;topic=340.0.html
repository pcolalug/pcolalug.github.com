You're really not gonna like me for this probably.  But I put together my own version of your prog just to see if I had the right idea.  You can run it and test it with your own file to see if it prints out the right results, and if I did what the directions asked.   Anyway, here it goes.

Code:


/********************************************************************
 *
 *  Assignment: Exercise 1-13. Write a program to print a histogram
 *  of the lengths of words in its input. It is easy to draw the
 *  histogram with the bars horizontal; a vertical orientation
 *  is more challenging.
 *
 *  The short directions are a little hard to understand, so let
 *  me know if I went off and did something completely wrong.
 *
 *  Solution by imnes@go.com
 *
 *******************************************************************/

#include <stdio.h>
#include <string.h>


/* Define TRUE and FALSE just to make program more readable. */
#define TRUE    (1==1)
#define FALSE   (1==0)

/* The max length word I want to track is set here, default is 11. */
#define MAX_WORD_LENGTH 11

/* This is where I store the word counts.  If (for example) a word
 * has 8 characters, then Frequency_Table[8]++;  For any words > the
 * MAX_WORD_LENGTH, store at MAX_WORD_LENGTH+1;                   */
int Frequency_Table[MAX_WORD_LENGTH + 1];

/* This flag will be set to true once the EOF has been reached. */
int Finished = FALSE;

/**
 * Clear the frequency table, setting all values to zero.
 */
void Clear_Frequency_Table(void)
{
int i;

for(i=0; i<MAX_WORD_LENGTH; i++)
Frequency_Table[i]=0;
}


/**
 * Record a word in the frequency table.  Basically just send the length
 * of the word and it gets added to the frequency table.  Didn't need to
 * make a seperate function for this, but why not.
 */
void Record_Word_In_Table(int length_of_word)
{
if(length_of_word >= MAX_WORD_LENGTH+1)
Frequency_Table[MAX_WORD_LENGTH+1]++;
else
Frequency_Table[length_of_word]++;
}


/**
 * This function will return TRUE if the given char is an alphanumeric
 * charcter, and false for all other chars.
 */
int Alpha_Numeric(char c)
{
if( (c>='A' && c<='Z') || (c>='a' && c<='z') || (c>='0' && c<='9') )
return TRUE;
else
return FALSE;
}

/**
 * This function will get a word from the input stream.  A word is
 * defined for my purposes as a stream of characters that begins with an
 * alphanumeric character, and ends with a non-alphanumeric (such as a
 * newline, whitespace, control code, or whatever).
 */
int Get_Next_Word_Length(void)
{
#define WHITESPACE 0
#define ALPHANUMERIC 1

int last_flag = WHITESPACE;
int flag = WHITESPACE;
char c;
int word_length = 0;

while(Finished==FALSE)
{
/* Get a character. */
c=getchar();

/* If this was a EOF marker, then we're done. */
if(c==EOF)
Finished=TRUE;

/* Is this an alphanumeric or whitespace? */
last_flag = flag;
if(Alpha_Numeric(c)) {
flag=ALPHANUMERIC;
word_length++;
} else {
flag=WHITESPACE;
}

/* Did we already get characters? */
if((flag==WHITESPACE) &&
  (last_flag==ALPHANUMERIC) &&
  (word_length>0))
/* If so, we got a word.  Let's exit. */
break;

/* If not, we're still waiting for a word. Continue. */
}

/* We got a word, return it's length to the main program. */
return word_length;
}


/**
 * Show the histogram results.
 */
void Show_Histogram(void)
{
int Max_Word_Count=0; /* Holds largest word count */
int i,c; /* Temp Counter Variables */

/* Find maximum word count.  This is used for the Y
* axis of our table. */
for(i=0; i<MAX_WORD_LENGTH+1; i++)
if(Frequency_Table[i]>Max_Word_Count)
Max_Word_Count=Frequency_Table[i];

/* Print Table Header */
printf("\nHISTOGRAM OF WORD LENGTH FREQUENCIES\n");
printf("====================================\n");
printf("YAxis represents the frequency a word length appeared.\n");
printf("XAxis represents the lengths of the words (0-%d).\n",
MAX_WORD_LENGTH);

/* Print the BARS */
for(i=Max_Word_Count; i>0; i--)
{
printf("%3d |", i);

for(c=1; c<=MAX_WORD_LENGTH; c++)
{
if(Frequency_Table[c]>=i)
printf(" * ");
else
printf("   ");
}
printf("\n");
}

/* Print the bottom line and word lengths */
printf("    +");
for(i=1; i<=MAX_WORD_LENGTH; i++)
printf("---");
printf("\n");

printf("    ");
for(i=1; i<MAX_WORD_LENGTH; i++)
printf("%3d", i);
printf("%3d+\n", MAX_WORD_LENGTH);
}


int main(void)
{
/* Used to store the length of the word from the user. */
int Length;

/* Prompt the user for input. */
printf("Word Length Histogram.\n");
printf("Enter some text.  ");
printf("EOF will stop the program and show results.\n");

/* Continue getting words until the EOF is received. */
while(Finished==FALSE) {
/* Get the next word's length */
Length=Get_Next_Word_Length();

/* Save it in the frequency table */
Record_Word_In_Table(Length);
}

/* Display the results. */
Show_Histogram();

/* Exit */
return 0;
}

I think I've found your problem.  The code portion that reads:

Code:

 if (c != ' ' || c != '\t' || c != '\n') {
            pos = IN;
            ++numchar;
         }


will never be evaludated to true.  What you're saying is
IF character Is Not a Space OR character Is Not a Tab OR Character Is Not a Return then...

From my understanding, if any one of those evaluates to true, then the statements inside the block will be executed.  So if C is '\t' then c!=' ' is true and c!='\n' is also true, the statements will be run.  I think you want to use AND operators instead of OR.  Try this:

Code:

if (c != ' ' && c != '\t' && c != '\n') {
            pos = IN;
            ++numchar;
         }




Which reads as:
If C is not a blank space AND C is not a tab character AND C is not a Return then execute the statements.  They must ALL be true for the next two lines to run, whereas with the OR's any one of them can be true and it will run.  Wow, this is confusing I hope I said it right and it makes some sense.  Try it and see how it works for you.
