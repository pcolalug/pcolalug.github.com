http://media.linspire.com/howto/kiosk.swf
The top left says "About Linspire", click that and switch it to "Linspire Tutorials".  Then when it shows the new menu, pick "Wireless Networking".

Don't get freaked out by the mixture of Linspire and Lindows names in the tutorial - they still haven't finished changing everything over ;)

Title: Wifi Success with Linspire (aka Lindows)
Post by: tuxmoose on November 26, 2004, 11:31:42 am
Success!!!!!
I E-Bayed an Orinoco Wifi (802.11b) card.  It is a Lucent card re-branded by Cabletron.  Shipping & handling came to a total of $32.99.
It is connected to my home network via a Linksys WAP-11 (802.11b wireless access point).

Lessons learned:  
1.  I normally run my access points with 128 bit encryption.  This older card only supports 40-bit encryption, so I had to select 40/64 bit encryption on my access point.  
2.  Linspire correctly identified the card, but the Control Panel software does not expose all of the significant configuration elements needed.   Recommend that you use the control panel to set channel number and to set the mode to AD HOC rather than using the "Automatic" options.
3.  Using iwconfig in a console window, I had to manually set the RTS and FRAG parameters.
4.  After having done all of the above, nothing worked until I rebooted--then everything lit up!!!!.

Title: Wireless woes...
Post by: danc on December 23, 2004, 08:48:56 pm
Thought I'd add on my recent experiences in the Wireless Wars...

I decided to give myself an early Christmas present, and go wireless on the laptop.  My wife approved of this mainly because it would mean I no longer would have the Cat-5 cable running down the hallway into the living room where I like to use the laptop...  Anyway, after a little research, I decided to go with the following hardware:
Linksys WRT54G wireless router/AP
Netgear WG511 NIC (pcmcia)

Went over to Best Buy and got them both.  Added the router to my existing LAN, acting just as an AP, and it worked just fine.  So far so good...

The WG511 uses the "prism54" driver, and so I went and got that, and installed it.  There's a great website for this software at:  http://www.prism54.org .   OK...... after about 7 hours of fighting with it, and using the IRC channel #prism54 to great benefit, it was determined that this newest WG511 has a different chipset than previous WG511's, and wasn't supported.  I was getting no lights or activity out of the NIC at all...

Decided to try the Netgear WG511T model instead, so went back to Best Buy and exchanged for it.  This card uses the Atheros chipset, and the "madwifi" driver.  Installed that driver, plugged the NIC in, and EUREKA!!!!  It worked the first try.  I've now got WEP encryption working too, and am very pleased with how things are working.

I'd recommend this wireless NIC for use with Linux.  The "madwifi" driver software is very much "beta", but seems to work flawlessly.  Happy Holidays!

Dan
