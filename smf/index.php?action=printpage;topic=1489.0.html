Post by: danc on March 19, 2004, 04:08:55 pm
Probably the easiest way would be to run the program "xwmconfig" from the command line...   :lol:

If you'd like, you can browse the directory /etc/X11/xinit where you'll see the scripts for starting the various desktops.  I believe all the program above does is set the symlink "xinitrc" to point to the desired desktop script.  You can change this manually if you're comfortable with that.

On a related note, if you'd like X to start at bootup, just change the default runlevel from "3" to "4" in the /etc/inittab file (the "id:3:initdefault:" line).

You mentioned you are using Gnome...  If you haven't already, you really should check out Dropline Gnome at http://www.dropline.net/gnome
This provides you with an auto-updater and keeps everything very up to date, not to mention that it looks beautiful too.  Check it out!

BTW, here is an EXCELLENT "book" on Slackware, provides all the answers you'll need...  http://www.slackware.org/book/

Glad you're enjoying Slackware!
