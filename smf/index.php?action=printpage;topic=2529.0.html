Title: Breaking SuSE
Post by: imnes on July 16, 2005, 07:54:09 am
SuSE 9.3 has some nice recent stuff in it, but if you want more bleeding edge / experimental (ie really neat) stuff there are resources out there.

You'll want to get Red Carpet installed (it's on your CD's, search yast for red-carpet, rcd, and rug).  

First off to get SuSE's Gnome behaving more like Ximian desktop, follow the instructions here:

http://primates.ximian.com/~rml/sanitize/

You should go into YaST and install all the perl-XML packages.  Then you can upgrade / install the latest Mono packages from here:

http://www.go-mono.com/download/suse-93-i586/

Now, you can get more updates to Gnome (through the ULB usr-local-bin channel) and lots of other SuSE specific projects, instructions for those are here:

http://www.evolutioncolt.com/~spayne/blog/?p=11

Will post more resources here as I come across them.
