Title: two part question (xine)
Post by: danc on April 12, 2005, 10:53:47 pm
Quote from: LinkZero
i've got mplayer, but i hardly ever use it since it has to be opened in console.  how do i open or install the codec package?  or do i just need to put it somewhere?  i know these are dumb questions but i'm new.


1.  Download the codec (with your browser)
2.  In a terminal, as root, in the directory you saved it in, type:
      installpkg <filename>     (where <filename> is the name of the codec package)

That's it!
