Scheme is definitely a cool language.  Another fun lisp derivative is LUSH
(http://lush.sourceforge.net/) , which gives you a huge number of support libraries
which are really nice for getting started with the language.  (Open a v4l window and
apply a sobel filter with only 6 or so lines of code!) It definitely made it easier to get to
the "Aha!" point for me. (I learned scheme via a class so I got forced into it with given assignments.)
It can also be compiled down to C and then to platform binaries for fast execution! [I think]

Quote from: rburt3

using the language, but I haven't yet found a good excuse to use it more in my spare time.
