Title: Re: Backup
Post by: rugmonster on January 10, 2007, 12:31:29 pm
I use dar and bzSarab for system backups. dar is the actual backup utility and bzSarab is a script that works with dar to do rotating backups, like you are wanting. Running the backup jobs through a crontab entry is typically how it's done. If you run into the problem of files being altered during the backup, you will want to look into LVM and filesystem snapshots. If you are not using LVM already, then you would have to at least make room for a new partition that would be setup for LVM. If you are interested, there is a howto out on the web.

As a side note, I keep my media storage and home directories backed up to an external drive using rsync. It works very well for me and should I need to restore something, I only have to copy the files back over.
