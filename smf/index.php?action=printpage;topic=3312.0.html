Title: Elive Linux Distro
Post by: djoed_pcola on February 22, 2008, 08:36:19 am
Another cool Linux distro.  Elive is based on Debian and uses the Enlightenment window manager.  Elive can be run as a live CD, or installed on your machine.  I installed it on an old 450 Mhz machine with 256 MB of memory and have been impressed with the results so far.  Enjoy.
