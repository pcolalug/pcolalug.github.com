Title: Google Earth Beta V.4
Post by: L.Rentz on June 14, 2006, 12:50:00 pm
Just if you didn't know, Google has release Google Earth Beta V.4 They have a linux version available now.  Get it here: http://earth.google.com/download-earth.html  It works great on my laptop running SuSe 10.1

Lee

Per Google:
Linux System Configuration
Minimum configuration         

    * Kernel: 2.4 or later
    * glibc: 2.3.2 w/ NPTL or later
    * XFree86-4.0 or x.org R6.7 or later
    * CPU: Pentium 3, 500Mhz
    * System Memory (RAM): 128MB
    * Hard Disk: 400MB free space
    * Network Speed: 128 Kbits/sec
    * Screen: 1024x768, 16 bit color
    * Tested and works on the following OSs:
           Ubuntu 5.10
           Suse 10.1
           Fedora Core 5
           Linspire 5.1
           Gentoo 2006.0
           Debian 3.1
           Red Hat 9
