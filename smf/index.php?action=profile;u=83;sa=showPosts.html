15  Distributions / Other Operating Systems / Re: A Very Much Biased, but Brief Vista Review on: January 01, 2007, 10:49:51 am
We've been using a copy of Vista beta 2 (or recently 3?) for quite a few months now at work.

Underwhelmed is an understatement.  I've yet to see any real benefit from using vista.  My biggest complaint is the ram usage of vista.  We have it trimmed down, but it still uses 350-400 meg of 512 ram on the system.  This leads to "vista bugs" in the software we test, where our code crashes simply from the lack of ram on the system (yes, our code is in java  )

IMO each release of an OS should not be significantly more bloated than the last.  However, with windows this seems to be the case.  Windows 2000 and XP required about 128 and 256mb of ram respectively to run decently on a system.  Vista on the other hand seems to require 800 to a gig to be usable.  Given that I've seen no *real* usability improvement since 2000 (with the exception of wireless drivers in XP), how in the world can they justify this?  I currently have windows server 2003 nestled on a small partition on my old hard drive.  The last time I checked, it used 128mb of the 512 I had on the system.  This is proof that microsoft *can* produce a system that is both functional and light weight, but they choose not to.  It seems they are in collusion with the hardware industry to boost sales of new computers.  Of course, given the fact that a computer from 8 years ago can still happily chug away checking email, and writing documents, I can see where the hardware vendors would be concerned.

OTOH, I have a linux / fluxbox install on my parents machine.  Ram usage?  A whopping 30mb  .
