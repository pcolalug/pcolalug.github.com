hey todd,

if memory serves, i believe varchar can hold a max of 255 chars.

if you want to store more than that, you'll need to use a TEXT format instead. note that you cannot index text fields.

apostrophe's are screwy in mysql, therefore, you'll need to do a simple regex to replace ' with \' and then, revert back to a standard apostrophe when outputting the data.

i know php does this on the fly with magic quotes...very handy.
