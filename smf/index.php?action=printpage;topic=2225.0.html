Some of that software may work, but most probably will not.  There have been compiler and library changes since Mandrake 9, and it's always a good idea (if you're using pre-packaged software) to stay with the same version as what you've got installed.  However...

One of the great features of Mandrake is it's "URPMI" functions, which allow you to pick and install software from a GUI (or command line if you prefer).  The key to using URPMI is setting up the proper online software package repositories.  Once they are set up, you can always browse the latest packages from the internet, which will likely be newer than whatever you may have on CD.

The most important first step with URPMI is to set up the "Updates" source.  Do this by going into Mandrake Control Center, then the Software Management section.  Then choose Media Manager, then Add, then Security Updates.  Then choose a Mirror (I've had better luck with the ones in France, especially "proxad.net" or "club-internet.fr").  This will download a large list of updates available.  Then...., back up a screen or two, and choose "Updates" (same screen that you chose Media Manager from earlier).  This should download (a LOT) of updates from the internet, and install them on your system.  Now you have an up to date installation!

Now, for further software goodies, you can set up other online sources.  The easiest way to get this done is to visit the following site, and follow the instructions given there:
http://www.urpmi-addmedia.org/index2.php

Another source of excellent packages for Mandrake (like Firefox, Thunderbird, Gaim, and much more) can be set up at this site:
http://norlug.org/?op=rpms
Just follow the "URPMI repositories link on that page.

Once you have these sources set up like this, you can just go back into the Media Manager, and browse them all, selecting what you want to install.  It's great!

Finally, another good Mandrake site for lots of different information is here:
http://www.mandrake.tips.4.free.fr/index.html

If you have any troubles with this stuff, or are looking for some specific packages and can't find them, let me know...  Good luck!
