Title: Re: kde light...where?
Post by: geekforGod on February 15, 2006, 05:19:40 pm
The load time should be better, I believe. No library loading. The executable would take up more space, since it has the library code included, but not large enough that anyone with a fairly recent system would mind. I think RAM usage would be higher, but I am not sure by how much. Probably not enough that anyone would notice. The main advantage to the shared library is only having to update one library when the library is changed because of a vulnerability or new functionality, instead of tracking down every application that uses it and recompiling or downloading a new binary.

I don't remember everything completely and didn't feel like breaking out my OS book, but that should be close enough, I think.
