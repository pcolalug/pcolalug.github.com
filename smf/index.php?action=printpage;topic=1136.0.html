Post by: imnes on November 07, 2003, 03:42:28 pm
What are you trying to do this on?  

If you're on BSD or Solaris Intel and you're using a Linux port of Java then you'll need a linux build of firebird.  All native or all emulated.

If you're doing this on Linux, (a recent linux) make sure you're using the gcc-32 version of the plugin from JDK 1.4.1_2 or whatever the recent is.
