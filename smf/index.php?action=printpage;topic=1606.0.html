Title: Shortcut keys (revisited)
Post by: dpeach on May 14, 2004, 10:25:57 am
This is a followup on a previous post (http://www.pcolalug.org/phpBB2/viewtopic.php?t=825&highlight=shortcut) where I was looking for a way to do a Cntrl+Alt+[any other key] to launch an application.

KDE 3.2 has given me the functionality that I seek. From the KDE control center go to Accesability | Keyboard Shortcuts. This was in KDE before 3.2, but not the ability to define your own combinations. The second tab is the Command Shortcuts. From there find the application that you want to launch and define the shortcut for it.

Thanks KDE!
