The following positions are available and currently open.  If you are interested or know someone who might be drop me a line.

Project Manager

Principle Duties and Essential Job Functions:

Leads medium and large stand-alone projects.  Responsible for the management of project Integration, Scope, Schedule, Cost, Quality, Human Resource, Communication, Risk, and Procurement activities, as required.  Contributes to the creation of project charters, scopes of work, and other relevant documents.  Defines work breakdown structures and project plans.  Defines work units and assigns them to appropriate project personnel for completion.  Manages project work according to an agreed-upon baseline of schedule, resources, and deliverables.  Manages stakeholder expectations through a well-defined change order process.  Possesses expert knowledge of Microsoft Project or other project management software, according to the needs of the project.  Accomplishes project work in a "matrixed" organization, where resources may or may not belong or report directly to the Project Manager.  Interacts with all levels of project stakeholders.  Advises project sponsors on the resources (financial, human, technical, and time) required to produce desired deliverables and results.  Updates project plans, status reports, and other project-related deliverables.  Works under the cognizance of the Project Office, if existing. 

Basic Qualifications:

Prefer Bachelor’s with 8 years, 12 years in lieu with H.S. Diploma.

Project Manager should possess eight years within the last ten calendar years of intensive and progressive IT experience and excellent communication and negotiation skills.  Project Management Professional (PMP) certification is desired.

Candidate must have the ability to pass a background investigation, and to obtain and maintain appropriate commercial certifications.

Clearance:  Possess or be able to obtain and maintain a SECRET clearance.

 

 

Individual Augmentation Support (NAS Pensacola)

Principle Duties and Essential Job Functions:

Exercises analytical techniques when gathering information, defining work problems, designing a solution, developing procedures to resolve issues, and quality assurance. Reviews requirements, analyzes data, compares alternatives, prepares specifications, and resolves processing problems. Provides direct client contact on deadlines, obstacles, expectations and status. 

Possess excellent number recognition, formula, and/or query skills.  Uses creativity, judgment, and initiative in solving highly complex technical problems. Demonstrates cohesive teambuilding in support of the project team.

Basic Qualifications:

Prefer Bachelor’s with 4 years related experience, 8 years in lieu with H.S. Diploma.

Review Individual Augmentation (IA) requirements from BUPERS. Run reports for all viable candidates based on new IA requirements received from BUPERS. Research possible alternate designators, NEC/NOBC, and/or grades. Review nomination submission to identify qualified candidates with least impact to command. Request PRD/EAOS extensions when needed with respective detailers. Liaison, as directed, between client, BUPERS and sub-claimancy questions, issues, etc. Enter IA new entry, nominations, notes, etc into the IA tracker. Create file folders for new IA, file communications in respect to specific IA tasking. Provide support to timelines, IA daily report, and weekly updates.

Knowledge and/or experience with Navy Ratings and Rank structure, a plus. Administrative Navy Rate experience desired.

Skills: MS Excel experience is a must, MS Office, Power Point, Access, quality assurance, and/or strong number recognition.

Candidate must have the Ability to pass a background investigation and the ability to obtain and maintain appropriate commercial certifications.

Clearance: Possess or able to obtain and maintain a SECRET Clearance.

 

 

Help Desk (Casual)

Provides help desk support for highly complex integration projects. Provides support and troubleshooting of system software programs.  Responsible for support and troubleshooting of applications dealing with the overall operating system, such as sophisticated file maintenance routines, networks, or mathematical/scientific software packages.

Experience in desktop and server support using Windows 2000 and 2003. Desired experience includes UNIX and Windows 2003 Operating Systems. 

High School plus two years IT or help desk experience. MCP or A+ certification preferred.

This position is "on-call" and may provide 0-40 hours per week (Monday-Friday) across two shifts, 6am-2pm and 2pm to 9pm.

 

 

SIPR Administrator

Principle Duties and Essential Job Functions:

Leads and participates in highly complex system integration projects. Installs, generates, maintains, tests, debugs, and integrate system software programs with COTS/GOTS applications and support equipment, develops instructions for integration of operating system software, and modifies/maintains integration packages via releases and utilities from manufacturers in a multi-operating system and/or network operating system environment. Evaluates existing operating systems/COTS/GOTS software and recommends changes. Optimizes systems software performance. Responsible for applications dealing with the overall operating systems. Engineers highly complex software that may relate directly to operating systems, their performance or devices controlled.

Basic Qualifications:

Prefer Bachelors with 4 years, 8 years in lieu with H.S. Diploma.

Provide system, hardware, software and technical support for SIPRNET. Support consists of, but is not limited to, performing server/workstation checks, performing server/workstation maintenance, rebooting servers as needed, hardware installations and upgrades. Add, delete, modify and reset user accounts and provide user assistance and training. Systems wide patch management and vulnerability scanning and reporting. Maintain Secure File Transfer Protocol (SFTP) server to include accounts, file maintenance, user assistance and training. Maintain Web server to include accounts, file maintenance, user assistance. Maintain Exchange Outlook Web Access (OWA) page. Perform all Microsoft Exchange server functions. Maintain and update System Security Authorization Agreement (SSAA) and all associated tasks/functions paperwork. Perform and maintain all aspects of the system wide anti/virus program. Monitor the EM7 system status program and respond as required. Provide ingress and egress access control for the NETC SIPRNET Facility.  Ensure appropriate logs are accurately maintained.

Candidates must have an active SECRET clearance.
