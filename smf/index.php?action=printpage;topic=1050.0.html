Title: freebsd ports, stay updated!
Post by: ctrlz on October 01, 2003, 05:43:03 pm
for anyone that's using freebsd, and is not aware of portupgrade, you must be aware of it NOW.

make sure you have updated your ports tree before doing this, or else...it won't be applicable! read about cvsup (http://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/cvsup.html)

now that you have your ports tree updated we'll need to install portupgrade:

Code:

bash-2.05b$ whereis portupgrade
/usr/ports/sysutils/portupgrade


found it! okie, let's install:

Code:

bash-2.05b$ cd /usr/ports/sysutils/portupgrade
bash-2.05b$ su
root# make install clean


la la la...wait, ok done!

now...here's the fun part  :) let's go ahead and see what ports need updating:

Code:

bash-2.05b$ pkg_version -L =
gstreamer-plugins


ok, obviously i'm a port fanatic, and update my ports like crazy. but if you had a fresh install, you'd see A LOT of packages.

now, su to root and go ahead and update that one package:

Code:

root# portupgrade -r gstreamer-plugins


la la la...more waiting. the -r option will 'recursively' find all the dependant pkg's. how easy it is!!

well, that's about it...i think. if i left anything out, let me know...OH YEAH!

Code:

bash-2.05b$ man portupgrade
