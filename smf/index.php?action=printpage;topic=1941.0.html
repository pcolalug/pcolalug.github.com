Well, I've been doing the kind of work you describe (ok, no jsf yet) for about a year and a half now. I've been using just plain Eclipse for the longest, save for the Tomcat plugin you mention. If you want all that J2EE helper stuff, I sugguest the Lomboz plugin. Its jsp editor does some syntax hilighting and code complete, I think. If you really want a well-augmented eclipse, I would also sugguest MyEclipse. Its cheap and has lots of plugins already installed and configured. Some folks at work love it.
