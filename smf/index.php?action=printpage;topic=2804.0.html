Title: XFS
Post by: Brent on February 25, 2006, 03:40:19 pm
Hey all,

I been feeling kinda experimental lately, so today I installed slackware on the XFS file system.  If you tell the slack installer to boot using the xfs.s image, it automatically supports creating xfs paritions in the install.  Neat huh?

I know there's a whole debate out there about which file system is the best, in this experience, I found rieser and xfs to be virtually identical, except for a few things.  XFS mounts faster than rieser (not that important to me, but I imagine it would be to someone with alot of partitions), and XFS *does* seem to handle large files faster.  This is good for me cause I have alot of mp3's and videos laying around on my home directory.  Now, I've heard that rieser handles small files better, and I believe it, but I haven't noticed any detremental performance thus far.
