Title: Upgrade from 8.10 to 9.04 ubuntu
Post by: bhoff on April 29, 2009, 01:13:54 pm
Hey guys I have done 3 upgrades now and it worked great and only had one small glitch that I am going to explain.

First I did upgrade via the update manager over the internet vs a disk.

It took about 45 minutes on cox and 1.6ghz machine...
Afterwords my settings and all were good. I did mess up and upgraded my wireless manager to wicd instead of that default one in Ubuntu.
Now I could not connect to my AP. would not get dhcp lease.
a little digging found the culprit it was the /sbin/dhlient3 script it is protected by apparmor.
to fix this do the following from command line
sudo mv /etc/apparmor.d/sbin/dhclient3 /etc/apparmour.d/disable/

that will take care of your little issue now reboot to make sure the security permissions are set for dhclient.
