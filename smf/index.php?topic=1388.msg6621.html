Jesse,

Haven't tried this personally, but remembered these tips from the Mandrake newsgroup, where it's been a hot topic...  

1.  When you run the nVidia installer script, add this option:
./NVIDIA-Linux-x86-1.0-5336-pkg1.run --add-this-kernel

This will rebuild an rpm, which you can then install the normal way.  I would assume you need the kernel source (headers) installed for this to work.

2.  Check out:  http://www.minion.de
They have some prebuilt packages there that reportedly work well.

Good luck,
