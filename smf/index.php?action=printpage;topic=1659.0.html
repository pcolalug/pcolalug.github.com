Title: Web Site Editor
Post by: geekforGod on May 25, 2004, 01:46:44 pm
Are there any programs out there that manage a website in linux? I have some particular tastes though.

I'd like something that does layout in CSS(no ugly hacks or workarounds for IE when possible).
A visual editor, though it does not have to be WYSIWYG(That way I can click to insert something where I want it instead of searching through code to find it).
I'd prefer the application to run on my computer and not on the server.
I want the ability to add and delete items quickly(such as news items, maybe a list of items to delete or when I want to add one I click where I'd like to add it and a pop-up asks for the pertinent information).
I want a template so new pages can be added quickly.
I want items to be grouped, so that if I add a menu item to a menu, it gets updated on all pages and I don't have to edit each page individually.
I'd like the ability to store the files on the server and check them out, so that only one person could edit it at a time.
I'd like an RCS like ability to revert to a former page if I mess up or just like the old layout better.
A seperate test and live code area so that I can test it without messing up the live site.
No database or web server required. The last thing I want to do is install a db and a web server for a non techie so that they can edit simple news items on the site.
Open source would be nice in case I need to extend the functionality, but I am willing to pay(not that those are mutually exclusive).


Anyways, I have looked at Mambo and Jetbox and neither seemed to be what I was looking for. I also played with Plone, but rather briefly. It is possible Plone could do some of this, but I confess I didn't take a lot of time to see. Does anyone have any experience with an application that meets some of those requirements? Maybe a windows app I could get to run under wine? I have never had any luck with wine, but who knows? Or is my best bet to install win2k and dreamweaver? Or to write it from scratch?

Title: Web Site Editor
Post by: imnes on May 25, 2004, 04:36:06 pm
I don't really understand.  Do you want a server-side content management system such as plone, or do you want something that runs on your computer to edit html files such as Dreamweaver?

If you are talking content management, and your site is relatively small I'd say just code from scratch.  If you are looking for something to run on  your computer to edit html files wysiwyg style I'd suggest looking into NVU for a purely linux solution, or if you have the money get Dreamweaver (it runs very well under Crossover's wine (http://www.codeweavers.com/) as you can see here (http://homeweb.org/~imnes/linux/slackware/screens/wine-4-apps.png).).

Post by: rburt3 on May 25, 2004, 07:11:12 pm
Don't ask for much, do ya ;)? I'm with imnes on this one. If you're looking for site design Dreamweaver can't be beat, IMHO. NVU looks promising as well. You may also like to try Quanta. No WYSIWYG as such, but a nice web development tool.

As for CMS, Zope/Plone is good. There are tons out there (slashcode comes to mind). However, the system having everything you describe doesn't exist, AFAIK, especially if you want templated, dynamic, non-server-side content. Sounds like an interesting project, though.
