Got a new computer after Ivan courtesy of Technologies For Tommorrow.  Installed SuSE 9.2.  Almost everything straightforeward.

Had one thing that perplexed me for awhile --- my DSL modem kept disconnecting itself.  It was just a configuration issue --- there was a default Idle Time-out setting that I changed to zero (infinity).  Now everything is slick.

It is a nifty box:
CPU AMD Sempron
Memory 512MB
HD Seagate 80 GB
NEC 17' LCD monitor

PGP installed on it no sweat (When I last installed Fedora Core 2 on my pre-Ivan box I had to hunt down and install some library files before PGP would install.).
