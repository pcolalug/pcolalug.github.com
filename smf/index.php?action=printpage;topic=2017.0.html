Title: Application of the Day: QEMU
Post by: steve on November 10, 2004, 08:12:26 pm
I found this program today while working with Windows Ultimate Boot CD (http://www.windowsubcd.com/).

This program, QEMU CPU Emulator (http://fabrice.bellard.free.fr/qemu/) is really cool. It allows you to run ISO images and IMG images in a window in Linux or Windows. There is a version for both OSes. It's great for downloading a bootable ISO and testing it before burning it on CD. I've only tried the Windows version so far but it worked great. It ran a Linux window inside XP and I was able to run the Windows PE bootable CD in a window in XP.

Anyway, give it a shot and see what you all think.
