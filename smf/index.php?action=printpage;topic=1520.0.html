Post by: rugmonster on March 31, 2004, 01:50:51 pm
Since I'm not very familiar with FreeBSD, I'm not going to be much help, but it sounds like you don't have the correct device selected for your mouse. One thing that helps me out to see if I'm using the correct device is to run cat on it.
Code:
$ cat /dev/[device file]

When doing that on the mouse device, when you move the mouse, you should get random characters output to the console. It works great. Once you have that, you have only to determine the protocol of the mouse.

Also, the same can be done for sound by directing some characters to the audio device file as long as you have write priveleges to it.
Code:
# cat /etc/passwd > /dev/dsp

If you get some sound out, then you're good to go. By the way, the sound won't necessarily be good, but it will at least show it's working.
