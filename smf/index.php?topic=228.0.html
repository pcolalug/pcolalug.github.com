I know nothing about the zaurus.  But I can tell you how it's usually done on my linux machines.

If it's nice enough to have a startup script similar to redhat, you may be able to:
/etc/rc.d/init.d/network restart

Or, depending on what utility the thing uses to get an IP address, (I dunno what your ethernet device is called, but for this example I'll use eth0):
dhclient eth0

or

dhcpcd eth0

or I think some distributions use a program called pump, I don't know what the syntax for using that is.  Maybe:
pump eth0

?

Just some ideas,

(Ok, assuming the Z uses dhcpcd). I do a ps ax to see if its running. If so, I issue a dhcpcd -k (as root) at the command line. This tells dhcpcd to release its current lease and terminate. Once I get the prompt back, just issue dhcpcd (again, as root) and it should attempt to lease an IP.
