I'm in the process of installing Ubuntu on my laptop and I am seriously impressed. 2.6.8 kernel with built in support for my wireless nic! I was able to select my primary interface, configure the SSID and WEP key and it was on my WLAN in a couple of seconds. That shocked the crap out of me. Now, it is in the process of installing X and Gnome, but I've only had to tell it a couple of things, like language and timezone. Everything else has been nearly hands free. I'm excited to see how this goes.

Edit- a couple minutes later...

Yep, I'm in X now with Firefox and this is freaking awesome. Sound, video, network all working out of the box as it should. Very well done Ubuntu. I'll be keeping this around for a while on this laptop. I have always had a love for Debian's apt, and this takes that and puts up to date packages in for what seems to be a great, snappy desktop system. Looks like ACPI is working as well. Wow!

Another edit...

I just ran apt-get install nvidia-glx and it installed all of the packages required for the nvidia drivers. Another command, nvidia-glx-config enable and it changed my X config. After restarting X, I had the nvidia logo. This is how a desktop is supposed to run! I think I would like to try X.org since I've never messed with it, but that may come later.

I put Ubuntu on my laptop last night.  I'm seriously impressed.  I'm not exactly a Linux power user but I've tried out a lot of distro's.  Ubuntu is my favorite right now.

Extremely easy install.  I have the same praises that Rubmonster has.  Found wifi on install, configured everything then downloaded needed updates.  Upon first logon everything seems to work right.  I like the default setup of Gnome.  That's usually how I arrange my desktop with the 2 panels, Ubuntu has it right out of the box.

I've had issues before when swapping between wired NIC and wireless NIC, and with Mandrake and Fedora it wasn't too painful, but it seems to be a lot easier with Ubuntu.  But I've only done it once.

I'm going to be sticking with Ubuntu on the laptop for probably quite a while.
