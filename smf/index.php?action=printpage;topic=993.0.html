Title: FreeBSD 5.1 Revisited
Post by: imnes on September 09, 2003, 04:06:44 pm
Now that it's had some time to settle I gave FreeBSD 5.1 another try.  NVidia drivers work (without hacking) and work well.  And my sound card works perfectly (something that linux still hasn't done in the last 5 years).  

Right now I'm running FreeBSD 5.1, Gnome 2.2 (with Epiphany).  Nice setup, so far I really like it.  

I installed enemy territory in it, and it works flawlessly.  For some reason I've run it under Windows and Linux and both platforms have the same problem, no sound in the intro / videos.  Doesn't seem to be a problem in FreeBSD.

I haven't done it under Linux in a while, but from what I remember kernel compiles take forever.  Kernel compile under FreeBSD was done pretty quick, I didn't time it but it wasn't more than a few minutes at most.

One thing is the fonts for the pcolalug site under epiphany are too small, I'll have to tweak the config I guess.  I'm gonna move on to more apps, see how games run under wine / winex in BSD, and see if Oracle 9i will run.
