Title: Getting past the firewall
Post by: bhoff on May 25, 2005, 07:24:03 pm
Hey just read a neat article about getting around firewalls. It was suggested that you could take a high end printer such as the upper H.P that has built in webserver for admin use and ftp server. The H.P mentioned has the passwords off as default these paticular printers run a version of choi java what was recommended is that you scan the 9001 port range for printers which are usally not protected on firewalls and crafting a java proxy then uploading to the printer's 8mb flash card then making the printer a proxy server that allows you full access to the target network.
After scanning a big target as a work related activity I found 3 printers with this opening that could have been exploited. So for all you admin's out there be sure to firewall those network appliances.
