Alrighty, this is kinda long and convoluted, but I'll give it a shot at explaining how I did it not too long ago.  I did this all from within an xterm, of course, since the installer won't work from the console.

First you'll need to mount Disc 3 (which is where the install script is located), and copy said installer (it's named 'linux-installer.sh' ) to somewhere like ~, or /tmp, or some other place on your drive that has plenty of space.

The installer takes a shell variable, SETUP_CDROM, to tell it where to install from.  Set this by doing:

Code:
export SETUP_CDROM=/mnt/cdrom


Substitute /mnt/cdrom for wherever the cd's actual mountpoint is.

Then change over to the directory where you copied the installer to, and type  './linux-installer.sh', and the installer should start.

When you change the CD, you'll have to manually mount/unmount them when it asks for the next disc before proceeding (I did this by bringing up another xterm). The names for each CD that the installer uses are different than what the actual CDs themselves are labelled.  So if the installer asks for the PlayCD, give it CD 1, etc.

Hope this helps! If you've got any questions about any of this, or if it doesn't work, post them here.

Title: Installing Unreal Tournament 2003
Post by: danc on April 17, 2003, 03:55:04 pm
Got it!  Although there is a catch...  I followed crowe's suggestions above, and was still getting the problem of the cdrom not being recognized as mounted.  After some teeth-gnashing, I thought I'd try disabling Mandrake's "supermount" function for removable drives.  It still didn't work.  So I wrote the changes to my fstab
Code:
supermount -i disable
and rebooted the machine.  That was it!  You do still need two terminal windows, one for the installer/output and one for the manual CD mount/unmount.  So the lesson learned here is that Supermount (although useful) is not always wanted...
