Post by: rugmonster on February 06, 2005, 05:36:38 am
You may be able to do the following...

Open XTerm or one of the Terminal Programs (usually <something>Term).

I believe Linspire runs all as root, so you should have a # as the last character of your prompt. Type the following to see if you can find digikam
Code:
# apt-cache search digikam

If it comes up with digikam, then you can type the following to install it
Code:
# apt-get install digikam


If you aren't running as the user root and don't have the #, then you probably won't be able to install anything.
