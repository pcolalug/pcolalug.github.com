Java Desktop System by Sun Microsystems

Ok folks.  We have yet ANOTHER corporate Gnome desktop for Linux.  This one created by Sun Microsystems.  Dubbed the Java Desktop System (because, apparently, the Java runtime is tightly integrated into the desktop).  Has some nice apps bundled with it, nice clean interface.  If you're a fan of Sun or Gnome, you might want to check it out.  It says it will be available December 2, 2003 (today) but not sure if it actually is yet.




You can read all about it, and view some screenshots at http://wwws.sun.com/software/javadesktopsystem/details.html

I don't know if a free version is going to be available or not.  I'll try to get a copy any way and evaluate it.

Are there any corporate KDE desktops available?  (Not a fan of KDE but I know the rest of you are)...

After a little more research.  The JDS will be available for $100 per year.  It bundles it's own version of Linux with it (based off of SuSE), or Solaris if you like.  There will be a free version available for home users (according to Economic Times).  Reported on the news link on the JDS site, nearly identical offerings by SuSE, RedHat, IBM, EDS, Oracle, and BEA could be available next year.
