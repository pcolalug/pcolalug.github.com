Post by: rugmonster on September 08, 2003, 08:31:05 am
That's a GeForce 4 I assume and should be supported. I have a GF4 and have no problems with it. It could be that you are having a problem with your bootloader. It could also be that your system is trying to start up using a framebuffer display.

At what point on the boot up does the screen go blank? Do you get the boot menu for grub or does it get past that and start the initial boot? If you don't get the grub menu, you might need to boot with a bootdisk, mount your partition where the grub config is and edit it by hand.

Post by: dpeach on September 08, 2003, 09:36:41 pm
Login as the user you created during install. Then type "startx" (without the quotes) and hit enter. I am not a RedHad user, but I think that is the way you start your X server there too.

That is, of course, if you want to use a GUI. Otherwise, your system should be ready to use from the command line.

Now just start goofing around with the examples the book gives on how to use it.

If at this point you are still not able to get X going because of a possible video card problem (or otherwise), then let us know and someone that has experience with that (or similar) card can help you out. I agree with Ruggy that it is probably a GeForce 4 card, and therfore should be supported. Dig around with the search tool here for information on nVidia drivers if you are needing more help.
