Welcome to the group!

Do you have a copy of Knoppix? If so, or you can download it, you might give it a try to just make sure that the combination will work with Linux. PCMCIA protocol is very well supported in Linux. I have had great sucess with any device I have used.

If Knoppix detects it, and it works, then I would say it is possible to run down your problem with the information Knoppix will give. If it does not work under Knoppix, then chances are it won't work. That is not to say that it would be impossible, but Knoppix detects just about everything you could imagine.

If you are not familiar with Knoppix, it is a "Live" CD. This means that you boot directly from the CD and your system runs in a RAM disk. Nothing is installed to the hard drive. So it can be used without "installing" and messing up your current setup. It also has great hardware detection.

Again, welcome.
