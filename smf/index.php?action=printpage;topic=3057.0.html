Title: Book Review -- Programming Ruby: The Pragmatic Programmers' Guide, 2nd Edition
Post by: crowe on November 12, 2006, 07:02:22 pm
I found this book to be very well written and concise, though clearly intended for someone with some level of prior programming experience.  My only real complaint is that some of the chapters, while covering a wide amount of material, seem to be lacking in depth.  Aside from that, however, I found this book to be more than adequate to give a solid a foundation in Ruby, and after it serves as an extremely useful reference for the classes and modules that are built-in to Ruby.

All in all, if you're interested in learning Ruby, this book should definitely be in your collection. 5 stars out of 5.

(This review also posted to oreilly.com and barnesnoble.com)

Quote from: djoed_pcola on November 16, 2006, 10:05:18 am
Did the book cover programming Ruby in a Linux environment?

There were brief sections covering programming for TK, the web, and win32 environments, but that's about as platform-specific as the book gets.  I emphasize the word 'brief' -- 18 pages on web programming, for example, aren't anywhere near enough to adequately cover such a topic.

Quote
  Is programming Ruby independent of the underlying O/S?

The base language is OS-independent.  But like most other languages, it can be extended to include OS-dependent modules (by way of RubyGems -- Ruby's counterpart to Perl's CPAN).
