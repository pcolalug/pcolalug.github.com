45  Misc / The Big Debate / Re: KDE or Gnome on: November 26, 2007, 07:29:41 pm
I'm a huge fan of Gnome because I like simplicity. KDE and Gnome both have the ability to be changed greatly, however, I want to be able to find mundane things quickly. For instance the KDE menu has about 15 different things to choose from to get to a sub menu; Gnome has like 5 default ones. Plus they are ordered better. For instance is the wifi connector internet or utilities?

WARNING: OPINION:
The people that like KDE like to edit the config file because they cant find it on their menu. (Thats a joke, don't cruicify me.) The people that use Gnome like the nice gui experience of X.

I also like the single layered panel. In KDE I almost feel like I should hide it somewhere so the big thing is out of my way. I feel comfortable keeping the gnome panel there at all times.

I do not have much need for the file browser because anything complex I want to do I do in the terminal. They both fulfill my basic wish's, so no complaints.

I feel like Gnome is closer to Windows, so it is easier for me to switch between the two and it is easier for me to show people the differences between the two OS's.

I would agree that KDE has more options with transparancy but other than that I have never seen a reason to confuse myself with it. I can run KDE apps on my gnome computer just fine.

PS: I started my linux journey on KDE with Knoppix, so I am familiar with it (at least whatever version 5.1 was using.) It probobly would have helped if all the programs were in English instead of German.
