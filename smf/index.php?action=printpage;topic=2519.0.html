Title: Bandwidth Meter....
Post by: rugmonster on July 09, 2005, 09:04:42 am
If you enable SNMP, you could use MRTG (http://people.ee.ethz.ch/~oetiker/webtools/mrtg/). That will give you graphs, via webpage as to the recent network or other utilization, such as CPU, memory, swap, mail load. It all depends on what MIBs you have loaded. Anyhow, it's a wonderfully simple and effective tool for monitoring a server or router.
