Well, I was using Mandrake 10.1 for a while. It is great for a desktop machine. But I could never get PyQT to install correctly and Mandrake only offers it as an RPM for the PowerPack+, which as a broke college student I can't afford. So I finally switched back to Slackware after about 4 years using Mandrake. I had to remember how to edit config files again. The Unofficial Revised Slackware Book Project helped a lot with that. Then I needed to install some new packages and Linux Packages was great for that. I am pretty sure I saw both of those mentioned on the forums at some point.

Anyways, after a few hours of work I got most everything I wanted working. My scroll mouse actually scrolls. I can mount and unmount my digital camera. PyQT is installed and working well. I just had a few quick questions. First, my fonts seem different in Slack than they did in Mandrake, not quite as nice looking. It isn't a big difference and maybe I just have a poor font chosen, but it seems like the fonts are uglier. Am I crazy? Secondly, on Mandrake when I plugged in my digital camera it automatically mounted it for me. Now I have to mount it myself. How would I get it to automatically mount? Thanks for any suggestions guys, you are always a great help.

as far as auto mounting your digital camera, if you use gnome 2.8.x+ (2.10 for example), it has a system called HAL that gives you the option to automount digital cameras and removeable media, also I think for this function you might look at supermount (if it's still around), as it used to provide the automounting functions for some of the more popular user friendly distros.

if you're a gnome fan, I'd recommend checking out freerock gnome, a clean build of gnome 2.10 for slackware. http://gsb.sourceforge.net

for the iso:
http://sourceforge.net/project/showfiles.php?group_id=130126&package_id=143903

Dropline Gnome still gets my vote for slackware.  

http://www.dropline.net/gnome

I assume you're using KDE?  You may need to enable anti-aliasing on your fonts in the kde control center.  (I think you just run kcontrol to get to it).  

For auto-mounting devices in slackware without Gnome, good luck.  I hacked my way through it once but it took a while, patching and installing a new kernel + deamon tools, and configuring it all to work was a pain.  If you go with one of the Gnome installs it should just work properly.  (Not though, Gnome does require Kernel 2.6 which is available on the Slackware CD under /testing I think.  If you need help installing that let us know.)
