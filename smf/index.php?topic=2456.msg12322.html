Allison,

You could try the following procedure to see if it'll get you back to the pre-Synaptic configuration.  Just follow the steps in order, and remember that in Linux, capitalization is important (type things just as I put them).

1.  Log in to the text mode console as you said you were doing.
2.  Type "su" (without quotes), then "Enter", and then the root password followed by "Enter".
3.  Type "cd /etc/X11"
4.  Type "cp xorg.conf.old xorg.conf" (and answer yes if it asks to confirm overwriting)
5.  Type "exit"
6.  Type "startx"

That should bring you back to a graphical mode.  Let us know if it does or not, and we'll go from there...
