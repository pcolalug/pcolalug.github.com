Title: Library Application Fixes and Improvements
Post by: imnes on September 21, 2003, 08:57:37 am
Doing some maintenance on the library code.  As usual if anyone encounters a problem please post a reply here and let me know, or Private Message me if you like.

Crowe has been a big help in finding some of the more annoying bugs so far.

Some improvements made recently:
1. When someone requests a book, the email going to the book owner now has a From field of the person making the request, no "Apache System User".
2. When checking out a book, the system will now accept a date in almost any format.  Just entering the phrase "Next Month" will calculate to a correct date for you.  More error detection to try and keep someone from accidentally entering an empty date, or some date in the past.
3. Adding some more information to the "My Collection" page, showing on the list who has your books instead of just saying 'checked out'.


If any more ideas for enhancements let me know.
Nick
