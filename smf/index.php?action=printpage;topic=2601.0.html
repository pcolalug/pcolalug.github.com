Title: digg.com
Post by: dpeach on September 17, 2005, 10:43:00 pm
I found this site recently (http://digg.com). It is a Tech/News site. It is not slashdot, but shows some promise.

You check out the categories of interest. If you like a story and think it should be on the front page, you vote for it. Then if you are just casually interested in what is going on, you just read the front page. Of course you can leave comments on the stories posted. You can also post stories for others to vote on.

As of this writing the site is really being pounded hard and is really slow. I wonder if they got slashdotted.
