Title: Openssl ain't just for secure transactions
Post by: bhoff on June 17, 2006, 04:46:12 pm
I was just flipping thru the lastest issue of Linux Journal
found these useful openssl hacks

Just a little knowledge.
openssl is now
US Federal goverment NIST FIPS 140-2 level 1 valadation
has TLS support the next generation SSL
36 commands


to wrap a file in base64 text encoding
openssl base64 <filename .bin> filename.txt
 
to unwrap a base64-encoded file
openssl base64 -d  <filename.txt> filename.bin

To encrypt a word
echo "Open-sourceenterprises.com" |openssl base64
T3Blbi1zb3VyY2VlbnRlcnByaXNlcy5jb20gfCBvcGVuc3NsIGJhc2U2NAplY2hv
IE9wZW4tc291cmNlZW50ZXJwcmlzZXMuY29tCg==

quick encryption
toor@laptop:~# openssl enc -aes-128-cbc <filename> filename.aes-128-cbc
enter aes-128-cbc encryption password:
Verifying - enter aes-128-cbc encryption password:

Note you will need to remember the form of encryption that you use thou
I put it on the tail of the file for ease

To decrypt file use the following
toor@laptop:~# openssl enc -d -aes-128-cbc -in filename.aes-128-cbc >filename.bin
enter aes-128-cbc decryption password:


generating fast passphrases

toor@laptop:~# openssl rand 15 -base64
757xC69JFsnGmz1lYBjK


There is many other things that openssl can do create hash marked passwords for linux passwd file
amongst other things

Check out the man openssl for details
