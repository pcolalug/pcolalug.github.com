imnes, you make a good point to this whole discussion:
Quote
But I guess it doesn't matter, most non-techie people won't like having to install windows any more than they would like having to install linux.

Your average Windows user that would not like running Linux has never even installed Windows. And that would be the sticking point. My Grandmother would not run Linux (neither of my Grandmothers really know what a computer is, so this is just an example) because she would not know how to install it.

If it was set up, like most people receive their Windows machines, then it would be much easier for her. Until people can easily buy Linux pre-installed on a machine, then it will not hit critical mass in the home market.

I think the important area to start is getting businesses using Linux. As users have it at work, then they can start taking it home and making it their home system.

All in all, I don't think Linux is ready for the home user just because the average user has never installed an OS. Not that Linux itself is not ready (which I don't think it is), but because people don't want to have to install thier own OS.
