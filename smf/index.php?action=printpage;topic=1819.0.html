Title: VOIP
Post by: rugmonster on December 25, 2004, 12:55:38 am
The deal with Asterisk is that it makes a PBX. You can get a POTS interface card to use regular phones with it. If you know phone stuff, think the old Merlin systems. Okay, now you can hook that Asterisk PBX through the internet to a VoIP service provider such as VoicePulse or Vonage. This is how you would get out to the commercial PSTN. The little router box you get for your VoIP box can be replaced with a system running Asterisk provided you are using the correct encapsulation and such to interface with their setup. It all depends on the VoIP provider.
