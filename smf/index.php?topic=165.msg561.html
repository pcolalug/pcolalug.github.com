Hard to say.  There's a few things you can check.

List the directory
ls -l /usr/lib/perl5

You should have two subdirectories
Code:

[imnes@imnes perl5]$ ls -l /usr/lib/perl5
total 8
drwxr-xr-x   41 root     root         4096 Oct  4 07:23 5.8.0/
drwxr-xr-x    3 root     root         4096 Oct  4 07:22 vendor_perl/


If you're missing vendor_perl you may need to install some developer tools off your CD's.  Or I can zip my vendor_perl dir and send it to you if you need it.  

Let me know what you find out.  Also let me know if the problem is resolved when you install the 9.0 final.  When you installed your system did you install the development packages?
