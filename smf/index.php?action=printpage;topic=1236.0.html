Title: Castle Wolfenstein install problem.
Post by: dpeach on December 27, 2003, 07:26:52 pm
Apparently you have the Game of the Year Edition (GOTY). You need to do this:
Code:

You need to download the wolf-linux-*-GOTY.x86.run setup file (replace '*' by the latest version). This file is much smaller than the full release (approx 10Mb).

You need to copy over the .pk3 files from the Wolfenstein GOTY CDs to /usr/local/games/wolfenstein/main directory. Most files are on CD 2, but you need to copy from CD 1 also (Setup/Data/Main directory). See the list below for details. You don't need Wine or a win32 installation, the files are stored as-is on the CDs.

ENTERING THE CD-KEY:

The first time you run the multiplayer game, you will need to go to 'Options', and click on 'Enter CD Key' button. Enter the CD key (it's on your CD Jewel case).
NOTE: you don't need to give a CD key if you are installing a dedicated server.

GOTY is better in that you don't have to have a Windows install of RTCW to install into Linux. Beyond that, I do not know the differences.

Lee,

Assuming what you are trying to install is the stuff from my CD at the meeting, you do need additional files, as David said above.  Those are just the Linux binaries, and they require the "map paks" from a retail CD version.  These map paks are generic for both Win and Linux, but you do need to own the game.

What we were playing at the last meeting was Enemy Territory, which is a (free) offshoot of the original Return to Castle Wolfenstein.  That was also on the CD I had there, and is freely available on the internet (homepage is:  http://www.splashdamage.com ).  Many (if not most) of the old RTCW players are playing ET now, and I like it better.  So....., you could always install that.
