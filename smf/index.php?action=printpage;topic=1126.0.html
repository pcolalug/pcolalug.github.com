Under linux all I would do is create two new files:

/etc/profile.d/java.sh
/etc/profile.d/java.csh

Make them executable, append the PATH in each, and everything was happy.  Not sure what the equivalent would be in FreeBSD.  From what I've read so far in my google searches I need to edit the path in the files

/etc/profile
/etc/cshrc

to get similar results.  I'll try it when I get home tonight and see if it works...
