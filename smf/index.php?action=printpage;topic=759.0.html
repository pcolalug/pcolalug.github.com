Title: Help on Commands at the Console
Post by: imnes on June 10, 2003, 12:39:15 pm
Man Pages are helpful, if you know what command you're trying to use.  If you don't even know that, then two programs are there to help.  

apropos and whatis

apropos will search the man pages for programs matching your keyword.  For example, if you wanted to know what programs are available for compressing files, you might type

Code:
apropos compress


It will list all the programs used for compressing files.  If you have the name of a program but are not sure what it does, you can ask whatis.  It will give you a brief (usually one-line) description of what the program does.

And then, when you need all the gory details, you can consult the man pages.
