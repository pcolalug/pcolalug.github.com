Title: IntelliJ IDEA
Post by: imnes on November 03, 2004, 11:00:21 pm
I'm playing around with some more Java IDEs.  This time it's IntelliJ IDEA by JetBrains (http://www.jetbrains.com/).

Seems good so far.  The GUI editor is very easy to use as a wysiwyg layout tool, but you still have to hand-code a lot of stuff behind the scenes - like action listeners.  JDeveloper and NetBeans do a nice job of writing that boilerplate code for you.

The CVS integration in IntelliJ is the best I've seen, it just works.  I've tried to use CVS features in JDeveloper and NetBeans and they either just flat out don't work, or have bugs / annoyances with them.  IntelliJ is full-featured, even allowing rollback of code changes from the IDE.

This tool also has J2EE web integration, supports I think WebSphere and Tomcat.  I will try to get a server installed and try out those features later this week.

The only problem with this tool so far: $500 per user license.  :(

I think so far the best open-source IDE for my users is going to be NetBeans, but I'd like to give Eclipse a good run before deciding.  Once I figure out all the plugins I need, and hopefully can find a plugin for wysiwyg GUI design.

Nick
