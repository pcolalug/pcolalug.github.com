Title: jTunes 4
Post by: imnes on August 30, 2004, 07:23:31 am
This is someone's attempt at an iTunes knock-off in java.  Found it on sourceforge, looks pretty good, I'm at work now though so can't test the functionality.  If you want to try it, download and unzip:

http://prdownloads.sourceforge.net/jtunes4/jtunes4.zip?download

(http://homepage.mac.com/macandpal/images/2003/jtunes4.jpg)
(This screenshot is from a Mac, on Linux and Windows your window control buttons will be on the right hand side, and you'll have the menu integrated into the window).

Then to run it go into the directory where you unzipped it and type:
java -jar JTunes4.jar
