Title: Having troubles getting on the net with Slackware
Post by: rburt3 on April 18, 2003, 08:18:10 am
Also, did you try running either dhcpcd or dhclient from the console (not sure which one slack uses)? If not, try typing
Code:
dhcpcd eth0
or
Code:
dhclient eth0
from the console and then type
Code:
ifconfig eth0
and see if you have an ip. Remember to replace "eth0" above with your network device.
